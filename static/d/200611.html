Nov.5,2006 (Sun)

■「終末少女幻想アリスマチック」(キャラメルBOX) [GAME]

+ 白衣&黒衣→伊織1→六花→小夜音→伊織2→冬芽で終了。

+ 双子。最初に倒してしまったわけだが、結果的に見るとキャラ的にはこの二人が一番好きな気がする。丼ウマー。えろシーンで白衣が無駄にエロいのがまたよい(ぉぃ

+ 先輩1。アレだなぁ思ってやってみたらなかなかカワイイ人であった。さておき、双子シナリオで先輩が何であるかはわかってた\のは話を進める上でイマイチだったかもしれない。最初にやればよかったかな。

+ 六花。りっちゃん自体は大好きなんだけど、あんな弱い小夜音なんか見たかなかったなぁというのがりっちゃんシナリオの感想だろうか。双子の時もたいがいそうだったけど、蔵人ってえろシーンに入ると言動がけっこうアレだな。

+ 小夜音。蔵人との対戦シーンが実に好みだった。ただ、後半突然出てくる小夜音のあの設定\がちょいとせつないねぇ。小夜音らしくはあるのだけれど、もうちょっと前振りが欲しかったというか。あと、せっかく中の人が俺の大好きなあの人なのに、えろシーンのエロさが足りないのが実に実に残念無念(ぉぃ

+ 先輩2。先輩1と違ってハッピーエンドなんだけどっ、そいつ\はちょいと厳しいぜっ、という感じ。

+ 最後にユメ。ユメは実によい娘だけど……なんかこう、先輩ルートその2もそうなんだけど、話の風呂敷が広がりすぎて逆に醒めちゃったというか。小夜音シナリオ規模程度に留めておいたほうが俺好みだわ。

+ まとめ。黒幕(*1)やそれがやろうとしていたことは、俺の興味がある方向ではないんでどうでもいいんだけど、黒幕の手先であるはずのストレーニアンの描写があまりにも小者っぽ過ぎるのは気になるね。ぶっちゃけしゃべりすぎ。もっと得体の知れない怖さが欲しかった。

+ 一方でバトル自体はけっこう好きかも。序盤の蔵人vs小夜音とりっちゃんvs信綱とか、りっちゃんシナリオの六花vs小夜音とか、小夜音シナリオの蔵人vs小夜音とか蔵人&小夜音vs"彼"(*2)とかね。勝てそうもない相手と戦って善戦して負けたり、僅差で勝ったりするのが俺的には好み。

+ ま、8点かな。よくできました。キャラメルBOXの次作はシナリオが複数人(ほしまるメインだとは思うんだが)なのが気になるね。ま、たぶん買うだろうけど。

*1: CだったりNだったりそれ以外だったり?……よくわからんぜ。
*2: 「工夫」がなんだったかは気になるが、まぁ、言わぬが花なのだろう。

Nov.13,2006 (Mon)

■「ウィズアニバーサリィー&ファンタ！ビジュアルファンブック」 [GAME]

+ 書くことがないまま一週間過ぎてしまったので最近買った「ウィズアニバーサリィー&ファンタ！ビジュアルファンブック」【 amazon / bk1 / 楽天 / 7netshopping 】の話を。

+ 雑誌・ショップ特典系のイラスト・イベントCG・ショートストーリー・インタビュー・ラフと、ファンブックとして普通の構成なのだが、カラーページが多めでイラスト・CGの掲載サイズが大きいのがよいと思う。その分お値段高めだが。

+ ショートストーリーは本編に出しづらそうな裏設定の話で、こういう話が(後づけだとしても)ぽんと出せるこのゲームの世界観はおいしいよなーと思った。

+ インタビューはイベントCGをからめた形式だったのだが、「とにかくパンツ」ってのと、すげぇ時間に追われてゲームを作ってたのだけはよく伝わる内容だった :-)

+ ラフ絵については、他のヒロインはそうでもないけど、アリスとソフィアは最終案とかなり感じが違う案が紛れてて面白い。ツーテール&むっちゃツリ目なアリス案3とか、デコメガネなソフィア案2とか本番のキャラより好きかもしれん。

+ ウィズアニ好きな俺にはなかなか楽しめた本でありましたデスよ。

Nov.20,2006 (Mon)

■SRM-007t帰還……の予定 [AUDIO]

+ STAX SRM-007tは明日――あ、もう今日だな――帰ってくるらしい。帰宅予定時間からすると受けとれるかどうか微妙なところだが。ま、最悪でも木曜に再配達してもらえばいいか。

+ けっきょくヒューズの溶断だけだったらしい。修理のついでに真空管の特性確認(*1)やら、XLRの極性を3番Coldに変更してもらったり(*2)したので送ったのは無駄ではないが。とても安かったし。

+ ま、SRM-007tについてはとりあえず真空管が大丈夫そうってことで安心かな。1,2年後再度確認のために送るとしよう。……うむむ、となるとやっぱ予備のドライバが必要だな。

*1: あと何年保つかはわからないがとりあえず現状は問題ないらしい。
*2: STAXの現行機は3番Coldなんだけど、SRM-007tは2番Coldだったのだ。

■「ちょこっと☆ばんぱいあ!」(Meteor) [GAME]

+ リオ→操→香菜→フラン→アリア→ハーレム→○○\で終了。

+ リオはよいわんこでした(←狼だって)。操さんは実にヘンな人だけど、ふりかえってみると印象が弱かったりする。香菜はさすがにバカすぎでユルすぎな予感。

+ お次にフランことフランチェスカ。フランシナリオ後半の人間っぽいフランも好きだけど、序盤でリオに頭ころころされて「あ、あ。だめなのです。頭部の機能維持に。問題が。問題が」とか言ってるフランも実にかあいいと思うわけです。彼女がどうやって作られたかって真相は実にアレだけど、話としても重くなりすぎずにわりと上手くまとまってたかな。

+ アリアシナリオについては黒幕が操シナリオと同じなのがちょっとつまらんけど、アリアが奴に切実な願いを訴えるところはちょっといい感じだ。ハーレムはみんなして無茶苦茶です。最後のヒトは……まぁ、今までにあの娘がアレなのは誰にでもわかるわけで意外性はないけどあれはあれで良いだろう。

+ ま、「Meteorらしい」感じのするゲームだった。適度にユルい話で気楽に進められるのがよいやね。CVの音質が妙に悪いのはアレだが。評価は8点あげてもいいかな。
