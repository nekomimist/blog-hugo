Mar.3,2007 (Sat)

■きょうのよしなしごと [MISC][COMP]

+ 登記が終わった旨の通知が来た。そういや今の家に来てまる1ヶ月が経過なんだな。早いもんだ。

+ 昨日は夕食にチャーハンを作ったのだが、「さあ食うか」という段になってスプーンがない事に気付いた。というかフォークもナイフも何もなかった(ぉ そんなわけで今日は近所ニトリに行ってそのへんの物と、調味料入れなどをゲットしてきた。

+ んで、帰りにニトリの横のヤマダ電機に寄ってAQUOS RXシリーズを見てきた。枠の写り込みが気になるよなぁ。値段を聞いてみたら思ったほど高くはなかったのだが……はてさてどうしようかなぁ。

+ で、帰宅したらなぜかKONA(Mac mini)のファンが爆音を立てていた orz。負荷もかかってないのに何故、と電源切ったり、PMUリセットしたり、電源抜いて放置したり、MacOS Xの方立ち上げたりしたけど直らねぇ。修理に出してKONAがいなくなるのも困るわけで、爆音状態でなんとかしのいでいる間に代替マシンを稼働させて、それから修理に出すしかないなぁ。

Mar.4,2007 (Sun)

■きょうのよしなしごと [COMP][MISC]

+ 昨日は別マシンを組むなんて書いたけど、メインPCのFIENAが24時間運転してるんだから、KONAはFIENAの上の仮想マシンとしての存在でもかまわんのだよなー。VMware ServerもVirtualServerも一応WindowsXPでも動くようだし……ふむ。

+ そろそろテレビ買おう。いろいろ悩むよりも、AQUOS LC-46RX1Wを買ってしまって後で存分に後悔する方を俺は選ぶぜ!(ぉぃ

Mar.5,2007 (Mon)

■きょうのよしなしごと [MISC]

+ ふと考えてみると今度の社長とオレって13歳しか違わんのな。うむー。

Mar.6,2007 (Tue)

■きょうのよしなしごと [MISC][COMP]

Mar.10,2007 (Sat)

■きょうのよしなしごと [MISC]

+ 買って一ヶ月で130kmしか走ってないけど(*1)Abios Tourerにもだいぶ馴れてきた。最初サドルを固く感じたがポジションがイマイチなだけだったようだ。ペダルまわりはもうちょっとなんとかしたいな。ハーフクリップでもいいのだが、元ランドナー乗りの俺としてはトゥークリップ&トゥーストラップにするのもよいかもしれない。

*1: 基本的に一日に6.7Kmずつしか走らんからなー。

Mar.18,2007 (Sun)

■きょうのよしなしごと [MISC][NOVEL]

+ ああ、くしゃみがひどいー。

+ いちせさんの感想を見て気になってた「ルナ」(沖水幹生/光文社カッパ・ノベルス)【 amazon / bk1 / 楽天 / 7netshopping 】を読んだ。

+ 中学生くらいにに見えるのに酒が大好きで強くて、わりときっぱりモノを言うけど根っこでは自信がない風乃が大変好みだったなー。もちろん人外スキーの俺がるなが嫌いなわけもなく。

+ そんなわけで尚基と風乃の関係がいい感じに進むのもよかったし、風乃とるながひどい目に合うのがイヤだったので終盤は大変ドキドキしながら読めて、たいへん楽しかったなーと。なんか、サスペンスもののエロゲーにあってもヘンでもなさそうな話かもしれない。

Mar.21,2007 (Wed)

■AQUOSが来た [COMP]

+ AQUOS LC-46RX1Wが届いた。デカいが重さはスタンド込みで33kg。その昔メインに使ってらCRTのGDM-F500Rが29kgだったと思うわけで、やっぱ液晶は軽いやね。

+ さて、HDDレコーダもHD DVDプレーヤもBlu-rayプレーヤもPS3もXBOX360も持ってない俺は早速PCにつないでみるぜ。そんなわけでFIENAのRADEON X1900XTにぶらさげてみたところ、何も考えずに1920x1080でDot By Dot表示できた。大変ありがたい。

Mar.23,2007 (Fri)

■きょうのよしなしごと [MISC]

+ 今日は給料日だった。うちの会社は給与支給日はノー残日なのだが、今日は事情があって残る羽目になった。にしても、業務をうまく回せない原因の根っこは俺の「面倒くさがり」のせいなわけだが、30ン年もかけて培ってきたこの自堕落さがそう簡単に直るわけはないわけで、なかなか難しい。……なんとかしたいとは思ってんだけどね。

+ で、さて。給料日なので自宅に帰ってから口座の状況を確認してみる。……ふむ、住宅ローンの支払いが始まっているな。長い長いローン生活の始まりだ。ま、できるかぎりさっさと返してしまうこととしよう。
