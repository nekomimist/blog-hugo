■RADEON HD4870のその後のこと [COMP]

+ ヤフオクで買ったRADEON HD4870だが、週末いろいろ楽しい事が起きた :-)

+ 動作確認してから、クーラーを外して水枕を取りつけ。前回と違って今回は無事起動して安心していたのだが、半日ほど経った後、使用中に画面が腐って直後にブルースクリーン。その後まともに起動しなくなった。あわててバラしたら、VGA水枕のフィッティング部で水漏れしてた orz。

+ まずはエアーダスターでRADEON HD4870のBGAの端子間の水をふき飛ばして乾燥させつつ、水枕のフィッティング締め直して数時間空運転して水漏れがない事を確認して、それで組み直して事無きを得たのだが、なかなかスリリングであった。

+ さっき起動したら、BIOSケーブルの「電源ケーブルを繋げ」表示が出て起動しなくなった。直前まで普通に動いてたやんかー。後で別マシンで動作確認することにして、FIENAはまたRADEON HD3870に戻した。なんか呪われてるとしかー orz

+ さて、壊れてたとしたらどうしよう。さすがに3枚目のRADEON HD4870を買うのはマヌケすぎるし……。消費電力と性能のバランスを狙うなら、4月以降に出るであろうRADEON HD4770が狙い目か。ひさびさにnVIDIAでGeForce GTS250あたりを買うのも悪くないやもしれん。ま、いずれにせよ買うのは4月以降だなー。

+ LULLIに差してみたけどやっぱエラーが出るので、電源まわりのどこかがショートしたときに過電流でイったのかね。しかし、2枚目は中古でだいぶ安かったとはいえ、10日間足らずでRADEON HD4870を2枚も壊す俺って…… orz

Mar.31,2009 (Tue)

■「ボクの手の中の楽園」(キャラメルBOX) [GAME]

+ エリノア→クリス→ルーツィエ→テアの順で終了。実に「ほしまるシナリオっぽい」話だったなぁ。

+ エリノア。最初イマイチかなーと思ったら中盤以降すげーイイ感じだった。CVの春乃伊吹も好きだしな。エリノアが「戻ってくる」ところがちょっと突然な感じがしたが……もう一度プレイして確認してみよう。このシナリオだと、ユウの記憶は戻って事の真相も半分ほど明らかになる上、エリノアの問題は全部解決しているので、すっきり度は高め。

+ クリス。……なんか、ユウとクリスの物語はこれからだ!みたいな感じやね。ルーツィエはクリスシナリオよりは真相に近づくと言えるが、テアシナリオと比べると五十歩百歩やね。クリスもルーツィエもいい娘なので、なんかちょっと惜しい。

+ 最後にテア。ユウの過去の全てに片がつく話だし、この娘がメインヒロイン扱いなのは明確だ。つーか、シナリオ順序制御つけて必ず最後になるようにしてもよいくらい。それにしても、薔薇騎士団長として凛々しい所もあり、料理とか服装とかにルーズな所もあり、実はあまえんぼな所もありで、頼りなく見えるが年上で世話焼きなユウと実にお似合いな娘だよ。CVの風音も良し。

+ サブキャラだが、鉄騎士団団長ディルク・アヒレスがなんか気に入った。最初はあんなに小物臭い言動をするヤツだったのに、ユウに叱責されて以降すっかり情に篤い兄貴になっちゃうあたり、実にイイヤツだ。それだけに音声がないのが惜しい。

+ 小品という感じだが、俺はこういうの好きなので評価は8点かな。「あえかなる世界の終わりに」同様、アフターストーリーを作りやすい話だと思うので、やるきばこ3が出るならば、そういった話を期待したい所。

■「スズノネセブン!」(クロシェット) [GAME]

+ えーと、美奈都→仁乃→すみれ→柚子里の順だったかな。

+ 美奈都。悪友ポジションかつアホかわいい娘。ポジション的には前作「かみぱに!」の天音とカブるが(CVも同じだし)、このアホさは天音にはない :-) このシナリオだと、幸村が美奈都のためにものすごく思慮深く行動するのが印象的。

+ 仁乃。膨大な魔力をうまく制御できなくてワーストセブンな娘。でしゃばらないキャラクターなので影が薄いが、裏のない実にかわいい娘だと思う。幸村を意識してテンパってるあたりはホントかわいいと思う。

+ すみれ。学園長の孫で高い魔法能力を持つが、やる気なしのためワーストセブン入り。この娘は小悪魔だよねー。本人は大変いやがるだろうが、実にあのジジイの孫っぽい。CVの佐々留美子がハマリすぎというか、似合いすぎというか。

+ 柚子里先輩。成績は超優秀だがとある活動のためにワーストセブン入り。柚子里ルートに入ると、幸村も柚子里先輩も壊れ気味で(「かわいすぎる」合戦とか)、美奈都シナリオと続けてやると大変な落差かもしれん :-)

+ 主人公の設定が同日発売の「祝福のカンパネラ」のレスターにちょっとカブるが、あくまで優等生キャラなレスターと異なり、能力は高いが発想が独特ゆえに成績は悪いあたりが人間ぽくてよいやね。

+ 総合的にはソツのないゲームだったなという感じ。御敷仁の絵は俺好みだし、「祝福のカンパネラ」がCVのせいでエロさ半減だったことを思えば比較にならんほど手堅く、かつありがちすぎないCV陣でよかったし。8点。

+ ……ところで、「かみぱに!」と併わせてFD出してもいいと思うんだがどうだろうかね。
