■水冷系強化のこと [COMP]

+ 休み前半にPCの水冷系を強化した。ポンプ(Laing DDC-3.15)も買ったけど交換が面倒だったので、ラジエータ(Magicoolの120S)と12cmファンの追加だけを行った。idle時の温度は変わらないが、フル運転時の温度上昇はちょっとは押さえられるのではなかろうか。

+ しかし、休み終盤に冷却水抜いて新しい精製水入れて洗浄してたら、追加したラジエータのフィッティングの所で水漏れ発生してケース内に浸水してしまった。おかげでRampage FormulaとRADEON HD4870が水びたし orz。ま、精製水で絶縁性が高かったせいか、十分乾燥させたら何事もなく起動したが、実に危なかった :-)

■Vista 32bitとWindows7 64bitの差 / そしてSSD [COMP]

+ RIPPLE(DELL XPS M1330)をVista Ultimate SP2 32bitとWindows7 64bit(RCね)で比較すると後者の方が速い。後者の環境はSSD(Intel X25-M)なので速いのは当然だが、ニコニコ動画再生で前者がガタガタになっても後者だと普通に見れたりして、HDDとSSDの差だけとも思えない差がある。CPUはCore2Duo T8100なので64bitでそう速くなるとも思えないし、グラフィックまわりの差(WDDM 1.0と1.1)だろうか……。

+ あと、やはりSSDはイイ。XPS M1330はHDDの発熱のせいでファンがすぐフル回転になるんだけど、SSDだとそうでもない。負荷かけるとブンブン言うけどな! IntelのSSDがこうも安く買える現状、ノートPCでSSDにしない選択は個人的にはもう考えられない。デスクトップもそろそろシステムドライブはSSDにしたい。Windows7が出る頃を狙って、SSDSA2MH160G2C1かSSDSA2MH080G2C1×2あたりを買おう。

■PCまわりのヨタな話 [COMP]

+ 昨日のことだが、帰宅したらFIENAの電源が切れていた。何かの拍子でフローレートが下がってRESERATOR XTがエラーで停止した後、次はRampage Formulaがオーバーヒート警告で電源断したようだ。コワいね。とりあえずポンプをM/Bのファン端子に直結して流量を稼いで逃げたけど、やっぱさっさとポンプをDDC-3.15に交換すべきだな。RESERATOR XTを分解する必要があるので面倒なんだけど。

+ しかし、最近FIENAはメイン3点のIntel Core2Extreme QX9650@3.6GHzとRADEON HD4870とRampage Formulaが入れかわらないので(*1)、いじるといえば水冷系ばかりでちょっとつまらない。Lynnfield(Core i7 8x0)とIntel P55 M/BとRADEON HD5xxxが出たら交換する手もあるが、現状から大幅に性能が上がるかというとそうでもないし……。

+ VistaはUltimate買ったけど、Windows7はProfessionalでよいかな。あるいはHome Premiumでもよいような気も。RIPPLE(DELL XPS M1330)もWindows7にする必要があるから、Family Packが日本でもし出るならそれを買うセンもあるか……。

+ あと、32bitと64bitのどちらにすべきだろう。FIENAはメモリは8GB積んでるけど、現状のシステム3.3GB、RAMDISK4.7GBで特に問題ないから32bitでいいかな。64bitはメモリが16GBくらいになったあたりを目途に移行する感じだろうか。RIPPLEはせっかくだから64bitでこのまま行くかな。

+ FIENAでNorton Internet Security 2010βを試しているのだが、えろゲのプロテクト(AlphaROM系)やネット認証系とかがかなりウイルス扱いされる。せっせと除外してしのいでいるけど、こりゃもうえろゲのプロテクトは破綻しかけてないかね。

+ RIPPLEのマウスはMicrosoft Mobile Memory Mouse 8000で、ちっちゃくてBruetoothなのがイイのだが、BlueTrackに慣れた体としては追随性に劣るのが気になる。BruetoothでBrueTrackなマウスは出ないのかね。まぁ、nanoレシーバでBlueTrackのモバイルマウスがぽろぽろ出てきてるので、いよいよ困ったらそっちに移行しよう。

+ テクノロジ的にはLogicoolのDarkfieldレーザートラッキングテクノロジーも気になるけど、モバイルマウスの方はホイールクリックがホイールモード切りかえ専用になっていて、ホイールクリック多用派の俺には受けいれられない。MX Revolutionみたいな機構的にとんがったマウスはもう作れないのかなー。

Aug.28,2009 (Fri)

■水冷更新計画 [COMP]

+ 走り出したからもう止まらないぜー。

+ 今はCPU水枕にRESERATOR XT付属のZM-WB5を使っているが、ZM-WB5はアルミ+銅という電蝕が怖い構造なので、これを安いわりに悪くない性能のEK WaterBlocks EK-Supreme LTに交換する。GPUの水枕もEKのだし、揃っててイイよね。

+ ラジエターはHWLabsのBlackIce SR1 480にするかMagicool 480S Radiatorで済ますか悩み中。値段差ほどの性能差はないはずなので、安いMagicoolでいいかなと思いつつも、見た目はSR1の方が好きなので悩ましい。ファンはKAZE-JYUNI 800rpmを4つつける予定。ラジはP182の上に乗せるので、本体と切り離せるようにCPCカップリングで繋ぐ。Zalman中心で組んでたからAPCDシリーズなら数個持ってるしな。圧損が大きい? こまけぇことはいいんだよ。

+ リザーバはXSPC Single DDC Dual 5.25 bay Reservoirに手持ちのDDC-3.15をつけて5.25inchベイに固定する。うちのケースP182はサイズのわりに中が狭いので、DDC-3.15の置き場に悩みたくないのだな。取りつけるときにDDC-3.15を分解するので保証がなくなるけど、こまけぇことはいいんだよ。

+ 折角なのでチューブは全交換しよう。汚れてきてるしな。Tygonと行きたいところだが、今も使ってるQ-Tube 200 ID3/8 OD1/2でいいや。フィッティング(ホースバーブ好きなのでBitspower G1/4-3/8)は手持ちじゃ足りないので買い足し必要。クーラントは精製水に混ぜるaquacomputer ACfluidがまだ余ってるけど、JINGWAYのICELAND COOLANTにしようかな。
