Aug.2,2008 (Sat)

■メインPCのFIENAのこと [COMP]

+ えいやっとCPUとGPUの水冷化を完了。RESERATOR XTで楽をしたのでさほどの手数はなし。 あんまり強いラジエータではないはずだが、なんとか耐えられそうな予感。RADEON HD4870の温度はぐっと下がって、かつ静かになったので、まぁ満足かな。今後のメンテが面倒臭いけどなー。

+ さて、FIENAのその他の事。現在の電源のAntec NeoHE-550に不満はないけれど、余剰のCore2Extrme X6800とASUS P5B-DeluxeとRADEON HD3870で一台PCを組む上で電源がないので、こいつを余剰パーツPCに移して、新しい電源を買おうかと思いはじめている。

+ 出たばっかりのAntec SG-650にしようかと思ったけど、奥行きが180mmもあるので、P182に内蔵するのはちとツラい。ZIPPY-660W HU2あたりにでもしましょうかね。ストレート吸気ならばAntec NeoPower-650あたりでもよいかもしれない。こっちの方がだいぶ安い。

鈴 : 「なんでそんな高い電源ばかり……」
猫 : 「趣味だからねー」

+ メインHDDのWD740GD×2のRAID0も、VelociRaptor1台に負けそうな程度の性能なので変更したいな。64GBのSSD×2とキャッシュつきRAIDカード(areca ARC-1200あたり)でウハウハっつーのはどうだろう。

鈴 : 「それって、また10万円以上かかる気がするけど」
猫 : 「趣味だから!」

+ ま、独身貴族(死語)とはいえ、趣味に使う金はほどほどにせんとね :-)

■最近やったゲーム [GAME]

+ 「借金姉妹2」(Selen)。主人公の名前の読みがヒロユキなので、実際にヒロユキな俺としては、ちょっとくすぐったかったりする :-) (*1) それにしても前作同様香純ちゃんのダメダメ度と香帆のいい娘度が高ぇ。メタっぽいギャグはちょっと多すぎかな。総合的にはまぁそれなりか。7点。

+ 「プリンセスラバー!」(Ricotta)。俺が守るとか女の子に言う癖に、実際に爺さんに対峙するとほぼなにもできない主人公に乾杯(ぉぃ まぁ、うまく何かができる状況でもないんだけど、もうちょっと頑張って欲しいよなー。キャラ的には聖華が一番好みだが特にお気にいりというほどでもなく。そんなわけで評価が辛めに6点。

+ 「Coming×Humming!!」(SAGA PLANETS)。いろいろ文句はあるけれど、美海(*2)と鈴音がキャラ的にたいへんよかった――お話としてはそこそこだが――ので総合的には満足かな。ひなたも次いでよろしい。んで、いかにもおまけ扱いの綾音としおり先生はともかく、メインになるはずの優月がキャラ的に弱いのが難点だよな。評価は7点。

+ 「ツナバン♥らぶみくす」(ういんどみる)。ノリとしては人を選ぶノリなのだが、本編の「ツナガル☆バングル」のああである以上、ファンディスクもこうじゃないとね、という感じで俺的には納得のデキ。真穂がすっかり淫獣扱いいつきさんがほぼ変態なのがヒドいといえばヒドいが実にこの二人らしいのでよかろう :-) そして悠夏はあいかわらず愛いヤツだ。パラレル☆バングルの二人のくっつき方も実にグッド。評価は8点かな。

+ 「飼育白書〜鎖に繋がれた同級生〜」(Delta)。犬・猫・豚・ポニー・馬・ロバ・猿相手のオンパレードで人間相手のシーンが4回くらいしかないというのはいっそすがすがしい :-) 買ってしまってから言うのもなんだがけど、こういうゲームが好きな人はアレだよなー。まぁ、俺は好きなので8点をあげるが(ぉ それにしても、Triangle/Deltaはそろそろシステムのダメさをなんとかしてほしいね。

+ 「タユタマ」(Lump of Sugar)。ゆみな→美冬→アメリ→ましろの順でやったが、アメリが一番好きだった。大変面倒な娘だが(*3)、そこがかわいいやね。ましろは小さいモードの方がイイとか鵺がかわいいとか思ったけど俺はロリじゃないですよ?(ぉ 総合的には7点。ちょっと印象が弱いゲームだったように思う。

*1: 音声ありが主流になってからは本名プレイはしない(できない)からねぇ。
*2: 美海と「WIZ ANNIVERSARY」(CROSSNET)の大賢者のCVが同じらしくてびっくりだ。
*3: 「HoneyComing」(HOOK)の朝陽に輪をかけた面倒臭さがあるよな。

Aug.5,2008 (Tue)

■FIENAの水冷化の結果 [COMP]

+ エアコン稼働時かつ通常使用時は水温は32℃〜35℃あたりを推移する。エアコン非稼働かつ放置状態だと40℃くらい。ま、こんなものかね。

+ もともとあまり熱くなかったCore2Extreme QX9650@3.6GHzはともかく、RADEON HD4870の温度が劇的に下がったのが嬉しい。ノース・サウス・VRMまわりは空冷――ちなみに12cmファンが500rpmで2つ回っている――のままだが、ケース内から熱源が消えているのでこちらも温度はかなり下がっているようだ。

+ 一方、予想どおりポンプの音がちょっと聴こえる。夏場のこの時期はエアコンの方がうるさいので気にならんが、秋になったら気にさわるかもしれない。

Aug.13,2008 (Wed)

■「BackStage」(TJR) [GAME]

+ 演劇を軸とした「ありがちでない」お話と、平均年齢の高めなヒロインたちと、はっきりと目標を持っているが現状うまくいっていない主人公、というのがよいね。個人的には、大人が主人公のゲームがもうちょっと増えてもよいと思うのだが。

+ ゆとりん。あたまわるいグラビアアイドルとしての序盤と、得体の知れない中盤と、「おにんにん」でバカップルな終盤の落差が激しくて面白い。中盤の展開があえて入っているのがこのゲームらしい。

+ 公子(アリス)。言ってることのヒドさが変わらないくせに、光明にぴっとりとくっついて離れられなくなる中盤以降が大変よろしい。あと、えちシーンで攻められっぷりもよい。しかしアリスみたいな声優キャラの声を当てる声優の人ってどういう気分でやるんだろうねー。

+ 安美さん。抱き上げないとキスできないくらいちんまいのに巨乳で年上。光明にめろめろなくせに、えろシーンで激しく言葉攻めしまくる安美さんと、わりと肉体派のくせに安美さんに攻められまくってめろめろな光明に乾杯 :-)

+ 沙織ちゃんはある意味一番普通。いや、初Hの時の服装はヘンだけど :-) 年齢差の関係で光明にまったく勝てないところがかわいい。が、他の娘の印象の強さに比べるとちと弱いよな。

+ 水鏡(すいきょう)先生こと京香。いろいろ普通じゃない人のくせに、光明相手にほんのりデレるのが殺人的にかわいい。口ぐせの「よし、よし」も好きだ。安美さんと並んで最年長のくせにあまりにロリな外見のせいでえちシーンが全て犯罪的なのもよい(ぉぃ アリスもゆとりんも捨て難いのだが、やはり光明の相手はこの人だよな。

+ 総評。正直あまり期待しないでやったのだが、大変面白かった。主人公に余裕のない序盤がツラい(*1)とか、絵が足りてないとか、文句がないわけではないのだが、「HOOKでできないことをTJRでやる」って感じがすごく伝わるゲームだった。CVはあまり見ない人がほとんどだったのだが、これもまたよかった。

+ 評価は8点。よくできました。

■「さかしき人にみるこころ」(light) [GAME]

+ 「嘘屋 佐々木酒人シナリオ」。全てがこれに尽きよう。主人公もヒロインもその会話も展開も実に嘘屋 佐々木酒人テイスト。好きな人は好き、嫌いな人は嫌い。でもこの人はこうであってくれないと俺は悲しいぜ!

+ 亜利美も大変いいのだが、サブのどんちゃんも真路乃もよいね。つーか真路乃が一番好きかも。

+ ロープライスで短いとはいっても「しょぱん!」(light)よりかなり長い(*2)。BGMや背景の流用などあるけれど別に気にならんし、個人的には大変よいゲームだと思う。8点。

■きょうのよしなしごと [MISC][GAME]

+ 昨日〜今日とお仕事。たるい。

+ メインPCのFIENAのこと。いろいろ考えてSSD化は次の機会に見送る事にした。そんなわけで、速度キープで騒音減と発熱減と故障率減を目指して、システム用の二代目Raptor(WD740GD)×2のRAID0を外してVelociRaptor WD3000GLFSに交換。どうやら目標は達成できたようだ。

+ HDD入れかえとほぼ同時にReadyBoostに使っていたUSBメモリが死んだ。もう1ヶ月早く壊れてくれたら保証期間内だったのにーとか思いつつ新しいのを交換。それにしてもSLCの4GBのUSBメモリも安いねぇ。どうやって利益上げてんのやら。

+ 「さかここ」といい、「Backstage」といい、「今時の"ありがち"」からちょっと逸脱しているゲームが好きなんだな、俺は。そんなわけでとても「今時の"ありがち"」な感じの「春色桜瀬」(purpleSoftware)を開封する元気が出ない俺なのでした。先に積みゲ崩すか……。

*1: むろん一回目のヤマを超えたときに意図がわかるし、同じように序盤がツラい「にーづまかぷりっちょ」(りぷる)よりはマシなのだが。
*2: 「しょぱん!」もすげー好きなんだけどね俺は。
