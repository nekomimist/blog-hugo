Feb.16,2009 (Mon)

■きょうのよしなしごと [COMP]

+ RESERATOR XT、チューブ潰れを解消しても、ファン速度&フローレート制御をAUTOにしておくとたまにエラーになる気がする。とりあえずチューブに潰れ防止の針金巻いて制御はMANUALにしておこう。これでもダメなら本気でラジエータとリザーバとポンプ買ってきて水冷系の組み直しだなー。

+ QX9650の4.0GHz運用は数日に一回ハードエラーで落ちる。コアや周辺の電圧喝入れが足りんのかもしれんが、CPU+GPUは水冷でもチップセット&VRMは空冷のままなので、あんまいろいろ喝入れすると不安だ。完全水冷化するか3.6GHzで諦めるかだが……まぁ、すっぱり諦めよう!(ぉ

Feb.19,2009 (Thu)

■RESERATOR XTのこと [COMP]

+ マニュアル制御でも速度を落すとエラーになる。昨年夏頃は大丈夫だった速度なので、流路の負荷が上がってるんだろうか。

+ ……ああもう面倒臭ぇ。RADEON HD4870用の静音クーラー(*1)と、新しいCPUクーラー(*2)買って空冷に戻すかなぁ。もともとRADEON HD4870がクソ熱くてクソ五月蝿いから水冷にしたわけだし。これならポンプ+リザーバ+ラジエータを揃えるより安く揃うしー……なんて覚悟を決めたらいつ壊れてもいい気がしたので、とりあえずRESERATOR XTが完全に壊れるまで騙し騙し使おう(ぉぃ

■「祝福のカンパネラ」(ういんどみる) [GAME]

+ メインライターのサイトウケンジの味をどう取るか。ま、みんな人の心を読みすぎだし、レスターを誉め過ぎだよね。数人そういう人がいるならいいのだが、そういう人だらけというのはアレな気がする。

+ ヒロインの中ではアニエスが一番好みだったな。キャラ的にもイイし、くっついた後のアニエス視点のアニエスの言動が楽しかった。次点はチェルシーさん。落ち着いた感じの女性のくせに方向音痴だわ、イジられると弱いわとよい感じ。あとは、ミネット、カリーナさんの順。ミネットに手を出すのはどうもなぁと思ったのは俺だけだろうか。あのコは娘として愛でていたいですよ。カリーナさんも別に悪かないんだが、最後にやったせいかもう飽きてた :-)

+ サブの中ではニナさんイチオシ。なんでニナEXシナリオはちょっとせつない終わり方なのだー。次点がガーネット。

+ 絵。かわいいのだが、なんかえろシーンの体の描き方がおかしいのが多かったような気がする。いや、それで挿れるのキビシイだろ！？という感じ。演出とかは特に問題はないのだが、終盤のバトルをデフォルメキャラでやられると、ちょっと笑ってしまう。

+ 総評。声が大きい人がイヤがりそうな物を排除した結果、「突き抜ける何か」に不足してしまったゲームだと思う。なんか、こう、足りない。ぶっちゃけ俺はアクの強い「ツナガル☆バングル」の方がずっと好きだぜ。ま、7点かねー。

■きょうのよしなしごと [COMP]

+ こんなこと書いていたわけだが、昨日もRESERATOR XTはエラーを出した。なぜか風呂入ってるときにエラーになる事が多くて腹が立つ。とりあえずファン/ポンプ速度上げてみたが、これで騙し騙し使うにしても、速度上げるとファンが五月蝿いのが問題だ。ファンだけ低速のものに交換してその分フローレートを上げるのがよいのだろうか……。

+ ……って、速度上げてもエラー出やがった。試しに最高速にしても以前ほどフローレートが上がらないということは、ポンプがヘタったか流路に何かあるかだな、こりゃ。とりあえず代替のCPU/VGAファンを発注して、週末にでもバラすとしよう……トホホ。

Feb.28,2009 (Sat)

■復活のRESERATOR XT [COMP]

+ 週末になったので、さあRESERATOR XTを外してCPUクーラーとGPUクーラーをつけようと思ってFIENAさんを停止。

+ で、水枕二つを外してバラしてみたら(*1)、CPU水枕内部に固形化したクーラントがびっしりと付着して流れを阻害していた。クソッタレな水枕やなーと一瞬思ったが、付着の様子を見て俺ははっと気付いた。ZM-WB5は中央側の口から水を入れ、外側の口から水を出さないとダメやんかー。なに逆に繋いどんねん、去年の俺よーと。

+ 水枕内部をざざっと清掃して、仮組して試運転したらフローレートが倍くらいに復旧したので、数時間水を回して様子見て水漏れがない事を確認してからFIENAさんに組み込み。RESERATOR XTの復活である。

+ 半年後くらいにまたフローレートが悪化すると思うので、来年の冬前にまた掃除しよう。ただ、できればCPU水枕は交換したいなー。

*1: CPU水枕のZM-WB5は「バラしたら保証外」のシールが貼ってあるのであまり真似しないように。
