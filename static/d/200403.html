■「愛cute!キミに恋してる」(ぱれっと) [GAME]

+ 「大小前後の立ちCGと横スクロールを活用して視点移動と空間の広がりを見せる演出」が多用されてるのは面白い。キスシーンをちょっと引いて見せる絵が多いのも好み。やはり女の子がちょっと背伸びしてキスするシーンは萌える♥ しかし横顔の描きかたが気にならんこともない。気になると言えば誤字の多さも気になる。これは修正してほしいな。

+ 評価は8点+αで「佳作+α」という感じかな。文句はあるけど、まぁ、楽しませていただきました、と。

■Clam Antivirusふにふに [COMP]

+ HIMIKOだけでウィルスチェックするのも鬱陶しいので(*1)、Clam AntiVirusを導入して、AKANE(FreeBSD)側でもチェックするようにしてみた。

+ Clam AntiVirusはFreeBSD portsにあるので、さほど悩むことなく導入できる。clamav-milterをつかうつもりだったので、いじるべきはsendmail.cf、/usr/local/etc/clamav.conf、/etc/rc.confといった感じで特に何事もなく導入終了。ちっ、もっと苦難の道が待っているかと思ったのにー(ぉぃ

*1: メール読んでると無駄にポップアップ表示されるからなぁ。

Mar.12,2004 (Fri)

■AKANEのMTAまわり変更と、HIMIKOのトラブル [COMP]

+ AKANE(FreeBSD4.9R)のメール周りをsendmail + clamav-milter + clamavの構成からpostfix + amavis-new + clamavの構成へ変更。amavis-newの設定が読みづらかったことと、rootにあった謎の.forward(*1)で怪しい転送が発生して首をかしげたこと以外は特に問題もなく終了。つまらん。

+ ところが、その作業の間にHIMIKOのエロゲ置き場パーティションが大破損。ハードウェア的問題っぽい。まずchkdskをかけると途中でエラー終了。次にパーティション内を再フォーマットしてみるも63%で一歩も進まなくなってこれも失敗。結局クイックフォーマットでしのいでその後chkdskで全面検査することでなんとか復旧。200MBほど不良になってるっぽい。

*1: いや、俺が置いたとしか思えんのだが ^^;

Mar.18,2004 (Thu)

■bogofilter→bsfilter [COMP]

+ bogofilter + kakasiから、bsfilter + MeCabに変更してみた。

+ とりあえずはHIMIKO上のMeadow+Wanderlustの環境から使うので、Cygwinでビルドする。MeCabのビルドは何も問題がなかったけど、MeCabのrubyバインディングはビルドに失敗 :-p 。その後ここを参考にしてやっと成功。

+ うちはspamの99%が英語でhamの95%が日本語という感じの状況なせいか、bogofilter + kakasiでは日本語spamを排除しきれなかったのだが、bsfilter + MeCabは今のところ日本語spamもうまくひっかけてくれていい感じ(*1)。ま、しばらくこれで運用してみようっと。

*1: データベースが日本語用とその他用に分かれているせいだろうか。
