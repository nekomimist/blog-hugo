■「よつのは」(ハイクオソフト) [GAME]

+ なんかヘンな感じのゲームだったなーという感想が残った。システム的にも、お話的にも。2006年に出たゲームで「WindowsXP SP2はサポート外です」ってスゴすぎです。

+ さて、ザッピング的に3年前の回想がぽこぽこ入れつつ進めるのは悪くないと思うんだが、3年前と現在とでみんな「かわらなすぎ」なんでねー。もうちょっとじっくり"3年も空白期間があったこと"を感じさせるお話にしたほうがよかったんじゃなかろうか。望んで空いた空白でもないんだし、ね。

+ ま、キャラはわりと好みだったし、わりとさくっと終わらせることができるボリュームだったし(重要)、飽きずに終わらせられたのでそんなに評価は低くないけどなー。

+ イチオシは圧倒的にのの。実にかーいくて良し。

Mar.7,2006 (Tue)

■「春恋＊乙女」(BaseSon)その3 [GAME]

+ 残りを一気に倒しちまおうとことでかなり飛ばし気味にプレイ。

+ 彩夏。中盤以降「なにこの後向きな人」って感じだけど、一方このシナリオでの主人公の行動はそう悪くないので、なんだか主人公が不憫。もっとも二人が上向くきっかけは皐月先輩と麗架先輩だったりするので、主人公が"何かやった感"は足りないのだがね。

+ 羽未。結衣佳が"ウミちゃん"と呼ぶのに意味があったあたりはちょっと「ほー」と思ったが、このシナリオギミックがお嬢様学園舞台の恋愛モノ向きかどうかと言われると、ね。最後の関係再構築はもちっとじっくりやってもよかったかも。結衣佳シナリオなみに時を置けとは言わんが。

+ 結衣佳。羽未と対なので真相は同じ。結衣佳がひたすらズタボロに沈んでいくのが痛々しいが、結局は結衣佳の自縄自縛なのに主人公が責められたりヘコんだりしてるのはちょっと不憫だ\。ラストちょっと時間置いてるのはよいけど、そのあと青姦突入では台無しです :-)

+ さて総評。お話として悪くはないとしても主人公がぐじぐじ言ってなかなか行動してくれない分、爽快感に欠けるのが残念かな(*1)。やはり男は「こう」と決めたら凛々しくないとな。まぁ、ヒロイン側も負けず劣らずぐじぐじしてるのでお似合いか。

+ イチオシは当然莉流。百合百合しい展開だが、純粋に恋愛なネタで事件が起きるのこの娘だけだし、最後に二人で立ちなおった感じがするのもよい。エピローグがバカップル風味かつ場違いにえろいのもポイントが高い。次点は、ええと……夏冬コンビ?(笑)

+ 評価はドリル加点で7点。

■AUDIO事情 [AUDIO]

+ 最近、SR-007よりSR-003の方が使用頻度が高い。えろげソングやえろげプレイにはSR-003の方が向いてるからついついSR-003が差しっぱなしに……。

+ 自堕落にもほどがあるとは思うが、要するに「聴きたいCDをラックから持ってきて、CDプレイヤーに火をいれてCD入れて、SRM-007tのの入力を切り替えて、SR-003抜いてSR-007差して、さあ聞こう」ってのがおっくうなんだよね。

+ うちのM-AUDIO DeltaDiO2496→(SP/DIF光)→KENWOOD DPF-7002(DAC代わり)→STAX SRM-007t+SR-007ってな構成で、iTunesでApple Losslessを再生するとどんな音質だろうなー。今度やってみよ。……本気でやるなら20万クラスの単体DACくらい欲しいけど。

+ そしてまた自堕落な方に流れてゆく俺なのだった。

Mar.20,2006 (Mon)

■「おしえて巫女先生」(あねせん) [GAME]

+ あねせんの成り立ちを考えると当然っぽくはあるけど、ハーレムのないとらぶーのゲームのようなそんな感じ。あと、巫女で先生である必然性が奈辺にあるのやら、という感じもするね。

+ キャラ的にはつばめが好み。女の子が大好きでオヤジ臭くて主人公と下ネタトークをくりひろげるほどに"男友達"っぽかったのに、「主人公に自分が女の子だと思われうる」ことに気付いてから妙にかわいくなっちゃうあたりがとても良し。……「春恋＊乙女」(BaseSon)の莉流に続いてまたコレ系のキャラに負けたのか俺様。

+ ま、総合的に見れば5点で「普通」かなねぇ。

■「お帰りなさいませ ご主人様」(あんでる) [GAME]

+ どうも話のテンポが悪い。話としてはドタバタ調なわけだから、もうちょっとテンポよい展開をしやすい主人公にしたほうがよかったのではなかろうか。独占欲強いわりにそれを溜めこんで鬱鬱する主人公では明るい展開になりづらかろう。

+ メインヒロインの冥(CV:韮井叶)なんかスカっとしないキャラでどーにも。「おしえて巫女先生」もそうだったけど韮井叶は最近低調気味か? 秋葉(CV:まきいづみ)も今一歩足りない。やっぱりまきいづみはドジっ娘をやらないとダメかもしれん(ぉ 他の娘たちは名前もよく覚えきれない有様で。そんな中でイチオシは真希さん(CV:金田まひる)。ま、俺はちんまい年上キャラLOVEだからしょうがない :-)

+ ついでにシステムのレスポンスも冴えなくて、全般的に冴えないゲームでした。前作「でふこん☆わん」は主人公とメインヒロインの空にムチャな勢いがあったおかげで、話のテンポはそう悪くなくてわりと好みだっただけにかなり残念。

+ そんなわけで評価は5点で「普通」。もうちょっとがんばりましょう。

■きょうのよしなしごと [AUDIO]

Mar.31,2006 (Fri)

■「さくらのさくころ」(minori) [GAME]

+ 「桜鈴奪還」見てねーけど書いちまおう。

+ 「はるのあしおと」のファンディスクなわけだが……サイドストーリーは妄想や夢ネタばっかなのだった。「Wind」のFDもこんな感じだったと思うので、まぁこれがminoriの芸風なのだろう(*1)。ま、悠やゆづきや和の後日談を見たかった人は御愁傷様としか言いようがないのだった。

+ しかし、個々のサイドストーリーは本編智夏シナリオで智夏とくっついた後と考えると矛盾が少ないようになっているので、智夏スキーな俺としてはなんだかちょっとだけ嬉しいのだった。

+ と、minoriのゲームの感想らしく「のだった」ばかりでまとめてみたのだった(そうなのか？)

*1: 俺としてはえろいお話より、甘い後日談が見たいと思うのだが……。
