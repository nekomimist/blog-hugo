May.5,2010 (Wed)

■連休の終わり [MISC][BIKE]

+ なんか連休をだらだら過ごしてしまったなー。

+ イベントは1-2日に古い友人共と集まって平城京遷都1300年祭に行ったり山でバーベキューやったりしたことくらいか。痩せてたせいで驚いてもらえてなによりだった :-) わりとしこたま食った気でいたのだが、体重はむしろ減った。謎。

+ 昨日は通勤用のAbios Tourerをバラしての大掃除。懸念事項だったチェーンリングとスプロケットはかなりきれいになったが、クランクとマッドガードが傷だらけで見栄えが悪いのが難点だなぁ、コイツ。RNC7のカタがほぼついたので、こっちもぼちぼちパーツ交換して行くか……。

+ 今日はRNC7のフォークコラムを切った。パイプカッターをこりこり回して切ってささっとバリ取って終了。RNC7本体の懸念事項はこれで特になくなったかな。固定ローラー台を発注したので、ローラー台用とそれ以外用でホイールを分けたくなる日がきっと来るだろうけれど :-)

+ AbiosとRNC7の整備をしていて思ったが、もうちょっとケミカルを揃えたほうがいいな。パーツクリーナーが要る。仕上げのポリッシュも欲しい。チェーン用にドライワックスが欲しい。

May.16,2010 (Sun)

■きょうのよしなしごと [COMP][BIKE]

+ 仕事が忙しいせいで日記もTwitterも沈黙気味。いかんなー。

+ 先日届いたSPDシューズ(PearlIzumi X-Road)に慣れるために固定ローラー台をくるくる回してトレーニングしている。つけるのも外すのも慣れてきたので、そろそろ外でも使ってよいかな。

+ 時間がちょろっと空いた時はローラー台を踏んでるので、ゲームの消化が悪い。最近終わらせたのは「幻月のパンドオラ」(Q-X)と「恋色空模様」(すたじお緑茶)の2本のみ。そして感想を書く元気はない。

May.23,2010 (Sun)

■きょうのよしなしごと [MISC][BIKE]

+ 金曜にローラー台+ジョギングでいつもより多く運動したら、太ももが筋肉痛になったので、ほとんど何もせずぼーっとしていた土日であった。まぁ、平日は忙しいから休日くらいのんびり気分でいいのかな。

+ ローラー台をちょっと踏むだけで汗だくだ。去年Amazonで買った安物サーキュレーターを自転車の前に置いてフル回転させているんながな。まぁ、流れ出る汗は拭けばいいしシャツは替えればいいのだが、パッドつきのインナーパンツが1枚しかないので、替えがなくて困る。とりあえず夏に向けてレーパンを1枚所望してみた。サイズが合うようなら追加で買い込もう……。

May.31,2010 (Mon)

■きょうのよしなしごと [MISC]

+ 仕事がどーにもこーにも。開発人員が減ったおかげで現状を支えられる人員が足りてないよなー。

■PCのこと [COMP]

+ 水冷系のおかげでAntec P182が手狭に感じられたので、新しいケースを発注してみた。CORSAIR CC700D。P182よりさらにデカいので置き場所には悩むが、まぁ、要加工ではあるが120x4のラジエータを内蔵できそうなので……。

+ さて、これでE-ATXのM/Bも無理なく載るようになるので、Core i7 Extreme 980X+Rampage III Extreme+メモリ12GB(2GB×6)を夏ボでえいやっと行けるぜ(ぉぃ

+ Core i7 875K+Maximus III Extreme+メモリ16GB(4GB×4)とどっちがいいだろう……。
