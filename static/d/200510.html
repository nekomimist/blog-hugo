Oct.5,2005 (Wed)

■「思春期」(RUNE) [GAME]

+ 智恵理→みあ→雪来→すずめの順で進めて終了。普通に良作な感じ? しかし野々原御大、絵をだいぶ変えてきましたなぁ。今回は模索中な感じがしないでもないけど。……一応元絵描きな俺としては、大幅に絵を変えようと思い切る時の気持ちはわからないでもなくていろいろ考えてしまうなぁ。

+ さて智恵理。ドジ巨乳眼鏡幼なじみ。巨乳ヒロイン……野々原原画でこれとは一体どうしたRUNEよ?(*1) 転回点の起こる事象\はありがちではあるのだが話の締めかたはよかったように思う。しかし、CVが草柳順子のキャラは決して俺的に最萌えにならない宿命でも背負ってますかそうですか。

+ みあ。常に前向き元気娘に見えるが実は……。こういう青山ゆかりも悪くないのう。しかし中出しばんばんのくせにアレ前かよー。こりゃやっぱり「初恋」に引きつづき今回も裏は中学生設定ですかな。ちょっとラストはとってつけた感じがあったんだけど、全部終わらせてみればあれで正解っぽい。

+ 雪来。主人公が昔住んでた田舎から来た物静かな美少女。スタイル良好で超敏感なところが良し。CV:清水香はわりと好きなのでまるで問題なし。話はどっかて見たような感じがするけれど、雪来ちゃんが実にせつない感じでよし。すずめシナリオのカギになる事項が出てくる話なので、雪来とすずめどっちを先にやるかは難しいところ。

+ すずめ。一番昔からいる義妹だが実は……。CVは立花舞……ってぽんこつか。ある事情により両親まで交じえてなかなかドツボな展開をするのだが(*2)、智恵理・みあ・雪来と友成&さっこをからめて上手く納めた感じでよし。このシナリオだけちょいとえろさが少なめですかな?

+ 総評。なかなかよいね。ただ、「Ricotte」(RUNE)より好きかと言われると微妙かもしれないが、評価は7点で「佳作」。

*1: いや、ベル先生(大)とか前例はないこたないけど。
*2: 「初恋」の杏シナリオのより本格版っつーか。

Oct.23,2005 (Sun)

■STAX SR-007断線キター (;_;) [AUDIO]

+ 2000年3月に買って5年半で4回の断線。最近あんまり荒っぽく使ってなかったつもりなのだが、まぁ毎日数時間使ってると思えばしかたがないとも言えるのかな。ま、定期チェックを兼ねて修理に出すとしますか。

+ 例によってSR-007を修理に出すとその間まともに音を出せるモノがなくなってしまうのが難点だ。うちでSR-007の次にマトモな出力っつーとSHURE E2cなんだが、あいにくヘッドフォン端子を持ってる機器はiPod以外ほとんど持ってないので役に立たない。

+ ……何SR-003を「ぽちっとな」してますか俺様っ(ぉ

Oct.26,2005 (Wed)

■「はぴねす!」(ういんどみる) [GAME]

+ ……現代+魔法な世界観なのに、登場する魔法使いたち全員が殺傷能力を十分に持った魔法を使えるってはどうだろうねぇ。だいたい何に使うんだその攻撃魔法(ぉ

+ まずは春姫(CV:榊原ゆい)。優等生風味な娘。話の展開的が王道的すぎて「はあ、そうですか」という感じがしないでもない。葛藤は主人公の側にあって春姫側にはあまりないからかね。キャラ的にはわりと好み。嫉妬深いとことか、脱ぐとスゴいとことか(ぉぃ

+ 次に小雪(CV:日向裕羅)。不思議お姉様。キャラ的にはすげぇ好みなのだが、中の人が日向裕羅なせいかえろシーンがイマイチなので残念。いろいろ裏の事情が出てくる話なのだが、締め方がちょっとあっけないかなぁ。

+ 次にすもも(CV:安玖深音)。王道義妹。うざいと言うほどうざいキャラではないのだが、わりと俺の苦手な方向の妹キャラで残念。ま、伊吹が意地っぱりながらもすもも効果で状況を受けいれるので話としてはいい感じではある。

+ そして杏璃(CV:成瀬未亜)。負けず嫌いでトラブルメーカー。前半ほぼドタバタで進んだのでギャグ担当かと思ったらそうでもなかった。王道的ではあるもの、うまくできなくて萎縮しちゃうところも、主人公に支えられて励まされて一生懸命がんばる姿はよかったな、と。

+ で、最後に伊吹(CV:みる)と沙耶(CV:三咲里奈)。伊吹側の事情は小雪さんルートでだいぶ既知なので、真ルートと言うのほどの新事実はないわけだが、最後の伊吹が実にかわいかったので許す(ぉぃ

+ 総評。冒頭のヨタネタのように設定自体にツッコミ所はあるのだが、そのへんを受け入れてしまえば大きな破綻はなかったかな(*1)。絵はかわいいし、えろシーン数もまあまあだし、いろいろよく動くし、わりと人に勧めやすいかな。ただ、「はぴねす!ならではの何か」はなんだろうと思うと悩むかな。評価は7点で佳作。

■きょうのよしなしごと [GAME]

+ はぴねす!の準。よいヤツだし見た目かわいいし中の人は佐本二厘だわでズルいのですが、CAGEのゲームをつい買ってしまうような俺には今一歩ものたりないです(ぉぃ

■STAX SR-003 [AUDIO]

+ 届いたのでSRM-007tに繋いで小一時間大きめの音量で適当に鳴らしたあと試聴。

+ 耳を覆う形のSR-007に対し、カナルタイプのSR-003なので、装着感がえらい違うのはしょうがないが、SHURE E2cよりはかなり楽に長時間つけていられるし、SR-007特有の暑苦しさもなくていい感じ。今度から夏場はSR-003で乗り切ることにしよう。

+ 聴いてみるとSRM-007tの威力もあるだろうけどちゃんとSTAXな感じ。ちょっと高域が丸いかな。あと音量上げるとうるさく感じる。ここはSR-007と比較するのが悪いってことで納得するしかあるまい :-) ま、もうちょっと聴きこまんとね。

+ さーて、バックアップ環境も手に入れたし、SR-007を修理に出しますかな。

+ さて、OMEGAIIは弱いか?ですが、Lambda Nova Basic、Lambda Nova Signature, SR-007, SR-003と4種類使ってきましたが、イヤースピーカー本体は一番丈夫です。

+ ただ、私はSR-007で音楽聴きながら部屋を動きまわってついケーブルを「ぴん」と張らせてしまったりすることが多いので、この負荷がケーブル付け根に蓄積してそれで断線するんじゃないかなと思ってたりします。

+ 席から立つときはきちんと外してスタンドに置くようにしておけば、そう断線することはないでしょう、きっと。

*1: 悪いヤツがいなさすぎな感もあるけど。
