Jul.10,2007 (Tue)

■Fastladderに移行してみた話 [COMP]

+ Bloglinesをずっと利用していたのだが、Fastladderに移行してみた。理由は正に「なんとなく」。BloglinesでOPML吐かせてFastladderにインポートしてみたらフォルダ名が化けたので、それをふにふに修正しただけで移行完了。

+ フィード数が少ないのでありがたみがよくわからんかも。でも使用感は悪くないな。

■「E×E」(ゆずソフト) [GAME]

+ 鈍感で何もしなかった前作「ぶらばん!」の主人公に比べると、今作の主人公・籐矢は強いわけじゃないが何もできないほどには弱くもなく、まっすぐ青くさく頑張るべき時に逃げない主人公で俺は嫌いじゃない。もうちょっとアタマを使って行動してもよいとは思うがなー。

+ まず未緒。籐矢との歳の差があるわりにはみょーにかわいい人で良し。しかし最近風音の声を聞きすぎな気も。次に夏希。「ん、肯定」。みるの声、合ってるなぁ。あと、イメチェン後に頬染めてる立ち絵がかわいすぎる。そして悠。素直になれない青山ゆかり声のキャラっつーのはとても罠だ。髪型も体型も好みだしな。デートのシーンが大変よろしい。

+ あとはサブヒロイン系。まず夏希ルートからの派生で沙耶(CV:一色ヒカル)。……夏希の立場がー。最後に円(CV:榊原ゆい)。一人だけ事件も何もなく妙に初々しくセイシュンして終わるのだが、これはこれですがすがしくて良し。

+ オチのつけかたとか話自体にはツッコミ所はあるけれど、雰囲気とキャラで押し切られた感じ。評価は8点で「佳作」。

■「うつりぎ七恋天気あめ」(キャラメルBOX) [GAME]

+ エリン・華・月砂・深雨あたりが好みな感じかな。前3人がほしまる氏で、深雨ねぇが嵩夜あや担当な気がするが、どうだろうか。

+ エリン(CV:青山ゆかり)の近視ネタがやたらくりかえさえるのだが、「ドドドドド、にゃー」のイベントCGのおかげで案外何度見てもよかったりする。華(CV:安玖深音)はよい妹なのだがイベント数が少ないのが難点だろうか。月砂さん(CV:内野ぽち)はあっちのモードに惚れた(ぉぃ 深雨ねぇ(CV:茶谷やすら)はエロ担当なのだが、それだけに終わらない含みがあるよい感じの話で○。

+ まぁ、いかにもキャラメルBOXらしいゲームだねって感じで。評価は7点で「佳作」。

■「恋する乙女と守護の楯」(AXL) [GAME]

+ 主人公の修司(妙子)に声がないのがさみしくもあり、でもどんな声ならいいのかがよくわからなくもあり :-)

+ 雪乃はちゃんと考えて覚悟してただ守られるだけじゃないことを選択するあたりがよい感じと言えよう。無謀だけど :-) 蓮は前半はともかく後半の行動が頭悪すぎる。鞠奈は悪かないのだが、エンディングがちょっと好かん。そんなわけで、メイン3人だと雪乃が一番よい。

+ だがまぁ、サブの有里と説子さんの方がもっと好きだ。共通部に有里の印象的なイベントが各種埋めこまれてるので、ルートに入る前に有里がすっかり気に入ってしまった。本来学生ではない年齢のくせに(これは修司もだけど)みょーにウブでかわいいのも○。文句なくイチオシ。

+ 説子さんは緊迫した状況でほっぺたむにゅーってされちゃうあのシーンがよい。あと、説子さんは、自身のルートもよいのだが、有里ルートラストや雪乃ルートラストもなかなか印象的。

+ まぁ、いろいろ詰めが甘い所もあると思うのだが、俺は好きだ。8点で「佳作」。

■そのほかのゲーム [GAME]

+ 「でるた！〜おねだり天使とひとつ屋根のした〜」(アトリエかぐや)。アトリエかぐやは飽きてきてたので最近買ってなかったのだが、新チームということで買ってみた。ま、案外悪くなかったかもしれん。各ルート終盤のお話は基本的に陳腐だが、志弦さんが好きーってことで許せる感じ(←乳に騙されてるだけという説あり)。評価は6点で「普通」。

+ 「アメサラサ〜雨と、不思議な君に、恋をする〜」(CUFFS)。なんというか……誉めにくい。キャラ的には光羽が好きだが、お話的にはアレだ。霖の話はメインらしい話で悪くないのだが、安玖深音の声のせいか霖という存在に「不思議感」が足りない。評価は6点で「普通」。

+ 「ドラクリウス」(めろめろキュ〜ト)。中盤までなかなか面白いのだが、終盤で力尽きた感じがするねー。詰めが甘いデスよ。さておき、なんでリアンだけ別ルートがあって、リカさんやベルチェにないねんっ。……まぁ、リアンルート自体はいろいろ「微妙」なんだけどな。評価は「佳作」で7点。

■ゲーム用HDDがおなくなりに [COMP]

+ Majestouch Wireless(茶軸)をFIENAにつないで遊んでいたら、その直後からFIENAがよく固まるようになった。Bluetoothキードードでシステムごとロックアップするのもヘンな話なので、いろいろ調べてみたところ、しばらく後にpagefile.sysを置いているゲーム用のHDDが死にかけなことが判明した。

+ しかし、バックアップを取ろうとしたら、その途中でマウントすらできない状態まで行ってしまった。しくしく、プレイ中の「にーづまかぷりっちょ!」(りぷる)のセーブデータが消えてもーた。まぁ、まだ1つしかエンディング見てないからマシだけどさ……。

Jul.22,2007 (Sun)

■Majestouch Wireless [COMP]

+ リビングでPCを使っていて、座ってる場所とPCの場所が離れているので、キーボードに線がつながっていないのは快適だ。巷のワイヤレスキーボードはヘンなキー配置の物が多いのだが、こいつは至極真っ当な109キー配置でキータッチもいい感じなのが嬉しい。単3電池2本が必要だが、eneloopが使えるので大きな問題はない(*1)。

+ 一方、IBM SpaceSaver Keyboard II→東プレRealforce89→東プレRealforce91UBKと長いことテンキーのないキーボードをメインで使ってきたので、ひさびさのテンキーアリのキーボードを使うとマウスの置き場にちょっと困る。そこが難点。

+ あと、OS(Windows)が立ち上がらないとキーボードとして使えないので、BIOS設定を弄るときは別のキーボードが必要。まぁ、キーボードは余ってるので(*2)さほど問題にはなるまい。

+ なかなか満足度の高い買い物だったように思う。

*1: それにしてもうちはeneloopだらけだな……。
*2: 前述のSpaceSaver IIもRealforce89も"使える"状態で保存してるしな。
