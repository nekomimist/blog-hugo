■Windows7のこと [COMP]

+ 今週発売なのな。FIENA用にWindows7 Professional Upgrade一本を注文済みだが、Amazonだけにkonozamaになりそうな気がしないでもない。

+ 当初32bitで行こうかと思ってたけど、VMware Workstation 7RCが予想以上にいい感じなので、作戦変更して64bit版を入れて、動かないゲームはVMware上のWindows XP SP3にまかせる予定。いっそのことえろげーは非常に性能が重要なもの以外は全部VMwareに入れちゃう方が悩まずに済むかも。HDD容量の厳しさは、今ゲーム用に割いている1TB-HDDを全部VMwareにあげれば解決するし。

+ んで、そんなデカいHDDイメージをWHSにバックアップされると困るので、その場合はホスト側のVMwareのHDDイメージをWHSのバックアップ対象外にして、かわりに仮想マシン側にWHSコネクタを入れて全部バックアップ対象にすればよいかな。

■アンチウイルス [COMP]

+ ESET Smart Securityのライセンスが全部切れた。Norton Internet Securityのライセンスも来月切れる。

+ ESETは俺的に拘る必要性がなくなったので止めることにするが、再度Nortonのライセンスを買うか、この際仮想マシンと同じMicrosoft Security Essentialsで済ますかで悩んでいる。

+ トータルソリューションとしてNorton Internet Security 2010はそんなに悪くないんだよなー。履歴見ようと思うとやけに反応が鈍くて困ることもあるけれど。

■自転車のメンテ [BIKE]

+ AbiosのVブレーキのパッドを交換した。新しいパッドはコレ。次にパッドを交換するのは2年後か3年後か。その時はVブレーキ本体を今のACERAからDEOREあたりに変えてもいいかも。

+ Abios関連の残りのやりたい事は、

  • チェーンの伸びチェックをして、必要ならばチェーンを変える(おそらく必要)。
  • 前見た時にヤバい感じだったので、リムテープを交換する。
  • 全体に汚れてきたので洗車したい。ブラシとか買おう。

+ あたりだろうか。ちまちまやろう。

■「ハルカナソラ」(Sphere) [GAME]

+ やひろ。駄メイド初佳さんならともかく、やひろさんが悠に惚れるのはちょっと無理があるよなーとか思ったりしないこともない。

+ 委員長。妄想ダダ漏れな娘で実に俺好みだった。委員長視点で進むのは、悠視点だと話が進まないからだろうが、委員長のダメかわいさを強調する効果があったように思う。今回のイチオシ。

+ 最後に穹。本編は狂気が垣間見えてちょっと恐いんだけど、今作は無邪気に笑っている絵が多くてよかった。

+ それにしても、「ヨスガノソラ」の初佳さんといい、「ハルカナソラ」の委員長といい、ダメっぽい娘が好きだなぁ、俺。

■Windows7の話、つづき [COMP]

+ 手持ちのPixus MP810のWindows7対応が気になってCanonのページを見た。ちゃんと対応してくれるらしい。Vistaの時も64bit対応していたので、たぶん64bit対応もされるだろう。なのでこいつは問題なし。

+ PCIeに挿さっているRADEON HD4870とXonar D2XとオンボードでMarvell Yukonはどれも大丈夫のはず。USB機器はまずはIntuos3でこれもOK(ほぼ使ってないがな)。hueyはどうかな。まあダメなったらもっとまともなのを買うだけか。

+ VMware内でDirectX使う場合、ホスト側のビデオメモリが効くような予感(ホストとゲストで共用するんだろうからね)。現状のRADEON HD4870の512MBより、RADEON HD5770やHD5850にして1GBにするのも案外悪くなかったりするのだろうか。
