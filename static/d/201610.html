+ さてこれで中部海域の基地航空隊も2つになったので、予定通り来週末に6-5をやりましょう。

*1: ZaraはLv109、PolaはLv105なので別にレベルは低くはないのだが

■艦これ。6-5反省会 [GAME]

+ さて。大破撤退は事故なのでしょうがないとして、問題はボス未撃破のA勝利が3回もあったことなので、改善策を考える。

+ ボスA勝利は道中で中破艦が増えて対空母棲姫火力が不足した時に発生した。摩耶さま&綾波さんor夕立or大潮として対艦火力を損わないほうがよさそうだ。……摩耶さまの運を上げて対空魚雷両カットインにすると実に有効そうだ。

+ 雷巡は耐久と対空重視で北上大井じゃなく木曾を入れたが、どうあっても雷巡は大破しやすい気がするので、来月は今回の木曾と同じような装備の北上さまでやってみよう。

+ 戦艦にはレベル上げを兼ねてアイオワさんを使ったけど、空母棲姫相手には夜戦火力不足な感じだった。大和型かビス子を使うほうがボスの安定度は高そう。道中安定を考えると大和型が無難だろうか。

+ 空母は触接率を上げてCマスの安定度を上げられるかは気になるところなので、来月は大鳳さん&加賀さんで攻撃機少なめかつ制空高めかつ彩雲を積むとどうなるか見てみようかな。

■艦これ。追加グラ [GAME]

+ 今回の追加グラは全面的にいいねえ。特に多摩。浴衣の時も極上にかわいかったし、多摩の追加グラは実に恵まれてるなあ。うちの最初の軽巡で指輪も渡し済みなので、かわいい絵が来るのはまったくもって大歓迎。

+ さて秋刀魚。3尾と8尾の任務を完了。8尾任務の報酬は、手持ちがWG42が6つ大発が7つという状況だったので、すごく悩みつつ大発を選択。WG42を連合艦隊であっても一艦隊に6つ以上積む局面が思いつかないし(*1)、大発は遠征艦隊3つに4つづつ使うと最大12コ必要になるからまだまだ足りないからねえ。

+ といいつつ、次にU-511が掘れるチャンスはいつ来るのやら。と思うとちょっと後悔しないこともないのであった……。

*1: 過去の実績では最大4つ
