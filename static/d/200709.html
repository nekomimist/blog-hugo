Sep.5,2007 (Wed)

■「にーづま かぷりっちょ!」(りぷる) [GAME]

+ システムがNScripterだ。最近見なくなったよなー。

+ ライトなハーレム物に思えるがそうじゃない。序盤の円(主人公)は過去の不幸もあって猛烈にネガティブ。生きてない。学校でも空気。一方にーづまとしてやってくるヒロイン連中のうち凛々と双子(雪菜&暖花)は人として何か足りてなくて円に対する態度もヒドい。幼なじみの愛は「キミはダメな人なの」とか言う容赦なさ。味方といえば永遠子さんだが「お姉さん」的立場かつ甘やかせタイプではない。そんなわけで救いはにーづまの一人の夏海だけだけど、家事は壊滅的の役立たず。で、そんなダメダメな彼らだけれど、物語が動きだしてみんなすこしづつマトモになって、きちんと家族っぽく成立していくのでした、という感じだ。

+ 双子。一番どうでもいいと思って始めたら大変かわいくて敗北。雪菜のブランコのシーンとか非常にかわいいのだが、やはり円を傷つけてしまう暖花がよい。あと、カスム君のエピソードもよい\。

+ 凛々。円が旦那様で自分はお嫁さんと決められたからそのように行動するけど、でも、正直円が死のうがなんだろうがどうでもいいとか思ってる一番「欠けてる」人。そんな彼女も、円の一生懸命なとりなしでななと友人として関わるようになってからは、円になでなでされて頬を染めたりとたいへんかわいらしい。

+ 遠子さん。相当年上のくせに存外かわいい。しかし永遠子さんはいくつなんだろう?(ぉ あと小春さんが容赦なくヒドい。と、思ったら最後の最後\がたいへんかわいかったので許す :-)

+ 夏海。凛々や双子の相手にへこたれそうな円をはげましつづける健気な娘。でも、序盤から円スキスキ\なので、他の娘たちのような落差がないので印象が弱いかも。まして、夏海シナリオ内では愛が劇的な変化を遂げるからなー。

+ 愛。頼りになれるようにいっぱい努力したのに、円がそばにいさせてくれなかったがために壊れていく円を見ているしかなかった娘。円に昔のように「愛ちゃん」と呼ばれてふんにゃりなりだすあたりからが絶品だが、夏海ルートからの分岐なので、夏海を捨てることになるのが非常に心苦しいのが難点 :-)

+ シナリオの一部矛盾や誤字脱字はともかく、お話は存外しっかりしていたゲームだったなという印象。ちょっとエロ薄なのが難点か(*1)。評価は7点で佳作。

*1: 数はあるのだが、なんか薄い。

Sep.17,2007 (Mon)

■「虹色あるけミカン」(しらたま) [GAME]

+ 魔法と錬金術が普通に存在する世界で、「あるけミカン」という人の意識を操るアイテムが出てくるわけで、なんとなくドタバタでかつえろえろな話になるのかと思ったら、そうではなくてまともすぎるとも言える話が展開する。そんなわけで、事前の想像との落差ですごく地味に感じる。ま、地味に「ちょっと短かめのイイ話」なので気持ちよくプレイ完了できるのだが。

+ 先輩もピカリンと委員長も悪くないけど、キャラとしてもお話としても桃花が好きだ。錬金術だけは超優秀だがそれ以外はダメな非常識な塊の桃花だけど、くっつく前あたりからの言動が実にかわいくてぐっと来る。

+ えろを求めるならイマイチかもしれんけど、こういうよいヤツらのよい雰囲気の話は嫌いじゃない。ま、もちっとあるけミカン使ってえろえろでもよかったのになーとか思わんでもないけど、主人公の雄大の性格設定とお話をブチ壊わさないとできなさそうだしな。

+ 評価は6点で「普通」。

■「Purely」(RUNE) [GAME]

+ 恋鳥→悠→愛歌→つむぎ→追憶で終了。

+ 恋鳥。あんまり聞かない感じの楠鈴音。主人公を「おまえ」呼ばわりする毒舌で孤高の人かと思いきや、最後はさんざん甘えてくるようになるあたりがツボ。温泉部の中で一番ちんまいくせに実は一歳年上っつーのもツボ(ただ年上設定は何の前フリもなく突然出てきた気がするが)。お話のオチのつけかたがある意味スゴい\が、恋鳥が大変かわいいので全て許す :-)

+ 悠。風音もそろそろ違う芸風のキャラを聴いてみたいね。この娘も強い娘かと思ったらそうでもない感じ。主人公の優哉がわりとヘタれずに行動するヤツなので、その分女の子たちはどこか脆く設定されてるんだろねぇ。亡くなった妹に対するしがらみがお話の柱なのだが、その妹が公式のキャラ紹介の通り、妙な形で姿を表していろいろのたまうので、あまり重くなくほどよい感じだ。

+ 愛歌。CVが桜木ハルキなせいかお口のシーンがみょーに長い。もちろんフェラスレ住人の俺とすると望むところだ(ぉぃ 終盤の展開は前に挙げた二名に比べるとなかなかキツい。状況が状況なので悠や恋鳥のように主人公が頑張ってもどうにもならないし、二人で乗り越える話にもならないし。とはいえ、ほろ逃さを残しつつきちんと前を向いたエンディングなので、プレイ後の感じはそう悪くない。

+ つむぎ。よく聴くまきいづみ。つむぎよりも義妹の千紗(CV:文月かな)に惹かれてしまった俺としてぇは千紗の救済が欲しい :-) 他と違ってやけに文章が細切れで、句点じゃないところでクリックさせられたのがツラかった。時間切れだったのかな?

+ 追憶。つむぎシナリオに輪をかけて時間切れ感が。Ver1.01の時点ではスクリプトミスで画面の外に文章ががんがん溢れてゆく。1行溢れるくらいならよく見るが、10行以上溢れてゆくのは他の追随を許さないのでは :-) ラストは安直に行かなかったのは好感が持てるが、主人公の状況を冷静に考えてしまうとちょっと微妙\。

+ 絵はむにゅう氏。CAGEでのろりぷに分は押さえられてむしろおっぱいな感じ。好きだがみんな同じ顔っつーのは言っちゃいかんのだろうなぁ……。BGMは安瀬聖らしい感じでよし。「Garden」(CUFFS)も期待してるぜ。

+ 雰囲気が悪くないゲームなだけに、そこここに見られるシナリオの矛盾やスクリプトミスやらがすげぇ気になる。Ver1.02でどこまで直ってるんだろう? 評価は7点で佳作。
