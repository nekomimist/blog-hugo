■「Lost Passage」(DEEPBLUE) 1stプレイ / 2ndプレイ [GAME]

+ ロリに食傷したので、年長方面からということで、教育実習生の理乃と、育美先生を終了。

+ まぁなんつーか「御子石氏らしい話やなぁ」と思ったが、デキ自体は観月と沙雪あたりを終わらせんと判断つかんなぁ。

■「それは舞い散る桜のように」1stプレイ / 2ndプレイ (BasiL) [GAME]

+ 主人公のノリ…というか会話内容がちょっと肌に合わない。まぁ、慣れるだろが。

+ 次は里美こだま。偽小鳥風味のくせに胸はおっきくてちと防御不能(ぉぃ 唯一の年上ヒロインのくせにかわいすぎます、こだま先輩。「終わり」はやっぱり唐突風味。イヤな感じと焦燥感が深まる様子はもっともっとひっぱってよいと思うんだがー。

+ 雪村とこだま先輩の話からすると、全員エンディングへの流れが同じなんだろうねぇ。
