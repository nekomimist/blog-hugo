■艦これ 秋イベント海域 E-4 [GAME]

+ 夜戦マップな上にゲージ回復、道中はカットイン・連撃だらけと、まさに鉄底海峡……もう二度と行きたくない。

+ 最初は重巡なし戦艦最低構成の省エネでゲージを削った。

  • 夕立Lv67/時雨Lv67/北上Lv80/大井Lv79/霧島Lv82/榛名Lv80

+ 夕立/時雨/北上/大井はほぼ固定。駆逐艦は小ダメでもバケツぶっかけたしハイパーズはMVPでキラキラしつづけてたので疲労なしで連戦。小ダメを放置してた戦艦だけは疲労するので、比叡Lv81/榛名Lv80/霧島Lv82でローテーション。金剛改二Lv89は燃費悪いので出撃なし :-)

+ この構成で20回出撃して姫到達8回、姫撃破1回で、ゲージ破壊圏内に入ったので三式弾×4にするために駆逐out重巡in。

  • 北上Lv82/大井Lv80/鳥海Lv66/摩耶Lv65/金剛Lv89/榛名Lv80

+ これで2回出撃して姫到達1回で無事撃破。最後は金剛が夜戦で大ダメージを入れて、昼戦で鳥海が最後のイチオシを入れてくれました。

+ 8:21〜12:19の約4時間の戦いでした。省エネにこだわったので、燃料6K、弾丸3K、鉄7K、バケツ56個消費と話に聞くより低消費で乗りきれてラッキーな感じ。

+ ただし大破進撃を3回する羽目になってダメコンが2回発動してしまい、時雨と榛名の轟沈ボイスを聞いてしまって心が痛い。

+ 残りの1回、榛名大破進撃で轟沈フラグ立ったままボス戦入って、殴られないまま昼戦に移行したあとボコボコ殴られたんだけど、小ダメのみでダメコン発動に至らなかった。夜戦→昼戦の場合、遷移時に轟沈フラグが落ちてる?

Nov.8,2013 (Fri)

■艦これ。秋イベントE-5威力偵察で絶望気分 [GAME]

+ 今日は有休消化で休みだったので、メンテ前にE-5をちょろっと威力偵察してみた。

  • 上ルート:5回トライ、ボス前到達4回したけど全部北西のおしおき部屋送り。なんじゃこらー。
  • 中央ルート:2回トライで1回ボス到達。ただ、支援攻撃があまり効かんかった上に、昼戦で重巡航巡4人中3人が大破に持ちこまれてほとんど削れなかった。

+ ボスの昼戦に耐えて、夜戦でボスに打撃を入れられる戦力をいかに残すかだねぇ。E-4はボス夜戦スタートなので、誰か残っていればボスを削れるチャンスがあったけど、E-5は昼戦スタートなので、残った戦力がさらに削られるのが厳しい。

+ まぁ、中央ルートで1/2で到達してるんだから、試行錯誤しているうちに削りきれたかもしれんし、いつか大井北上が夜戦で倒してくれたかもしれんが、ちょっと精神が耐えられないなあ……。

+ とりあえず夜戦でボスに大ダメージを入れられるカットイン陣を増やしたい。といってもレベルと回避と運が高いのは時雨改二とハイパー北上さましかいないが(*1)。となると大井連撃装備・金剛型×2連撃装備・重巡航巡の誰か連撃装備・時雨カットイン装備・北上カットイン装備で、北回りはでどうか？

+ あとで潜水艦削りもやってみよう。主力の疲労抜きのときやってもよさそうだし。

*1: 雪風が欲しかったなぁ。

■E-5突破後の艦これ。 [GAME]

+ イベント開始前に持っていなかったのは、大和・長門・雪風・飛龍・翔鶴・初風・夕雲・長波・三隈・熊野。E-5突破時に持ってなかったのはイベント限定ドロップの阿賀野。

+ 大和は大型建造が来ないとどうにもならない。長門・熊野はイベント終了後にデイリー建造で戦艦レシピやって狙おう。三隈はそろそろ建造落ちするだろうし5-2でも出るので後回し。雪風はドロップは期待せずレア駆逐レシピでの建造を狙う。飛龍は4-1/4-2/5-2あたりでいつか落ちることに期待。夕雲・長波は5-xでのドロップを狙えるはず。まだ5-3やってないしな……。

+ そんなわけで、秋イベント限定の阿賀野と、ドロップが3-4ボスのみと凶悪な初風を狙ってE-3を無心に回す。すると、ボス2回目に翔鶴が、ボス16回目に阿賀野がドロップした。あとは初風が出たら5-2ボス巡回に戻れるのだが、やはりなかなか出ない orz。

+ 10月作戦は500位には入ったはずなので、二式偵察機とスツーカが手に入るはず。スツーカは潜水艦任務と合わせて2セットになるのでこれはこれでよいかな。航戦推しな俺としては試製晴嵐欲しかったけど! 欲しかったけど!

+ 衣笠さん、重巡改二になるとうちの主力の鳥海さんと比べてそんなに遜色なくなる上に、夜戦火力は火力値+雷装値なので夜戦火力で鳥海さんを上回っちゃうじゃないか。鳥海改二まだー。さておき、20.3cm(3号)連装砲を持ってくるのがポイント高いのと、妙高・鳥海の次に来た初期入手重巡でLv27まで育ってるので、Lv55を目指してみようか。

+ 改二を目指す衣笠さんはLv42になって、鳥海Lv71/摩耶Lv70に次ぐ重巡三番手にまで上がってきた。あと13上げればよいのでとりあえずはこれに注力。
