Oct.12,2010 (Tue)

■STAXとFOSTEXの新製品 [AUDIO]

+ STAXの次期フラッグシップ。形状的にはΩ系ではあるが、SR-Ω→SR-007→SR-007Aと来て次はなんだろうな。SR-007を持っていてSR-007Aをスルーした俺としては、大変心惹かれる。

+ FOSTEX GX250。よさげだし、ペアで70万なら出せなくはないのだが、これを置く場所がAQUOSの横にないのが残念だ。アンプが釣り合わない問題もあるし、スピーカー買うならやっぱペアで40万以下くらいが限度かのう。

■「汐見崎学園演劇部 恋ぷれ」(Meteor) [GAME]

+ 旧Meteorの権利がSilverbulletに移ってMeteor復活。Cometとの関係が微妙だよねー。ま、Cometも音沙汰ないけど……。

+ 鮎香さん。「瀬那様のメイドであること」が彼女の原点だったのに、そこが恋愛ロールプレイこと「恋ぷれ」を通じてゆらいでしまったっつー話なんだが、そんなお話なんかどうでもよくなるほどエロメイドっぷりがいっそすがすがしい。「さっきの続き、いたしましょ？」「おいしそう……」とかねー。

+ ここあ。瞳と眉の描き方が特徴的だがちゃんとかわいいと思う。大変残念な事に脱ぐとイマイチなんだよな。鮎香シナリオや瀬那シナリオと比べると特に障害はないせいか地味な感じであった。ここあは脇役としてのほうが光ってる気がする。

+ 瀬那。十二分にエロいお嬢様キャラなのだが、鮎香さんのエロメイドっぷりと比較するとちょっと霞んでしまう :-) 昔の自分達と今の自分達でちょっとすれ違う話だが、さほど重い話にはならない。つーか、くーちゃんモードの瀬那と最後の瀬那ってあまり変わらんような……。

+ システム的には微妙。フェードアウト時に立ち絵全体のαをいじるから背景が透けて見えるのは狙ってやっているのかどうか。あと、クリックによる音声カットしたくない俺としてはそれがカスタマイズできないのも気になる。

+ お話。日頃活動していない演劇部ってことは、つまりはダラダラしていただけなわけで、それにしては部長の瀬那先輩が演劇方面で持ち上げてたりして「演劇を無礼るな」という感じがしないでもないが、エロメイドに免じで全て許す :-)

+ まぁ、冷静に考えれば7点だろうが、なんか鮎香さんのエロ暴走っぷりが見事なので8点。

Oct.16,2010 (Sat)

■オーディオ系のよしなしごと [AUDIO]

+ STAXの新フラッグシップは30万〜50万円とか。イイ値段だぜ。30万ならなんとか手を出せなくはないが、50万だと気分的に手を出せんなー。

+ そういえば、LavryBlack DA10を個人輸入して4年半ちょいか。そろそろDACも更新したい気もすんな。LUXMANのDA-200あたりはどうだろうなー。

Oct.24,2010 (Sun)

■きょうのよしなしごと [MISC]

+ マンションの管理組合理事当番が回ってきた。これもまたこの家で暮らす上の義務なわけで、仕方がないといえば仕方ないのだが、仕事がこの有様なのになー。

■冬の散財計画 [COMP][AUDIO]

+ ストレス発散で買い物しようにも今は時期が悪いものばかりなのが困りもの。

+ PC系はSandyBridgeが出た後にCore i7 2600Kあたりにする予定なので来年だし、メインTVの交換はアナログ停波後のつもりだからやっぱり来年以降だし、オーディオ系もイヤースピーカーはSTAXのC32の時期と値段次第なのでやはり来年以降だ。

+ オーディオ系のネタとしてはメインスピーカーの交換もあるのだけれど、今のPIONEER S-PM300以上のグレードのスピーカーにこの設置状況(*1)を引きつぎたくない。……と考えると、リビングのモノの配置を変える所から始めないとマズいわけで、いろいろ億劫だ。

+ まぁ、リビングのPCとオーディオまわりの配置は満足していないので、ちょこちょこ小物の散財でうまい形に落とし込めるよう努力してみますかなー。

*1: テレビ台の端に厚さ5cmの大理石台を置いてその上に乗せている。
