Apr.2,2009 (Thu)

■ああ、もう新年度か [MISC]

+ 2009年もあっという間に1/4が過ぎたかー。そろそろ次の開発に本腰が入るので、また忙しくなるんだろか。俺は別チームにレンタル中なのでどうなるのかよくわからんが :-)

+ 食費が順調に減っていっている。夕食の自炊率が上がったことと、会社での飲み物を全て持参のもので済ませているのも地味に効いているようだ。会社に弁当を持っていくようにすればさらに下がるのだろうが、さすがに毎朝それをする元気はないなー。

Apr.4,2009 (Sat)

■きょうのよしなしごと [MISC]

+ なんか木曜ぐらいからずっと腹の調子が悪いな……。

Apr.6,2009 (Mon)

■きょうのよしなしごと [MISC][COMP]

+ 会社に行ったら仕事でそこそこ縁があった方の訃報が入っていてヘコむ。

+ 去年11月に買ったMicrosoft Explorer Mouseのホイールクリックの効きが悪くなった。ついカっとなって、Sidewinder X8 Mouseを買ってしまった。Microsoftなんだから電話で文句言っていろいろ送れば交換してくれるはずなのに、何やってんだかね、俺。

+ Emacs-23.0.92を使っていてふと気付いた話。japanese-jisx0208の0x213d、0x215d、0x2171、0x2172、0x224cあたりが全角になっていない(しかし0x2141, 0x2142は正しい)。大変「いかにも」な文字がおかしいのだが、俺の設定が悪いのか、pretest中のEmacsが悪いのかわからん。とりあえず、ほっとこう。

Apr.12,2009 (Sun)

■きょうのよしなしごと [MISC][COMP]

+ REGZAの 55ZA8000がちょっといいなーとか思ってしまった。とはいえTVくらいは自社製品を買ってあげたいわけで、画質も機能ももうちょっと頑張ってほしいねー＞SHARP。

+ RADEONのドライバをATi Catalyst 9.4に上げたらDVI経由で800x600をうまく表示できなくなった。今までは800x600@56Hzという謎なフレッシュレートながら表示できていたのだが……。LC-46RX1WとRADEON HD4870の食い合わせの問題だろうか。試しにHDMI経由に変更したら800x600@60Hzで表示された。GPUスケーリングも動くようになった。よくわからんが、こっちの方が望ましいのでHDMI経由で使い続けることとしよう。

+ 固定資産税と都市計画税の納税通知書が来てた。存在自体を忘れてたぜ。4月も黒字の予定だったが、この金額を一括で支払うと赤字転落だなぁ、トホホ。

Apr.13,2009 (Mon)

■水冷の話 [COMP]

+ 水温が30度を超えている。夏場の無冷房時に一体何度になるのやら恐しくなる。

+ 冷却を今より強化するにはラジエータの強化が必要だが、RESERATOR XTのポンプにこれ以上の負荷をかけるのは危険な気がするので、やるなら水枕以外全交換だろう。となると、ケースをCOSMOS等のデカイのに交換して全部内蔵するとか、今使ってるAntec P182のケース側板に穴あけてデカいラジエータつけるとかしたくなる。……ま、ボーナス時期まで悩みますかね……。

+ CPU水枕が前回詰まった理由の一つが電触っぽいので、何にせよ夏までに水枕は交換しよう。GPU水枕とメーカーを揃えるならばEK Supremeだが、Alphacool HF14 Yellowstoneなんかも捨て難い。

Apr.15,2009 (Wed)

■水冷の話またまた [COMP]

+ 使ってるPVCチューブが固くなったり色がついたりしてきてるのと、一部ケーブル取り回しに無理があってチューブが潰れ気味だったりするので、チューブ・フィッティング・45度中間アングル・チューブ折れ防止コイル等々を発注してみた。週末にでも交換しよう。

Apr.16,2009 (Thu)

■水冷の話これまた [COMP]

+ 掃除するためにRESERATOR XTの中を開けてみた。なんかポンプがコレにそっくりだった。性能的にもほぼ一致するので同一のモノなのだろうな。揚程もフローレートもほどほどって感じだ。

+ 中の広さ的にはポンプをLaing DDC-3.15 TPMCあたりに交換できそうではある。将来的に拡張する場合はそういう手も候補に入れてみようか。

■きょうのよしなしごと [MISC]

+ 明日は計画有休で一回休み。固定資産税を払ってこようかね。

Apr.17,2009 (Fri)

■節約なのです [MISC]

+ 税金支払いにいったらサイフの現金が6万円以上も減った。残業三昧で月収がウハウハだった去年ならともかく、今年はこの程度の支出がホント痛いぜ。

+ しかし予定外の支出があったけれど、先月並の合計支出に押さえられそうな感じではあるな。自炊率を上げて飲食費を押さえているのが地味に効いているのだな。

+ しかし、趣味に使う金を増やすために、より飲食費は減らしたい。そこで家の飲み物もペットボトルで買わないことにしよう。今はアイスコーヒーを常備しているが、手軽さからすると麦茶か烏龍茶あたりにしたいところ。容器を買ってこねば……。

Apr.25,2009 (Sat)

■DVDマルチドライブがへたった話 [COMP]

+ FIENAのDVDドライブ(LITE-ON LH-20A1S)、なんかヘタってきたのかマウントできないDISCが出てきた。まだ2年しか使ってないのだが、ヘタるの早くないか? ……まぁ、文句言っててもしょうがないので、とりあえずLULLIのLITE-ON iHAS120-27と交換して逃げてみた。

+ Blu-rayドライブがもうちょっと安くなったら買おうかなー。

Apr.29,2009 (Wed)

■「Flyable Heart」(ユニゾンシフト) [GAME]

+ 天音→くるり→茉百合→桜子→結衣→すずの→最後と進めておしまい。

+ UNiSONSHIFT Brossomらしい「すこしふしぎ」な設定。そして、「Piece@Peaces」と「ななついろ★ドロップス」と同じように、幸せに終わるけれど、大切な何かを得るために同じくらい大切な何かを失ってしまうような、そんな話ですかな。

+ ネタバレせずに感想を書くのが非常に難しいので細かいことは書かないけれど、フライヤの機能とか晶くんの設定とか、ご都合主義的に過ぎる部分があるけれど、うまく伏線を消化できてまとまっているゲームだなと思う。一方、「ななついろ★ドロップス」のすももシナリオほどじっくり進めるシナリオはない。まぁ、3人しかヒロインがいなくて、その中でもすももが最優先されてたゲームと比べると仕方なくはあるが……。

+ ヒロインは全員わりと好み。桜子も結衣もすずのもかわいいし、くるりの「しょー」や茉百合さんは衝撃の裏モードにも惹かれるが、やはり天音がイチオシだなー。

+ 脇役。マックスはロボのくせにまさに親友ポジションでいいヤツだ。男か女はか決まってないらしいので、晶に恋してあっち\の姿を選ぶなんつー話を見てみたいような気もする。ぐみちゃんも愛玩動物っぽいかわいさがある。

+ 音楽は水月陵でいつもどおりよい感じ。またサントラを買ってしまうのだろう、俺は。

+ 総合的に見ると……演出その他に今までのユニシフに感じない何かを感じるものの、話自体がちょっぴりこぢんまりしすぎな気はするかな。評価は8点。
