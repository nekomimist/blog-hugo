  • 蒼龍改二/141 流星改Lv7/彗星(江草隊)Lv7/彗星一二型甲Lv7/32号対水上電探

+ 夕雲さんがドラム缶積んだままとか適当すぎだな。途中で夕雲さんのキラが切れたので磯波さんに交代。

  • 磯波改/115 10cm連想高角砲x2/22号対水上電探改四

+ こんな感じで出撃。3戦目とボスでPTが出る構成もあるから、すぐにギミック発動するだろうと思って、ACGJKLNの最短ルートで出撃しつづけてたら、7回出撃してもPTがあまり出ずに撃沈数が9止まりで、1回S勝利6回A勝利という体たらくだったので、カっとなってABEHルートで2回出撃してPT小鬼8撃破してギミック発動。

+ その後4出撃で4S勝利してやっと嵐ドロップ。

■艦これ。2015秋イベ総評 [GAME]

+ さて総評。

+ 成果としてはE-1〜E-5全甲突破で甲4となった。ドロップ限定だったグラ子・嵐を取得できた。2015年冬イベに取りそこねた朝霜をゲットできたのも実に嬉しい。あとはまるゆ4三隈1。

+ うちにいる娘で育ててない娘はいないので、誰がルート固定になっても困らないというのもあるだろうが、毛色が違ってちょっと面白かったかな。

Dec.16,2015 (Wed)

■キーボード変更 [COMP]

+ ThinkPad Bluetooth Wireless Trackpoint Keyboardを使っていたのだが、キータッチが好かんのと、トラックポイントをほとんど使わないことに気付いたので、キーボードを変えてみた。

+ 今回はMajestouch MINILA Airの赤軸。正直無理矢理キーを減らした省スペースキーボードはあまり好きではないのだが、そうはいってもテンキーレスでBluetoothでまともなキータッチのキーボードなんてこれくらいしかないのだよね……。

+ 以前Majestouch Wirelessを買ったときは茶軸だったけど、会社で今使ってるのはRealforce91Uだし、軽いキータッチのほうがよいかなと思って今回は赤軸にしてみた。これは悪くない選択だったと思う。

+ で、まず、左Altの位置が左すぎだったので、のどかで無変換キーもaltにマップした。左FnをAltにできるといいのだが、ソフトからは見えないキーなのでしかたがない。

■艦これ。 [GAME]

+ 今月も第三群に入ったので、試製46cm連装砲とネ式エンジンを貰った。

Dec.24,2015 (Thu)

■艦これ。 [GAME]

+ 由良さんの追加ボイスが甘くてすばらしい。

+ お飾り材料集め。南西任務消化ついでに2-5南まわりを十数回ぐるぐるして材料7つ、プレ箱1つゲット。駆逐艦2に見張員5探照灯1積んでトータルで1/2くらいのドロップ率かな。ちょっと引きが悪かったかも。バケツ消費は20くらい。

*1: 由良125/隼鷹124/千歳124/千代田124/磯波115/敷波115
*2: 利根119/筑摩119/千歳124/千代田124/磯波115/敷波115

Dec.26,2015 (Sat)

■艦これ。お飾り材料集め [GAME]

+ ほぼ2-5のみで完了。実際には2-5で30コ、デイリー潜水艦で行った1-5で2コ。プレゼント箱は3つのみ。バケツ減は遠征込みで30ほど。まあ、1週に2つ未満という感じ。

  • 筑摩改二/120 20.3cm(3号)連装砲+7,20.3cm(3号)連装砲,試製晴嵐+7,瑞雲12型+7
  • 隼鷹改二/124 天山一二型(村田隊)+7,天山一二型(友永隊)+7,九七式艦攻(友永隊)+7,熟練艦載機整備員
  • 飛鷹改/124 九七式艦攻(友永隊)+7,天山一二型(村田隊)+7,九九式艦爆(江草隊)+7,彩雲
  • 夕雲改/128 熟練見張員,熟練見張員,探照灯
  • 初霜改二/90 熟練見張員,熟練見張員,探照灯

■2015年ふりかえりと来年の抱負 [GAME]

+ 仕事方面は会社の経営危機の話ばかりでねえ。俺自身も現在の肩書で貰える月俸上限にほぼ辿りついたので、これ以上を望むには現状とは何か変えなきゃいけない。悩みどころだねえ。

+ 何も趣味で"モノ"を作ってないので、何か作りたいところ。Electronベースかなあ。UWPもよいかも。

栞 : 「そう言っててまたelispでちょこちょこやるだけで終わるとみた」
猫 : 「ひどい！」

+ ライトノベルはあまり読んでないなあ。リアルタイムにきっちり読んだのは「陽抜、抜錨します!」シリーズと「この恋と、その未来」シリーズくらい。あとは「スカイワールド」と「疾走れ、撃て！』シリーズはちまちま遅れつつ読んでる感じ。来年もたぶんこの調子。
