■艦これ。EO処理(1-5/2-5/3-5/4-5/6-5) [GAME]

+ 2/11からイベントということは月末にEOを処理している余裕がないかもしれない……ということで5-5と1-6以外を処理した。5-5もメンテまでには処理してしまいたい。

  • 綾波改二/134 10cm連装高角砲+高射装置+6/10cm連装高角砲+高射装置+7/試製61cm六連装(酸素)魚雷+7/Bofors 40mm四連装機関砲

  • 瑞鶴改二甲/146 天山一二型(村田隊)Lv4/噴式景雲改Lv6/Re.2005 改Lv7/彩雲Lv7/増設バルジ(大型艦)
  • 大井改二/132 10cm連装高角砲+高射装置+8/10cm連装高角砲+高射装置+7/甲標的

+ 武蔵は大和と、大井は北上さまと入れかえつつ。摩耶さまは最初は「20.3cm(3号)連装砲+6/90mm単装高角砲+10/零式水上観測機Lv7/13号対空電探改+10/Bofors 40mm四連装機関砲」だったが、やけに大破撤退要因になってくれたので最後の2回はバルジを積んだ。

+ 村田隊と景雲はそれぞれ一回づつ全滅したので、最終的にはLv7ではなくなっている。逆に言えば、そうそう全滅はしないとも言えるかな。橘花改+噴式景雲改による噴式爆撃は、まあ、ザコは落としてくれるのでわりとありがたい。

+ これで9出撃7回ボス到達で6回空母棲姫を撃沈。1ケタで済んだのでまあ悪くないかも?

  • 伊8/68 61cm四連装(酸素)魚雷+8

+ この艦隊だと輸送成功でTPは32減る。4回かかるのはしょうがないなと思っていたら、瑞雲より試製晴嵐のほうがTPの減少量が多いらしいと聞いて、しおいとニムの瑞雲を試製晴嵐に変更。

  • 伊401改/59 試製晴嵐×2
  • 伊26改/95 瑞雲/試製晴嵐

+ で、この装備での3回目の出撃で残りTP36を削ることができた。あとは同じ艦隊でもう一度出撃して、ほぼ無傷で舞鶴に到達してE-1甲終了。

+ 支援はなし。基地航空隊は一式陸攻(野中隊)/一式陸攻二二型甲/一式陸攻二二型甲/銀河。TPを削りきるまではB×2、その後はB/Mとした。基地航空隊を2つに分けたのは、遠征が帰ってきてなかったからの苦肉の策であって、それがたまたま上手くいっただけの感じではある。

+ バケツが1つも減らないどころか、4出撃で食らったダメージがイクが食らった1だけだったりして、過去最大に楽だったE-1だったように思う。

+ さてE-2は丙で藤波を掘ってから攻略かな。

■艦これ。2017冬イベ E-3丙掘り [GAME]

+ さて丙掘り。彩雲輸送は潜水艦6隻で特に悩むことはない。

+ で、ボス叩きはだいたい下記のような感じ。

  • 第一艦隊 摩耶改二/151,金剛改二/146,霧島改二/146,Warspite改/129,隼鷹改二/135,飛鷹改/135
  • 第二艦隊 由良改/134,綾波改二/134,初月改/94,木曾改二/131,大井改二/132,北上改二Lv133

+ 金剛霧島は、比叡改二/146,榛名改二/146とローテーション。由良さんは五十鈴改二/134とローテーション。最初は木曾の代りにPrinz Eugen/133を入れてみたのだが、ボスS勝利安定度がちょっと低かったので開幕の手数重視で木曾とした。装備は改修高角砲をベースとした夜戦連撃装備ばかり。

+ 丙だと低速入れてもボス直行なのでWarspiteさんと飛鷹さんと隼鷹を入れてみた。軽空母装備は艦戦7彩雲1で置物。

+ 基地航空隊は下記の構成でボス集中。

  • 零式艦戦52型(熟練)+10/銀河/一式陸攻(野中隊)/一式陸攻二二型甲
  • 零式艦戦21型(熟練)+10/一式陸攻二二型甲/一式陸攻二二型甲/一式陸攻三四型

+ これで輸送1+S勝利4を1セットとして、3セット目で無事伊13をゲット。合計でS勝利8回A勝利3回。雷巡3にしてからは必ずS勝利だった。

+ さてあとは甲で攻略するのみ。

+ 第一艦隊は戦艦4軽空母2で行くか、摩耶さま+戦艦3+軽空母2とするか悩み中。戦艦はIowa/129を中心に金剛型4人とItalia/142とRoma/142を適当にミックスの予定。甲は高速限定なので軽空母は千歳/135,千代田/135に変更。

+ 第二艦隊は軽巡枠に阿武隈/121、対空枠に秋月改/118を投入。綾波と雷巡3は掘りと同じの予定だけど、あまりボスを削れないようなら木曾を外して鳥海さんを入れるかも。第一艦隊を戦4にして第二艦隊に摩耶さまでもいいかもしれない。

+ というわけで明日は資源回復させつつ甲攻略の作戦を練ることとしよう。

■艦これ。2017年冬イベ感想 [GAME]

+ E-1甲は楽すぎて印象なし。ニムを追加で掘る元気もないしねえ。

+ E-2甲は輸送で攻めきったけど、初期から輸送作戦はあまり嫌いではないせいもあるだろうが、やはり嫌いではない。最後まで輸送でやりきれたのもよかった。藤波丙掘りもさほど苦労はしなかったし……。

+ E-3は丙掘りも甲攻略もさほど苦労しなかったので悪印象はなし。今までの改修と甲突破報酬の積み重ねで、基地航空隊の劣勢調整が楽だったのも大きいのかもしれない。久しぶりに金剛型並べて最終海域ボスに挑めたのもよかった。

+ 総評としては、規模的にも難易度的も悪くなかった印象だな。新艦も3スロ潜水艦×2で運用の幅が広がるし。駆逐艦×2はいつもどおりだけど、1人は初期から待ち望んでいた藤波だったしね。

■艦これ。冬イベドロップ成果 [GAME]

+ レアドロ観点だと、E-1で瑞穂、E-2で天津風、E-3で時津風×2、高波×2、朝霜、照月くらいだろうか。積極的に掘りにいってないので、そんなに数はいない。

+ 津風は金背景だしキャラデザ同じ人だしで伊13と似ていてすごくビクっとした。その勢いでつい解体してたけど浦風砲目当てに牧場してもよかったかもなあ。

+ 天津風はLv20で改だし、新型缶持ってくるしでありがたく牧場させていただきます。

+ 照月はE-3のLで出たけど1%未満らしい。すごい。秋月砲が2つ増えるのも、2人目の照月そのものもありがたい。照月Lv96もさっさと嫁にしないとなー。

+ 高波も改で22号電探・13号電探・見張員といいモノを持ってくるのでありがたく牧場させてもらう予定。嫁高波はLv112になったし、改じゃない高波も持っておきたいしねー。

+ 朝霜はのんびり牧場する予定。改Lvが45なので本来は牧場には向かんのだけど、ちょうど手持ちの牧場朝霜改が誰かの餌になったばかりなので丁度いい :-)

+ 伊26や伊401をE-1で掘るのもよかったかもしれないが、その元気はなかった。
