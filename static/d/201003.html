Mar.4,2010 (Thu)

■自作ロードバイクの事 [BIKE]

+ 今のAbios Tourerはフルアルミだし、安いカーボンはアレだし、懐古趣味的な興味からはクロモリでホリゾンタルなフレームに興味がある。手に入れやすいのは国内のブリヂストンかパナソニックあたりで、前者ならRNC3、後者ならFRC23なら手が届くが、さて。

+ コンポも悩む。Shimano 105がいいのだが、モデルチェンジを控えている今現行のモノを買うのはどーも気が進まないし、新しいのが手に入るのは5月以降あたりだろうし、これまた悩ましい。

+ ま、PCと同じで悩んでいるうちが一番楽しいよな :-)

Mar.7,2010 (Sun)

■きょうのよしなしごと [MISC]

+ 有休くっつけて3連休だったのだが、金曜にちょっと無理して走ったのが祟ったのか、土日は体調不良で寝て過してしまった。まぁ、天気も悪かったので走りには行けないのだが、いくらなんでももったいない時間の過ごしかたであった。

■自転車のこと [BIKE]

+ 自転車。Anchor RNC3フレームを買う方向でほぼ決断。色はイリュージョンターコイズにちょっと惹かれるものがあるが傷ついたら泣きそうなのが難点だ。んで、これでコンポをAll Tiagraにしちゃうと完成車(RNC3 Sport)そのものになって悲しいので、やっぱ105狙いで行くべきだな。

+ しかし、約20年前に乗ってたTravezone(ランドナー)もRADAC(ロード)も、3年前から乗ってるAbios(クロス?)も全部ブリヂストンなのに、次にAnchorを買っちゃうとまたブリヂストンじゃないか……。

Mar.15,2010 (Mon)

■きょうのよしなしごと [MISC]

+ 最近ジョギングで脚のつけ根が痛くなるようになって困っている。40分くらいでいい汗をかくにはちょうどよい運動だったのだが……。回復には走らない以外の解がなさそうなので、しばらくジョギングは休んで踏台昇降で行くことにしよう。

+ 体重がやっと79kgを切りそうなあたりに来た。忙しくなって夕食時間が後にズレた上に運動頻度が下がってペースは大幅に落ちたので、6月までに75kg達成は厳しそうだが、まあ、減ってるからいいとしようか。

+ それにしても、身につけるものがどれもこれも大きすぎて困るようになってきた。服やら腕時計やらがとにかくデカい。たぶん夏用の指なし自転車グローブも緩いだろうな。痩せて出費が増えるというのは嬉しくもあり悲しくもあり。

■PCと自転車で散財する話 [COMP][BIKE]

+ ひさびさに残業が増えてストレスが溜まっている。んで、食欲を抑制しているせいか、物欲がほとばしってしまい、DDR2-800で2GBのメモリ2枚とIntel EXPI9301CTを1枚買ってML115G1に入れてしまった(*1)。こうなったら2TBのHDDを3台足してzfsのraidzにしちゃうぞー。

+ しかし、ロード組み立ての方も調整して買わねばな。先月安ホイールとチューブとタイヤを買って、今月安工具を買ったところで小休止中だし……。ま、資金に余裕がある今月中にフレームの発注をして、残業代が減って固定資産税支払いのある来月に安いHDDを3台、5,6月あたりにハンドル・サドル・ドライブトレイン一式を購入って所だろうか。夏ボまで断続的に出費が続くなぁ……。

*1: ちなみにIntel EXPI9301CTのBOOTROMを殺しておかないとML115G1が起動しないので注意。詳細はググれ。

Mar.23,2010 (Tue)

■iPhone3GSトラブルの巻 [COMP]

+ 先週末の話。iTunesと同期が突然失敗した→iPhoneを見るとハングしているように見える→しょうがないのでひっこ抜いてリセット→アップルマークのままで止まって正常起動しない→リカバリーモードに入れて復旧を試みる→復旧シーケンスが進まず復旧できない、という感じで雪崩れをうつように状況が悪化していった。

+ アップルサポートに電話して、いくつかのアドバイスをうけたものの結局解決しなかった。近所のソフトバンクショップに行こうかとも思ったけど、一発でケリをつけたかったし、祝日だし行楽気分で一度行ってみるのもイイかなと思って、心斎橋のApple Storeに行ってGenius Barでサポートを受けてきた。

+ んで、近鉄一本で難波まで出て、ちょっと歩けばあっという間にApple Store着。Genius BarでMacからのリカバリを試行してもダメだった時点で、サクっと良品に交換してもらって終了。保護シートの張りかえまでやってもらってわずか15分で対応完了。サポートのお姉さんと会話できてちょっと嬉しかったりも(←キモい)。

■QuietComfort15とQuietComfort3 [AUDIO]

+ せっかくApple Storeに行ったので、展示されていたBoseのQuietComfort15とQuietComfort3を試してみた。

+ QuietComfort15は無音時でもほとんどノイズレスでびっくり。以前使っていたSONYのカナル型の安物とはえらい差だ。まあ3万9900円と値段にもえらい差があるが :-)

+ しかしながらQC15は持ち歩くにはあきらかにデカい。その点ではQuietComfort3がよいのだが、世代が古い分か無音のノイズの量はQC15より多かったし、そもそもQC15より高いわけでなかなか厳しい。

Mar.26,2010 (Fri)

■きょうのよしなしごと [BIKE]

+ 残業代で給料が多めだったのと、円高でちょっとだけ差益が出たのもあって、えいやっとRNC7フレームを発注してしまった。4月末にはモノが入ってくるだろうか。楽しみである。

+ あと買う必要があるのはハンドルとステムとサドルとシートポストとペダルとコンポ一式だろうか。コンポ一式はNew105を買うつもりだが、場合によってはUltegra+現行105の混成も考えておこう。ハンドルとステムとサドルとシートポストとペダルは、フレームがRNC7なので超軽量を狙ってもしょうがないが、重すぎるのも何なので、ほどほどのレベルのものを買っていかねばいかん。
