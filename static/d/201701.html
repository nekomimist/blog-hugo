■謹賀新年 [MISC]

+ あけおめことよろ。

■golang [COMP]

+ 年を跨いでgoでふにふにとプログラムを書いてみている。こいつは楽だな。SSEとか使う必要がない限り、支援ツールはCやRubyじゃなくてgoで書いてもよさそうだ。

+ 主に使うエディタはEmacsなので

  • go-mode
  • go-autocomplete
  • go-projectile
  • flycheck
  • quickrun

+ などをpackage.el経由で入れた。Windows環境でもわりと普通に使えるのはありがたい。が、もっと速いCPUが欲しくなった。Core i7-4790kを圧倒するCPUが早く出ないかなー。

■艦これ。新年 [GAME]

+ 晴れ着(?)鳳翔さん。なにこの良妻っぽい感じ。中破で肌色がほとんど増えないくせにむっちゃ色っぽいのがまたよい。うちの鳳翔さんはLv130だから立ち絵も指輪しててくれたらよかったのに……。

+ 秋津洲は晴れ着が更新されたけど、なんでかな? ……大艇ちゃんの尾翼が「2016」だったからか。去年とボーズが同じで差分っぽいが、落ち着いた色でこれもまたよし。

+ とりあえず新年任務は2つ目まで終わった。いつものことだが最初の任務の1-3が強敵だった。3つ目の4-2と4つ目の3-5は明日以降ウィークリーにひっかけてやる予定。

Jan.2,2017 (Mon)

■艦これ新春任務つづき [GAME]

+ 新年任務その3。航戦旗艦で航戦2航巡2で4-2。敵は羅針盤だし、日頃戦っている羅針盤なので特に悩まず対空カットイン装備の山風と制空をちょっと盛ったAquilaを入れて突入。わりとボスに行ったのでさっさと終わった。

+ 新年任務その4。空母旗艦で駆逐2を入れて3-5。最近は3-5は重3装空3でやってるので駆逐艦が脆いのが問題。ただ、せっかく駆逐艦を入れるのだからと秋月型2人を入れて、カットイン発動時に固定撃墜が発生したスロットは全滅させられるのを狙ってみた。

+ 結果としてだいたいこんな構成になった。これで3回出撃して3回ボスS勝利だった。任務完了後にこの艦隊でもう1回出撃して3-5のゲージ破壊もできた。

+ 道中のヲ級と北方棲姫のスロット全滅は2回〜3回あったけど、ヲ改を沈黙させることはできなかった。まあ、開幕爆撃事故抑制効果はたぶんあっただろうから悪くない。重巡3装甲3と違って三式弾担当がいないので北方棲姫をあまり落とせないのがデメリット。

Jan.6,2017 (Fri)

■艦これ。今月の6-5 [GAME]

+ 補強増設にバルジと機銃が入るようになったので、ちょっと装備を変えてみた。

+ 最初の4回は下記の艦隊。顔ぶれは概ね先月と同じ。大和と武蔵は適宜ローテーション。

  • 大潮改二/125 10cm連装高角砲+高射装置+7x2/Bofors 40mm四連装機関砲/応急修理要員
  • 武蔵改/142 46cm三連装砲+3x2/九一式徹甲弾+6/零式水上観測機Lv7
  • 北上改二/133 10cm連装高角砲+高射装置+7x2/甲標的/Bofors 40mm四連装機関砲
  • 摩耶改二/151 20.3cm(3号)連装砲+6/90mm単装高角砲+10/零式水上観測機Lv7/13号対空電探改+10/25mm三連装機銃 集中配備
  • 大鳳改/149 流星(六〇一空)Lv7/零式艦戦53型(岩本隊)+7Lv7/烈風改Lv7/Fw190T改Lv7/増設バルジ(大型艦)
  • 瑞鶴改二甲/146 天山一二型(村田隊)Lv7/烈風改Lv7/零式艦戦52型(熟練)+10Lv7/彩雲Lv7/増設バルジ(大型艦)

+ 大潮と北上さまで艦隊防空を稼いて、大和・大鳳・瑞鶴の個艦防空を補う戦略。北上様は10cm連装高角砲+高射装置+7×2で個艦防空はわりと充分なのだが、他に積むものもないので機銃を積んだ。

+ この艦隊で4突入して4ボスS勝利。最後はさらに道中が厳しくなるので、破損が大きい状態でボスに行く可能性を考えて駆逐艦の火力を増したいと考えてみた。

  • 綾波改二/133 10cm連装高角砲+高射装置+7x2/試製61cm六連装(酸素)魚雷+7/Bofors 40mm四連装機関砲
  • 大和改/142 46cm三連装砲+3x2/九一式徹甲弾+6/零式水上観測機Lv7
  • 大井改二/132 10cm連装高角砲+高射装置+7x2/甲標的/Bofors 40mm四連装機関砲
  • 摩耶改二/151 20.3cm(3号)連装砲+6/90mm単装高角砲+10/零式水上観測機Lv7/13号対空電探改+10/25mm三連装機銃 集中配備
  • 大鳳改/149 流星(六〇一空)Lv7/零式艦戦53型(岩本隊)+7Lv7/烈風改Lv7/Fw190T改Lv7/増設バルジ(大型艦)
  • 瑞鶴改二甲/146 天山一二型(村田隊)Lv7/烈風改Lv7/零式艦戦52型(熟練)+10Lv7/彩雲Lv7/増設バルジ(大型艦)

+ 駆逐の連撃では空母棲姫の装甲は抜きづらい(*1)。大潮は夜戦火力No.5だし、夜戦火力No.1の夕立に変えても同装備なら5%も変わらない。というわけけで綾波さんの砲雷カットインとした。これなら夜戦火力は245×2で大和型と同じで悪くないし、運61だし旗艦効果もあってかなりカットインは出るはず。

+ これで4回出撃して2ボスS勝利。4戦目で大井さん1発大破と2戦目摩耶さま1発大破と事故が2回あったけど、ボス戦の感触はよかったし来月もこんな感じでやってみたい。

+ あと装甲空母二人は艦隊の3番目4番目に上げてよさそう。来月はそれでやろう。

*1: 空母棲姫の装甲はおそらく180だが、上記大潮の装備でだと夜戦火力は203×2。
