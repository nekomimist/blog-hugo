Sep.24,2013 (Tue)

■修羅場を抜けた話 [MISC]

+ いやもう、なんか去年10月からの入った開発が修羅場って、5月以降日記を書く余裕がまったくなかったけど、まぁ、なんとか終わりつつあるので、ゲーム系の話から再開。

■艦これ約1ヶ月 [GAME]

+ リンガ泊地開設当日(8/17)に開始して1ヶ月ちょい。2-4突破が9月6日、3-2突破が9月22日で、現在は3-4/4-3を攻略中。のんきに育てているせいか、進みは遅い方かと思う。

+ 2-4は主力が羅針盤に嫌われて入渠状態で、レベリングのつもりで適当な編成で2-4-1に行ってそのまま進軍してみたらあれよあれよとボス直行→勝利。しかし今見るとこの編成ちとおかしいな :-(

+ 3-2は島風LV31+暁LV31+響LV31+雷LV31+電LV31+磯波LV32で攻略。羅針盤に嫌われて北に流されることが多くて難儀したが、最後は電大破でダメコン頼りで進軍してドキドキしたよ! 結局ダメコン発動もなくS勝利だったけど。

+ 今は現行の主力で攻略を進めつつ、新規にドロップした主力になれる艦はレベリングして主力に投入し、駆逐艦たちで1-1をまわしてキラキラさせて遠征に出すのをくりかえしている。まだ出てきていない艦娘も多いのでもうちょい楽しめそうな感じ。

+ 3-4/4-4を超えるにはもちっと強化しないとダメそうなのが悩ましい。

■我が司令部の艦娘状況 [GAME]

+ 最初の秘書艦が電なので初駆逐艦は電。キラキラ準備と遠征だけで改造を迎えた娘も増えてきたのでけっこう愛着が強いですな。漣とか睦月とかキャラ的に大好きです。

+ 軽巡。初getは多摩。8月のE-1用にあわてて五十鈴と由良を対潜用に育てた。でもリンガ泊地組は一週間しかなかったのでE-1超えられなかったよ……。今は長良・名取・球磨あたりのLv上げ中。あとは遠征引率で天龍と龍田と夕張が勝手に育ってる感じ。もちろん軽巡から雷巡となった北上・大井コンビは主力。北上さんがやっとハイパー化したので大井っちもハイパー化しよう。

+ 重巡。初getは妙高だけど、今は鳥海・摩耶が主力。重巡は4-3攻略に有用だけど、他のメリットがあまりなかったので育ってないねぇ。この前のメンテで夜戦が強くなったのはよいけど、あとはもうちょっと燃費と入渠時間が優遇されるとよいのだが。

+ 軽空母。初getは祥鳳だけど、今は飛鷹・隼鷹・千歳改二・千代田改二でローテーション。あと一部遠征用に鳳翔さんとちよちと甲を維持。瑞鳳ほしいですが、まぁ、建造で狙わずドロップで頑張るということで。

+ 戦艦。初getは伊勢で、今は金剛・比叡・榛名・霧島と航戦となった伊勢・日向で回している。金剛級が来るまで主力だった扶桑・山城も改二が来そうなので最近レベル上げ中。長門と陸奥? いえ、知らない子ですね……。最近ちと火力不足なので46cm砲開発に勤しんでいるけど、あまり出ないので泣ける。35.6cm砲つけてる艦娘も多いので、41cm砲でもよいのだがな。

+ 正規空母。任務報酬の赤城が初getで、それ以降提督Lvが70になっても赤城のみだったけど、先日ドロップで蒼龍が建造で瑞鶴が出たので育成して主力に合流中。しかし一航戦も二航戦も五航戦も一隻ずつってのは何の呪いだ :-) 正規空母と軽空母が充実した分だけ艦載機が足りなくてツラい。特に艦戦が足りない。

+ 潜水艦は一般解放後にぽろっと出てきたイムヤが3-2-1被害担当として頑張ってます :-) ゴーヤさんも欲しいです。

Sep.27,2013 (Fri)

■艦これ [GAME]

+ 五十鈴の改二化と大井っちの改二化が終了した。

+ 北上さんは先日ハイパー化してたので、これでこれからの攻略に必要になりそうな、北上大井ハイパーコンビが完成である。ところで北上さんの「ありがとねっ」はじつにかわいい :-)

+ 五十鈴は改二になったらえらくキリっとした感じになった。もともと対潜用に育てていたのでありがたい。さておき、五十鈴をLv29からLv50に上げるのに、3-2-1レベリングや2-2/2-3/3-1/4-1/4-2巡回と演習にさんざんつれまわしていたのだが、夜戦強いし食らわないし燃費いいし良いね軽巡。やはり長良と名取も球磨も育てておこう……。

+ さて、今回の二人のレベル上げで主力級がLv50〜70になったし、バケツも資材もまあまあ溜まったので、そろそろ攻略を進めよう。4-3→4-4→3-4→5-xだな。

Sep.28,2013 (Sat)

■艦これ 4-3攻略 [GAME]

+ 4-3終了。重巡二隻入れて経路固定しても最初の羅針盤でハズれを引いて事故ったりしたけど、まぁ、3度目で攻略。顔ぶれは 北上改二Lv57/金剛改Lv68/榛名改Lv49/赤城改Lv71/摩耶改Lv39/鳥海改Lv45。摩耶さま中破、赤城・鳥海小破でB勝利でした。

+ さて次は4-4。駆逐艦2と重巡1がいると経路固定ができるようなので、その方向で。重巡1は4-3にも使った摩耶さま、駆逐は島風・響で挑む予定。

+ フラ戦エリ戦相手が厳しくなってきてよく事故る。食らうのはしょうがないけど当たらないのは厳しい。うちは金剛級4隻(金剛Lv70/比叡Lv53/榛名Lv50/霧島Lv60)と航空戦艦(伊勢Lv70/日向Lv55/扶桑Lv50/山城Lv50)しかいないので火力不足なのよねー。
