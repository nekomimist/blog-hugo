  • 青葉型 青葉Lv94/衣笠Lv140

  • 最上型 最上Lv98/三隈Lv122/鈴谷Lv98/熊野Lv98
  • 利根型 利根Lv97/筑摩Lv97

Dec.12,2014 (Fri)

■艦これ。 [GAME]

Dec.21,2014 (Sun)

■艦これ。 [GAME]

+ 出撃数が50000を超えて、司令部Lvもそろそろ120という状況。先月も500位以内に入れたけど、今月はどうかなー。

+ やはり改二は足柄さんだったか。うちの足柄さんは那智さん同様Lv94で改二に問題は全くないけど、プリンツ・オイゲン(*1)以外の全ての重巡はこの二人より上のレベルなんだなこれが。飽きて止める前に高雄型と最上型の改二が来てほしいものだなー。

+ まるゆ改が5人揃った。今回も改二が来なかった鳥海さんにプレゼントだなこれは。

■FIENAのシステムドライブ換装 [COMP]

+ FIENAのシステムドライブを東芝THNSNH256GCST(HG5d)から、SanDisk SSD Extreme PRO 480GBに変更。256GB→480GBなので広くなって満足。

*1: Lv80。

Dec.23,2014 (Tue)

■艦これ。 [GAME]

+ 司令部Lvが120になってカンストした。我ながら廃人よねぇ。

+ まるゆ改が5人揃ったので、今回も改二が来なかった鳥海さんにプレゼントして運23になってもらった。あと5人いるまるゆも鳥海さんにプレゼント予定。まるゆがもっともっと欲しいし、また大型建造するかなー。

+ 足柄さんはLv95なので何の問題もなかろう。とりあえず加古と青葉(現在Lv96)とプリンツ・オイゲン(現在Lv84)のLv99到達のほうが優先である。

Dec.27,2014 (Sat)

■艦これ。 [GAME]

+ 足柄さんは改二はすげぇ「足柄さんっぽい」感じ。持ってくる装備が2号砲×3で笑った。これで2号砲も9本になった。ありがたい。

+ 吹雪はLv62からLv66に上げた程度。育成すべき駆逐艦が多くて手が回ってないので、このあたりで足りるとありがたい。夏イベ艦の磯風がLv44でやっと改が見えてきたところ。春雨・時津風・早霜・清霜は改になってLv33〜34付近。秋イベ艦は秋月はLv62になってるけど野分がLv28でまだまだだ。個人的な気分としては、秋月とは早めにケッコンカッコカリしたい。

Dec.28,2014 (Sun)

■艦これ。 [GAME]

+ 気が向いたので1-5で山雲を掘ってみた。結果はボスS勝利24回目でドロップ。まあまあか。

+ これでうちにいない娘はビスマルクと朝雲の2人。秋イベで朝雲が掘れなかったのは痛かったなぁ……。

Dec.29,2014 (Mon)

■艦これ。 [GAME]

+ 水上反撃部隊2-5任務。礼号作戦リスペクトな任務やね。

+ 史実では、霞・足柄・大淀・清霜・朝霜・榧・杉・樫。艦これ実装済みなのは最初の4人。しかしながら、うちの清霜さんはまだLv34なので、代打として姉の夕雲さんを投入。残りは初戦の事故を避けるために火力上等な駆逐二人で編成。

  • 霞改 Lv64　10cm連装高角砲,10cm連装高角砲,33号対水上電探
  • 足柄改二 Lv96　20.3cm(2号)連装砲★2,20.3cm(2号)連装砲,紫雲,32号対水上電探
  • 大淀改 Lv90　20.3cm連装砲,九八式水上偵察機(夜偵),紫雲,20.3cm連装砲
  • 夕立改二 Lv97　10cm連装高角砲,10cm連装高角砲,33号対水上電探
  • 夕雲改 Lv123　12.7cm連装高角砲(後期型),12.7cm連装高角砲(後期型),33号対水上電探
  • 綾波改二 Lv123　61cm四連装(酸素)魚雷,61cm四連装(酸素)魚雷,33号対水上電探

+ 司令部Lv120なので索敵は火力を損わない範囲で最大限を目指してみた。持っててよかった紫雲×2と33号電探。制空権を絶対取れない以上、大淀の夜偵は2戦目以外意味ないわけで、照明弾あたりでよかったかもしれない。

+ 初戦はフラ雷巡なし構成を引いて無事故で切り抜け、二戦目は夕立が連撃食らって中破たのみで、無事ボス到達。……味方がこの構成だとフラ戦1エリ戦2って極悪だなぁ。まあ無事にS勝利できたけど。
