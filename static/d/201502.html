Feb.4,2015 (Wed)

■艦これ [GAME]

+ 2-5を水上反撃部隊でS勝利したあと、任務にチェック入れてなかったことに気付いて盛大にヘコむ。その後10回くらい大破撤退した後にやっと達成。この任務だとフラ雷巡編成引いたらほぼ撤退確定だよねぇ。

+ 3-5は例によって雷巡1軽巡2潜水艦3で削った。1回目ボス到達時に全員ボコボコにされてD敗北した以外はいつも通りで、全体としては1/2〜1/3程度の率でゲージを削れる感じ。のこり一回は月末にやろう。

Feb.6,2015 (Fri)

■艦これ。冬イベE-1 [GAME]

+ 当然難易度"甲"を選んだぞ。さて、どこまで行けるかな。

+ E-1は対潜掃討ということで軽巡駆逐。とりあえず、由良Lv119・初春Lv81・ヴェールヌイLv104・初霜Lv74・吹雪Lv76・潮改二Lv78で突入。装備は全員三式爆雷投射機+三式水中探信儀に、残り1スロは由良さんは紫雲で駆逐艦は33号電探。

+ 由良さんは五十鈴Lv119に変えたりもしつつ、駆逐艦の顔ぶれは固定。7回出撃5回ボスS勝利2回大破撤退(*1)で終了。それにしても五十鈴と由良さんが対潜戦闘していると、2013年夏E-1を思い出すなぁ……。

+ 開始時が油266424/弾257318/鉄299956/ボーキ267299/バケツ1553で、終了時が油266150/弾257561/鉄299952/ボーキ267343/バケツ1543。減ったのほぼバケツだけだな。

+ さて、E-2とE-3は明日でいいので先にウィークリーをかたづけちまおう。

■艦これ。バレンタインネタ [GAME]

+ 睦月・如月・時雨・浜風・叢雲にバレンタイン専用グラフィックあり。全員かわいくてよい感じである。

+ そして鳥海さんと摩耶さまと由良さんにバレンタイン専用ボイスがあって大歓喜。金剛型にもあるらしいが、まだ聞いてない。衣笠さんと三隈さんにも欲しいけど無理っぽいのがなぁ……。

+ あ、磯波さんにもバレンタインボイスがあった。すばらしい……。

*1: 左回りの道中単縦潜水艦と、右回りの道中水上艦+潜水艦混合艦隊で、1回づつ。

■艦これ。2015冬イベ終了 [GAME]

+ E-5甲で空母棲鬼の所まで行く掘りをやってたが、成果はまるゆ1三隈1だけと大変マズい感じであった。疲れたのでこれで終わり。前回の朝雲同様に朝霜は諦めよう。

+ さて、今回の振り返り。

+ Bismarckがいたら、第二艦隊の陣容にもうちょっと自由度が出ただろうか。夜戦火力としては事実上妙高改二や羽黒改二より下になる(*1)けれど、昼の安定度が上がるであろうことは想像に難くない。再び大型建造に手をつけますか……。

+ あとは41cm連装砲と35.6cm連装砲は★5をもうちょっと作っておけばよかった。だいたい500位以内を維持しているので、試製砲はいっぱい持っているけれど、第一艦隊と道中支援と決戦支援の戦艦を賄うには足りなかった。3号砲と2号砲の★4ばかり作っている場合ではなかったのだ!

+ とはいえ3号砲/2号砲の★4以上がいっぱいあったおかげで、重巡・軽巡・雷巡の装備は悩まずには済んだので悪くはなかった。

+ MVPはなんといっても扶桑山城改二。次点は渋い働きをずっとしてくれていた神通さんと初霜改二かな。

*1: 46cm三連装砲+徹甲弾なら上回るけど、フィット補正がある今そんな装備には絶対しない。
