Mar.1,2014 (Sat)

■艦これ。正規空母たち [GAME]

+ 5-4巡回で加賀さんがついにLv98に突入した。大鳳もLv93になった。難海域に空母入れるときは加賀さん&大鳳がクセになってるし、MVPをよく取るのでレベルがガンガン上がる。卯月は出ないが。

+ 中破でも艦載機を飛ばせる大鳳はともかく、加賀さんより回避が高い翔鶴瑞鶴を活用してもよいと思うので、もうちょっと烈風を作るかなぁ……。

Mar.3,2014 (Mon)

■艦これ。 [GAME]

+ 週末に5-4巡回で卯月がドロップした。「ウザかわいい」感じだねぇ。結局5-4は30回くらいやったのかな? 銀背景→「おお!」→隼鷹or龍驤orゴーヤ→「あー……」ってのが多かったなぁ。

+ さて、これで全員揃ったことになる。あとは大和を改造すれば埋められるところは全部埋めたことになるかな。さあこい海外艦。まあ、鳥海と衣笠とケッコンカッコカリした俺としては、第一次ソロモン海戦参加艦で唯一実装されていない夕凪の実装も期待したいところだが……。

Mar.4,2014 (Tue)

■艦これ。駆逐艦 [GAME]

+ 卯月をレベル上げのために旗艦に置いて南西消化のために2-2巡回をする(*1)。実にウザい。そしてウザいのがかわいい。睦月も同方向のかわいさなのだが、より強化されている感じ。

+ そういや敷波に放置ボイスが追加されてっけ……と聴いてみると、おお、なんという破壊力。敷波は一度主力に組み込んでレベルをLv50まで上げてから、なんとなく遠征組に戻したんだけど、やっぱもう一度主力に戻そうかなー。

*1: 3-2-1で高速育成とか大嫌いなんでね。

Mar.5,2014 (Wed)

■艦これ。重巡 [GAME]

+ 低レベルだった航巡が5-4巡回のドラム缶運搬役でメキメキレベルが上がってしまう事態になったので、負けずに低めのレベルの重巡も鍛えなければならぬ。

+ 鳥海Lv116と衣笠Lv112がいるのだから、古鷹Lv53と青葉Lv53と加古Lv51もカンストまでもっていって、三河艦隊の重巡全員とケッコンカッコカリとかしてみたい気もする :-)

Mar.7,2014 (Fri)

■艦これ。[GAME]

+ 5-4巡回は航巡+重巡×2+戦艦+空母×2に変えた。雷巡は装甲が低いのでわりと事故要因になりやすいし、あまり雷巡のレベルも上げたくないし、ちょうどよいかな。

+ 大和がLv60を超えた。演習しかしてない大和だけど、最近近代化改修の適用先がなくて悩むくらいの状況だし、弾2500鉄3000使っても改造すべきだろうなー。

+ 春イベントか。戦力は概ね揃ってるので、装備開発をもうちょいやるか。

  • 戦艦・航戦に2つづつ積めるくらい46cm砲がほしい。
  • 戦艦・航戦に2つづつ積めるくらい32号電探と14号電探が欲しい。33号でもいい。
  • 重巡全員に15.5cm副砲×3を積めるくらいほしい。
  • 重巡全員に20.3cm(3号)砲が欲しい。三隈掘るしかないから無理だが。
  • 主力空母に2つづつ積めるくらい烈風が欲しい。
  • 主力駆逐と主力軽巡に行き渡るくらい三式ソナーと三式爆雷が欲しい。

+ こんなもんか。副砲以外つらそう……。

Mar.9,2014 (Sun)

■さいきんのわたし [MISC]

+ 仕事が検討段階なので早く帰れるせいか、酒飲んで艦これやってばかりである。

■艦これ。ケッコンカッコカリ [MISC]

+ ケッコン済みの鳥海と衣笠はLv117とLv113となった。Lv117→118には26000が必要なので、Lv93→94よりすでに多くなっているのだな。こわいこわい。

+ Lv99未到達組筆頭はLv98の加賀さんで、あと経験値60000くらいでLv99になる。ああ課金せねば……。

+ 加賀さんを追うのはLv97の4人。摩耶さまがあと一回の出撃でLv98で筆頭、赤城・伊勢がデットヒートで、ちょっと離れて金剛という状況。このへんも順次Lv99になっていく気がする。

Mar.10,2014 (Mon)

■艦これ。[GAME]

+ ひさしぶりにランキング500位以内で推移している。リンガ泊地サーバは戦果がゆるめとはいえ、なんてことだ。

+ 気がつくと大鳳がLv96になってLv1順1ページ目に入りしている。しかし、大鳳を使ってて思うのは回避が低いこと。加賀赤城蒼龍に比べて10低く、翔鶴瑞鶴飛龍に比べて20低いせいかカスダメが蓄積しやすいなぁ。体感だけど。

Mar.13,2014 (Thu)

■艦これ。[GAME]

+ 加賀さんはLv99まで経験値1万を切った。摩耶はあと10万ほど。5-4巡回は航巡・重巡・重巡・戦艦・空母・空母で回っていて、どちらも順調に経験値が入っていっているので、加賀さんは今週中に、摩耶さまも来週にはLv99到達だろうか。

+ そして赤城さんもLv98に到達。大鳳もLv97直前だし、そろそろ5-4につれていくのを蒼龍L94・飛龍Lv92・翔鶴Lv93・瑞鶴Lv95や、千歳千代田飛鷹隼鷹のLV93軽空母陣にに切りかえていかないとアレだな。

+ 弥生がかわいい。見た目も言動も何もかもが好み。ここは夕雲・綾波と共にLv99を目指すかのう……。

Mar.15,2014 (Sat)

■艦これ。[GAME]

+ 加賀さんとケッコンカッコカリした。うむうむ。

+ 伊勢・金剛が本日Lv98になったので、残りのLv98勢は摩耶・赤城・伊勢・金剛。

+ Extra Operation 1-5は夕雲Lv86・綾波L86・由良Lv82・扶桑Lv89で5回行って5回で終わり。今回はなかったけど、今後は1戦目エリ戦に撃ちぬかれるケースも出てきそうな雰囲気。

+ Extra Operation 5-5はとりあえずは行こうとも思わんな。資源とバケツが余るほどになったら行こう。

Mar.18,2014 (Tue)

■艦これ。[GAME]

+ レーベを旗艦にして、ビスマルク狙いで4000/6000/6000/2000/20で大型建造。4時間で榛名だった。不幸であるが、まあ、また一週間に一回づつくらい回そう。通常建造のレア駆逐レシピ8回でマックスが出たのでそう不幸ではあるまい。きっと。

+ 母港枠が厳しくなってきた。アルペジオコラボイベント開始時から増えたのは、長波・しおい・大鳳・まるゆ・あきつ丸・大和・弥生・卯月・レーベ(Z1)・マックス(Z3)で10人。そりゃ狭くもなるわな……。

+ 駆逐で一番好きな夕雲さんと綾波さん(*1)がLv89になった。Lv99目指してまだまだ行くよー(白目)

+ ケッコンカッコカリ済み勢は、鳥海さんがLv120に、衣笠さんがLv116に到達。さすがに高雄Lv82・愛宕Lv84と比較すると、同じ装備でも当てる率が高く食らう率が低くなっていると感じるな。ちなみに加賀さんはLv100のまま。上げ出すとぐわっと上がるのはわかっているので、もうちょっと鳥海・衣笠のレベルが上がってから上げにいこう。

*1: 二人いることにはつっこんではいけない。

Mar.20,2014 (Thu)

■艦これ。[GAME]

+ 摩耶さまがLv99になったのでケッコンカッコカリした。わりと追加セリフがデレ系だねぇ摩耶さま。これで鳥海Lv120・衣笠Lv116・加賀Lv100・摩耶Lv100とケッコン組みが4人となった。

+ どうも大火力高装甲艦のレベルが低めなのが気になる。陸奥Lv85・武蔵Lv85・長門Lv78・大和Lv73。扶桑・山城だってLv90だというのにね。5-5の攻略を見据えつつ底上げを進めよう。

Mar.21,2014 (Fri)

■通勤用自転車 [BIKE]

+ 新しい通勤用自転車が来るのは4月になりそうだ。24年ぶりのランドナー回帰。

+ 24年前の大学1年のときですら、メジャーメーカーのランドナーはブレジストンのトラベゾーンか丸石のエンペラーか二択だったはずだ。当時はトラベゾーンに乗っていたが、今はもうトラベゾーンは売ってない。せつないねぇ。

+ しかし丸石は今もランドナーを売っている。ダブルレバーだし、クイルステムだし、ヘッドパーツはスレッドだし、実に懐古趣味的である。値段が高すぎないのもまたいい。がんがん輪行して乗ってどんどん痛む自転だしね、ランドナーって。

■艦これ。 [GAME]

Mar.27,2014 (Thu)

■艦これ。[GAME]

+ 46cm砲は戦艦系全員に1本づつ載せられる12本はあるのだが、もうちょっと欲しい。烈風も11個あるが、流星改が6個しかなくてちょっと足りない。電探は32号・33号がだいぶ増えたが14号は1つしかない。……やはり46cm砲と艦載機と電探の開発は続けないとダメやね。
