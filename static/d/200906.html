Jun.1,2009 (Mon)

■HDDのRMA申請の話ふたたび [COMP]

+ 年末に壊れたWD10EACSのRMA申請をまだしてない事を思い出した。最近はヒマだから次の有休の時にでもやっとくかね。1TBのHDDが安いとはいえ、RMA申請するほうがさらに安いしなー。

+ しかし1TBが戻ってきたとしても使い所が難しい。一番容量が欲しいWindows Home ServerのLULLIは1TBを4台積んでて3.5インチベイがいっぱいなのだ。1.5TBや2TBなら1台交換すれば総容量が増えるのだが。まぁ、SATA端子はまだ余ってるので5インチベイにでも入れればいいか。

■すきすきBazaarの巻 [COMP]

+ Bazaarがイイ感じなので、手元のsubversionレポジトリを撲滅してTortoiseSVNを消そうかなーとか思ったのだが、Baraar 1.14.1のWindows Setup版付属のbzr-svnではSubversion 1.6.x系のレポジトリ/ワーキングコピーをうまく取り扱えない感じ。仕方がないので、svnadmin dumpして、svnadmin create --pre-1.6-compatible(*1)して、svnadmin loadして作った旧形式svnレポジトリから、bzr svn-importすることで回避。

+ 自宅は上記の通りだが、職場はやっとCVSからSubversionへの移行が始まりつつあるという感じで周回遅れ気味である。ま、CVS時代はCVSNTにさんざん苦しめられたが、これで楽になるだろうか。中央レポジトリがSubversionならbzr-svn経由でBazaarが問題なく使えるので、ちょっと幸せになりそうな予感がする。

*1: 1.5にしたかもなぁ。

Jun.8,2009 (Mon)

■DELL XPS M1330にWindows 7RC x64をインストールした話 [COMP]

+ Intel OEMのPQI X25-M(6526-080GR101B)を買ったので、DELL XPS M1330の内蔵HDDと交換してWinows 7RC x64をインストールしてみた。なんでノートPCにx64かっつーと、メイン機を近い将来64bit化できるかどうかをテストするただそれだけのためだったりする。一応メモリ4GB積んでるから64bit全く意味がないわけではない。

+ インストール自体は何も悩むことなく終了。Vistaもそうだったけど、最近のWindowsのインストールは楽だな。既報の通りX25-Mだと自動デフラグが活きたままだったので停止。アンチウイルスはNorton Internet Security 2009の対応版(Beta)をとりあえず突込んでみた。他にいろいろインストールして試さないとなー。

+ M1330はファンがうるさいので、SSDに交換したってマシン自体はうるさいのだが、それでもアクセス音が全くしないのが地味に気持ちわるい。

+ そして初めて使うWindows7。なんつーか、Vistaにすっかり慣れきってるせいか「普通」としか思えん。XPS M1330+メモリ2GBでVistaはぜんぜん重くなかったしなー。ただSSDはイイな。これはメインPC(FIENA)もSSDにしてしまいたくなる……。

■「天神乱漫」 (ゆずソフト) [GAME]

+ 佐奈。テンパると暴走して謎の毒舌が出るのはそこそこ面白いし、安玖深音も嫌いじゃないんだが、どーも、この手の妹系キャラは得意じゃないな。あと、佐奈がどんな存在かは共通ルートで明かされちゃってるわけだが、お話がその正体から当然予想される通りそのまんますぎなので、個別ルートに入ってから明かすべきだったような気がしないでもない。

+ ルリ。最近成瀬未亜が好きになるサイクルに入った気がする(「シスタ×シスタ」とかね)。そんなわけで、思ってたよりかわいいキャラだった。お話自体は、終盤3人の話になっちゃうので、ルリが中心からちょっとズレちゃう感じがする。まぁ、その分ルリがヘンな重さを背負う必要がなくなって、良い意味で「軽く」終わったと思うけど。

+ 紫先生。ウブなところとドSな人が共存するところにサクっとやられたぜ。中の人(夏野こおり)も好きだし、なんかちんまいっぽいのもポイント高いぜ。しかし、バニースーツ似合いすぎだろう……。

+ 葵。お互いを意識していない所から始まる幼なじみモノとして王道っぽい展開なのだが、葵視点を多用してじっくりやってくれるので大変こっぱずかしくてよろしい。くっついてからの初々しい二人もよろしい。終盤も重くなりすぎないのでよいだろう。CVの佐本二厘もベストマッチで俺的イチオシ。しかしメインヒロイン4人の中で1人だけえちシーンが1回少ない(*1)のはなんでやねん。

+ まひろ先輩。シナリオ展開からして実は黒い人ってオチかと思ったらそうじゃなかったので肩すかしです(ぉ CVは遠野そよぎで、俺的にはわりと好きなんだが、もうちょっとくだけた役の方が似合っている気がしないこともないなー。

+ 最後に姫。途中でドツボにハマって動けなくなってしまうので、その分損をした感はあるか。姫も悪くないのだが、なんか老竹先生がなんかカッコいい話だった :-) CVは高志麻矢……ってヒデぇ名前だなぁオイ。まあ一聴してわかるあの人ですが。

+ 最後におまけのえちシーン。うーん、オチなし :-)

+ システム的には文句ない。AQUOS LC-46RX1Wでプレイしている俺は16:9なのは大歓迎だし、同じ16:9の「水平線まで何マイル?」(ABHAR)よりベースの解像度が一段高いのも良い。一方でデータがデカいので非力なPCだとスキップとかがキツいかもしれんが、「ティンクルくるせいだーす」(Lillian)みたいに遅くてプレイに困るわけでもないしな。

+ 総合的に見ると、過去作に比べて大幅に進歩しているわけではないが、相対的にアラが少ないので、ゆずソフトのゲームでは一番他人に薦めやすいゲームだったように思う。評価は8点。今までも買ってきたし、次も買うでしょうなー。

■ブルースクリーン [COMP]

*1: サブの紫先生より少ないとはなにごと。いや紫先生のを減らされても困るが。

■水冷のこと [COMP]

+ FIENAの冷房なしで38℃付近と高い。真夏に45℃とかになるのもアレなので、ラジエータを追加したいが、RESERATOR XTのポンプ(JINGWAY DP-600同等品で、かつ回転が絞られてる)が厳しそうだ。揚程と流量がアップしたDP-600Pが出回ったら交換しちゃえばいいのだが……(*2)。

+ ま、うだうだ言っててもはじまらない。Black Ice SR1がcoolinglabに入ってくるようなので240サイズあたりを買って排熱強化して、それでポンプがダメならフル回転固定に改造して、それでもダメならポンプを交換しちまおう。

*1: 第2世代iPod touchならBluetoothが有効になって嬉しいかもしれないが……。
*2: 消費電力が1.5倍くらいなので、これはこれでRESERATOR XTが燃える危険が :-)
