■艦これ。1周年だそうで [GAME]

+ 俺はリンガ泊地サーバ開設時の大規模開放で入ったので、一周年は8/17なので、あまりピンとはこないのだが。

+ 瑞鶴もLv99になったのでケッコンカッコカリした。9人目。誰か止めてくれと思わんでもない。……といいつつ現在のLv98は夕雲・綾波・霧島・日向。次にLv99になるのは、5-4で稼ぐ霧島さんか、4-2で東方任務をこなす夕雲・綾波か、演習で稼ぐ日向か。

+ 3月作戦は500位以内に入れたので瑞雲12型と20.3cm(3号)連装砲をいたたいだ。ランカー報酬3本目の20.3cm(3号)連装砲でこれで手持ちは5コ。

Apr.24,2014 (Thu)

■艦これ。春イベントその1 [GAME]

+ E-1。水偵・水観を1つづつ積んだ重巡×2・金剛型戦艦×2に、彩雲を載せた空母×2を加えた構成で突入。着弾観測攻撃がよく掴めてなかったので装備はわりと適当だったが、まあ、6回突入でクリア。楽勝であった。

+ E-2。これも金剛型戦艦×2・重巡×2・空母×2。E-1から装備を下記のように変更。

  • 戦艦は 46cm三連装砲×2/水偵or水観/電探(or旗艦のみ探照灯)
  • 重巡は 20cm連装主砲(or3号砲)×2/水偵or水観/電探
  • 空母は 烈風(or烈風改)/烈風/流星改/彩雲

+ 制空権確保あるいは優勢を狙える場合は、戦艦・重巡は昼夜連撃装備がよいね。重巡の場合15.5cm三連装副砲は空母なし突入時以外は使わないでよい感じ。

+ んで、6回突入であっさりクリア。ボス前夜戦は旗艦の比叡改二Lv97が2回中破くらったのみだった。空母編成時はなにか補正でも入ってるのかもしれない。

+ さて次はE-3だが、手をつけるのは週末かな。

Apr.26,2014 (Sat)

■艦これ。春イベントその2 [GAME]

+ E-3。金剛改二Lv105→Lv106・比叡改二Lv97・榛名改Lv97・霧島改二Lv98・大鳳改Lv111・加賀改Lv116の固定編成で突破。9回突入8回到達ボス削り2回ボス破壊6回でクリア。

  • 金剛型は 46cm三連装砲×2/水偵or水観/三式弾
  • 大鳳は 烈風/烈風/流星改/彩雲
  • 加賀は 烈風/流星改/烈風改/烈風

+ なお支援攻撃はなし。怖いのはE-3-1の潜水艦だしね。

+ 三式弾を忘れてボス削りきれなかったのが1回、昼戦でスカスカ外して夜戦にいっぱい残して削りきれなかったのが1回と、E-3-1で榛名が潜水艦先行雷撃で一発大破で途中撤退が1回。まあ、わりと楽だったかな。

+ これで天津風をゲット。しかし谷風はE-1〜E-3攻略ではドロップせず。

+ さて次はE-4だ。三式水中探信儀は6コあるけど、三式爆雷投射機は2コしかない。九四式爆雷投射機でなんとかなるかな？

+ 顔ぶれは回避が高いほうから選ぶと、駆逐×5は、島風Lv90(回避95)・雪風Lv76(回避91)・夕雲Lv98(回避89)・時雨Lv89(回避89)・綾波Lv98(回避88)。二順+対潜補完用の航戦は伊勢Lv105(回避79)かな。

Apr.27,2014 (Sun)

■艦これ。春イベントその3 [GAME]

+ E-4。予定通りの顔ぶれで突入。

  • 夕雲改 Lv98 - 三式水中探信儀/三式爆雷投射機/ダメコン
  • 綾波改 Lv98 - 三式水中探信儀/三式爆雷投射機/ダメコン
  • 島風改 Lv90 - 三式水中探信儀/九四式爆雷投射機/ダメコン
  • 雪風改 Lv76 - 三式水中探信儀/九四式爆雷投射機/ダメコン
  • 時雨改二 Lv89 - 三式水中探信儀/九四式爆雷投射機/ダメコン
  • 伊勢改 Lv105→Lv107 - 試製晴嵐/試製晴嵐/カ号観測機/ダメコン

+ 順序はてきとうに入れかえたが、旗艦は必ず夕雲or綾波だった。他は改二or異能駆逐艦だけど、この二人は普通の駆逐艦なので……。なお、キラづけはしたが、はがれたら再度つけるだけでcond85維持はしなかった。

+ 7回出撃で1回ボス中破、6回ボス撃破。大破進軍が4回でダメコン消費は2つ。1回は雪風大破進軍→ボス開幕を運悪く食らって発動、もう1回は旗艦の綾波が大破したために、進軍自体に発動が必要だったため。というわけで雪風の轟沈ボイスを聴いてしまったのだった。

+ 伊勢は最初の2回は46cm三連装砲/46cm三連装砲/カ号観測機/ダメコンだったけど、どうせ二戦目は負け前提だったので最後は対潜最大を狙った。実のところ伊勢は1回小破しただけで他はカスダメだけだったのでダメコンは不要だった。

+ さて、これでE-5は開放された。行くか? 行くのか? 俺。
