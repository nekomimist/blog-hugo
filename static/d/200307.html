■夏はSTAX使いに辛い季節なのです [AUDIO]

+ 夏はイヤーパッドが汗でベトつく季節。STAX SR-007はイヤーパッドの密着感が強くてツライので、Lambda Nova Signatureに入れかえてみる。

+ さておき、ドライバのSRM-T1Sをそろそろオーバーホールに出したいのだが、これがないと部屋で音楽を一切聴けなくなるのが大問題。今度実家に帰ったら、行方不明のSRM-Xhの発掘に挑戦してみるかなぁ。

+ ……いっそSRM-T1Sを予備機に回して、SRM-007tかSRM-717でも買ってしまおうか(ぉぃ

Jul.21,2003 (Mon)

■最近のゲームとか [GAME]

+ 「こころナビ」の話題以降、日記にえちゲネタを書いてないが、いちおうそこそこプレイはしていたりする。だが、どーも楽しくないんだよねぇ。

+ 「モノグラフ」(DISCOVERY)。もののけって設定を十分に活かしてるかというとそうでもない。「けもの学園」(Tactics)を思い浮かんだが、あっちの方が楽しい。まぁ、娘太丸絵ファンだしな(ぉ …当然「ピチカートポルカ」(ぱじゃまソフト)も期待しているのだだだ。

+ 「こもれびに揺れる魂のこえ」(ユニゾンシフト)。眠いゲームだ、と思ってたらけんじょさんも似た感想だな。良し悪しはさておき俺の嗜好にいまいち合わない。そんなわけで途中放棄。

+ 「天巫女姫」(Grooming)。しかとみよ原画の巫女さんゲー。同じスタッフの「うたかな」に似た突き抜けの足りなさを感じるな。特に悪いとは思わんが、別に良くもなく、まぁ「普通」。完了済み。

+ 「メイド曜日はHがいっぱい」(Studio Air)。音声発生前に一呼吸システムの反応が止まるのがすげぇイラつくので放置中。うちの環境が悪いのかもしれんが。

Jul.27,2003 (Sun)

■最近のゲームとか [GAME]

+ さらにゲームを消化しようとしてみたり。しかし消化する数よりも増える数の方が多いな、最近。

+ 「アキバでお茶しよ!」(AngelSmile)。完了。や、別に悪いとかすげーつまんないとかはないんだけど、終わったあとに何も残らない感じがする。とはいえヒロイン連中はかわいいので許す。……そこっ、ロリってゆーなっ(ぉ

+ 「お姉ちゃんの3乗(きゅーぶ)」(Marron)。序盤。素のままのNScripterは萎えるなぁ。ゲーム内日時くらいセーブデータ画面に表示しようよ。ノリはまぁアレなので最後までやるとは思う。

+ 「ドキドキお姉さん」(アトリエかぐや)。みちると葉月先生のエンドをそれぞれ2つづつ回収。みちるはキャラといいバッドエンドといい人を選ぶ予感がするが、ハッピーエンドの雰囲気は悪くない。葉月先生はバッドエンドなしで2つともらぶらぶエロエロ風味。しかし頭が悪いゲームだ。こんなゲームをやっている俺の頭も相当だ(ぉ
