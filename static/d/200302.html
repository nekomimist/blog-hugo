■「ときどきシュガー」(Xuse) 4thプレイ [GAME]

■「ときどきシュガー」(Xuse) 5thプレイ [GAME]

+ 気をとりなおしてやりなおしてトゥルーエンド。うーん。

■「ときどきシュガー」(Xuse) 落ち穂拾い [GAME]

■「ときどきシュガー」(Xuse) 6thプレイ [GAME]

+ さてラストの亜緒(CV:須本綾奈)。一人称が「ぼく」で少年風のすっきり外見の女の子。

+ うひぃ、頬染めるとごっつかわええのう (^^)

+ …まぁ、この娘が\…ってことはこのシナリオに入るまえから判っていたわけだが、そういうことか。ふむ。

+ で、一回バッドエンドに行った後ちょっと試行錯誤した後トゥルーエンドへ。亜緒たんハァハァって感じ :-)

■「ときどきシュガー」(Xuse) まとめ [GAME]

+ 全員トゥルーを見たところでまとめ。

+ あまあまえろえろ万歳な俺としては、鬱っぽい展開が入るのが残念だったかな。あと、「フローラリア」もそうだったけど、話がちと青くさい。とはいえこのように一気に終わらせてしまう程度には面白かったけど。

+ 評価は…んー、ちょっとおまけ入るけど7点の「佳作」かな? ただ俺としては「フローラリア」の方が好きかも。

*1: んんと、大野まりな=神崎ちひろなのかな?

■「てのひらを、たいように」(Clear) 2ndプレイ [GAME]

+ 「明日も仕事だし寝ろよ」と俺の脳がささやく中、最初から始めて穂シナリオへ。一部選択肢が明示的じゃなくてやーね。まぁ、すぐにわかるからやりなおせばよいのだが。

+ あーもう、強気ですぐ手が出るくせに純情だなんてホント防御不能。ついでにおかーさんの穂波さんも実に素敵でもう母娘まとめてくださいって感じ(ぉぃ

+ 「食え!」…素直に誘えない穂ちん。いいなぁ。

+ 全面的に1stプレイと大筋同じなので、既読スキップを駆使してると萌えなシーンを連続して見ることになって悶えまくってたら終了。

+ …最後の最後が永久(ノーマル?)エンドと同じっつーのはどうか。でも、穂はイイね。
