Mar.1,2005 (Tue)

■「MAGICAL WITCH ACADEMY」(アトリエかぐや) [GAME]

+ えろシーンの数は多いしCVもがんばってるとは思うがどうにも微妙な感じを受けた。お話をまともに見せようとしてみたが結局話自体はありがちに終わり、かつシリアス部分があるせいでえろえろな部分が浮いちゃった感じか。まぁ、ヒロイン陣がどうも気に入らずに、結局脇風味のイリス学院長(CV:大波こなみ)が一番好みだった次点で俺は負けていたのだろうな。6点。

■日本語入力方法のこと [COMP]

+ もう3月かー。仕事が忙しいせいかあっという間に月日が経つな。

+ さて、またもや懲りずにATOK移行をしようとしてたのだが、結局またSKKに戻ってきてしまった。もうSKKから逃げようとするのは止めたほうがいいのかもしれないな。12年も使って馴れきっているものに勝つのは難しいよやはり。

+ もともとATOKに移行しようとしたのはSKKの同音異義語関連の弱さ(*1)なわけで、そのへんを補おうとふたたびbskkサーバを上げてskk-bayesian.elによる学習を再開させてみるのだった。効いてるのか効いてないのかよくわからないが、肥やしを撒いていればきっと報われると信じて :-)

*1: 強みでもあるがね。

Mar.9,2005 (Wed)

■きょうのよしなしごと [MISC][GAME]

+ 萌え分が足りないせいか、毎日ヘロヘロだ。覚悟はしていたものの5月連休もあまり休めなさそうでがっくし。夏休みもまともに休めなさそうだし、今年はマジで仕事三味だのう。

+ 「秋色恋華」(Purple)は先日終了。ヒロインたちはヘンなものの話自体は概ね王道。ならばべたべたにくっさい展開にする方が好みなのだが、そうでもなくてなんか微妙。真由(CV:大花どん)と葵(CV:木村あやか)あたりがキャラ的に好みだったかなぁ。評価は6点で普通。

+ 続いて「つくしてあげるの!」(PeasSoft)を開封したもののいろいろ耐えられなくて投げ出してしまった :-)

+ しょうがないので「魔法はあめいろ？」(Sirius)や「ゆりね」(CAGE)あたりのリプレイをして萌えを補給していたりするここ数日なのだった。

Mar.14,2005 (Mon)

■さよならGDM-F500R [COMP]

+ ずっとうちのメインディスプレイの役目を担ってくれていたGDM-F500Rの廃棄の処理を始めた。これの後継機だったGDM-F520ですら生産終了している今、こういう高画質のCRTは事実上入手不可能なわけで、ちょっとせつないね。LCDは"サイズ"と"重量"と"特定解像度のくっきりさ"以外ではまだ勝負にならないしね……。

+ ところで、GDM-F500RはPCリサイクル法の範囲内のモノなのだが、重量30Kgオーバーなこれ、エコゆうぱっくで送れるのかね? まぁ、チャレンジだ。

Mar.21,2005 (Mon)

■きょうのよしなしごと [GAME][AUDIO]

+ 「プリンセスうぃっちぃず」(ぱじゃまソフト)の体験版を引き続きプレイしていたら、"ソースごはん"なる単語が出てきた。 俺的にソースごはんと言えば、「真夜中のソースごはん」(*1)なので、当然のように猛烈にさねよしいさ子の歌を聴きたくなった。

+ ので、当然のように聴く。ちなみにうちのiTunes管理下の曲のうち、ゲーム関連曲以外で一番聴いているのはさねよしいさ子だったりする。

*1: 「風と空のことばかり」(さねよしいさ子, 1990年)所収。

Mar.27,2005 (Sun)

■ゲームなよしなしごと ― 「ひなたると」(Tarte)と「シス×みこ」(XANADU) [GAME]

+ 3/25発売のゲームが6本も来たが、土曜は仕事だったので消化も進まなくてもう大変。

+ 「ひなたると」(Tarte)。笛氏の絵が微妙にやらしくなったよーな気がする。しかし元のゲームの雰囲気を考えると、姉妹丼ばかりの内容はどうかと思わないこともない。日向さんとのまったりラヴラヴした日常ももっと見たかったよ、俺は。

+ 「シス×みこ」(XANADU)。一部で話題の通り序盤〜中盤の音声がヘンだ。せっかくまきいづみ&青山ゆかりなのにー。内容は「メイドさんしぃしー」(Le.Chocolat)を継ぐものにふさわしく、すがすがしいほどにはてしなくバカでしぃしーなのだが、俺は「メイドさんしぃしー」の方が好きかなぁ……。

Mar.30,2005 (Wed)

■「幼なじみな彼女」(イージーオー) [GAME]

+ 一周目はよかったんだがなぁ、田中美智の声好きやし、二人とも素直じゃないし、主人公はヒロだしー(*1)。ただ三周するとなるとちと話的に厳しいねぇ。えろシーンも似たよーなのばっかで飽きるし。数と尺があればよいというわけでもないな、と、まぁ、そんな感じか。

+ 総プレイ時間を見ると8時間強。一周目はセリフをちゃんと聴いてたのでわりと眺めかな。ただ、3順目は最後まで聴いたセリフは皆無に近かったかも。でもまぁ安いし、こんなもんでしょう。少なくとも「シス×みこ」(XANADU)よりはずっとおすすめ。

*1: 俺、ヒロユキだから。
