Sep.24,2016 (Sat)

■DMC-GX7mkIIのこと [MISC]

+ DMC-GX7mkIIはレンズにLUMIX G VARIO 14-140mmを追加した。ちょっとデカいけど、日常はこれ一本でいいね。狭い室内onlyの時は12-32mmを使えばよさそう。

+ ついでに「バウンスとシンクロフラッシュができる安いストロボが欲しい」と思って、こんな中華ストロボを買った。

+ で、ちょっとマクロレンズが欲しくなった。LUMIX G MACRO 30mm/F2.8が3万5千円くらい。うーん。

■Canon New F-1の不調 [MISC]

+ で、上記の流れてCanon New F-1に触りたくなって、カメラーケースから取り出したら、　シャッターが切れることは切れるが、切れるまでの時間が不定な感じ? レンズ外して空シャッターを切ってみると、ミラーアップがクソ遅い(。

+ こんな症状は自分ではどうにもならんし、なにしろ30年以上前のカメラなのでオーバーホールに出してみた。あまり使ってないから見た目きれいな状態のままだけど、この不調は使っていないせいだろうから、もうちょっと使ってあげないとね……。

Sep.27,2016 (Tue)

■メモリアル休暇その1 [MISC]

+ 勤続20年っつーことでメモリアル休暇を10日もらっていたのだが、やっと半分を消化する。

+ 午前中に広島に出発。大和路線直通で大阪駅→新大阪から新幹線→広島→宮島口でフェリー乗って宮島へ。厳島神社周辺を買ったばかりのDMC-GX7mkIIで適当に撮りつつ歩きまわったが、とにかく死ぬほど暑かった。でも旅程上、晴れていたのはここまでだった。泣けるね。

+ 歩きまわって死ぬほど疲れたので、広島移動は来たときとは変えて、宮島→平和公園をひろしま世界遺産航路フェリーで一気に。船長がいろいろ話をしていてわりと面白かった。到着後原爆ドーム周辺をうろうろした。とりあえず原爆ドームのポータルキーは取っておいた :-)

+ 暗くなってへとへとになってホテルに移動。ちょっと休憩した後にメシを食いに外に出て、風呂入って酒くらいつつ艦これデイリーをこなして(←病気)就寝。

Sep.28,2016 (Wed)

■メモリアル休暇その2 [MISC]

+ 朝食食って、艦これデイリーをある程度こなして(←病気)出発。外は雨だったので歩きまわるのは諦めて、広島駅に出てから呉線で呉へ移動。

+ 呉についたら雨が強くなってきていた。呉市海事歴史科学館(大和ミュージアム)に移動。適当に撮影しつつうろうろしてたら3時間くらい経過していた。なかなか面白かったかな?

+ その後、雨の中屋外展示を横目に見つつ海上自衛隊呉史料館(てつのくじら館)に移動、カフェで一息ついてから、館内をのんびり巡回。戦後の掃海の苦闘と潜水艦の歴史でこれもなかなか面白く見れた。あきしお内部はマジ狭かった。1時間半くらい館内にいたかな?

+ その後外に出たら雨が小降りになっていたので、歴史が見える丘→アレイからすこじまと歩いて行ったけど、荷物が重くて死ぬかと思った :-) その後バスで呉駅まで戻ってホテルにチェックイン。

+ 前日よりさらに疲労していたので、夕食も適当に近所でとった後に、艦これデイリーの残りをかたづけて(←病気)就寝。

Sep.29,2016 (Thu)

■メモリアル休暇その3 [MISC]

+ たった2日でむっちゃ疲労していたのと、持ってきた服の選択を間違えて全て汗だくになってしまったため予定より早めに呉を出て帰ることにする。

+ ホテルで朝食をとって、艦これデイリーをある程度こなして(←病気)出発。小雨降る中、入船山記念館へ。開館してすぐだったので人があまりいなくてよかったが、とにかくむし暑くてつらかった :-(

+ その後また大和ミュージアムまで行ってみやげを買ってから、呉線に乗って広島→新幹線で新大阪→大和路線で家へ。

+ 初日前半以外天気悪かったのがちょっとイマイチだったな。呉も広島も機会があればまたリベンジしたいところではあるなー。あまりグルメに興味がないとはいえ、食い物があまりに適当すぎたし。

+ さて、メモリアル休暇は今年度中にもう1度取らねばならない。次は一箇所に腰を据えてゆっくりしたいところだが、どこに行こう。

■スマートフォンのこと [COMP]

+ 今回の旅行で思ったが、Nextbit Robinは電池が保たないのがつらい。Xperia ZL2は電池交換時にタッチパネルの調子が悪くなったし、交換した電池が純正じゃないせいかこちらの電池保ちもあまりよくないわけでこっちもつらい。

+ 出歩く上では2つ持つのは正しい戦略だと感じたし、2つ持ちならNextbit Robinの電池持ちは致命的ではないと考えると、新しいものに変えるべきはXperia ZL2だろう。

  • 電池が保ってほしい
  • あまり熱くならないこと (Robinは熱いからね)
  • おサイフケータイは対応していてほしい (nanacoをよく使うのだ)
  • 指紋認証はぜひ欲しい
  • CPU性能はCortex-A53だけだとイヤかな
  • 防水もあって欲しいいけどそこまで重要ではない

+ Xperia X Compactあたりが国内で出ればいい気がする……。
