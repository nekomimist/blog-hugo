■艦これ。2016春E-5甲 [GAME]

+ さてE-5。駆逐4航巡4がセオリーだが、通常艦隊戦なので駆逐艦の大破が多そうと思ったので、補強増設済みかつ耐久が4nではない娘を選定。結果ほぼ主力ばかりになってしまった(*1)。

  • 秋月改/111 10cm連装高角砲+高射装置+4/10cm連装高角砲+高射装置+1/13号対空電探改+9/応急修理要員
  • 雪風改/111 10cm連装高角砲+高射装置+4x2/Bofors 40mm四連装機関砲/応急修理要員
  • 鈴谷改/138 20.3cm(2号)連装砲+8/20.3cm(3号)連装砲+6/三式弾/瑞雲12型Lv7
  • 筑摩改二/123 20.3cm(3号)連装砲/20.3cm(2号)連装砲/瑞雲12型Lv7/三式弾

+ 秋月は対空カットイン、江風雪風がBofors 40mm四連装機関砲、綾波がWG42という構成。航巡は最初から最後まで上記装備(瑞雲は熟練剥げたら予備に交換)。駆逐艦は旗艦をローテーションしてキラの維持に努めてみたが、剥げるもんは剥げるので、キラづけ出撃したり伊良湖を使ったりして全キラを維持した。

+ 支援艦隊はE-2のラスト時と同じ構成で最初から最後まで本気構成。基地航空隊は第一に零式艦戦52型(熟練)x3+零式艦戦53型(岩本隊)、第二に零式艦戦21型(熟練)/零式艦戦21型(熟練)/一式陸攻/一式陸攻と配備。

+ 第一は空襲マス+ボスを出撃指定したが、空襲マスの事故があまりなかったので最後はボス×2に変更。第二はボス前Lとボスを指定していたが、あまり機能しなかったので艦戦増やしてボス×2として、削りと制空削ぎをする方向に修正した。

+ なお基地はよく空襲で大破していたが、小破でも第二基地航空隊の攻撃はあまりダメージを出さなかった。何が悪いのやら。

+ これで11回突入して9回A勝利、1回D敗北、1回大破撤退で終了。正直一度もS勝利を取れないとは思わなかった。リコリス棲姫撃破は4回。削り中も結構撃破できてたので、あまり心配してなかった。

+ ちなみにダメコンは2つ使用。江風が旗艦の時に1回、綾波はボスで1回発動。

*1: E-6はヴェールヌイLv120と霞Lv94、大潮Lv84にまかせりゃよいかな、と。

■艦これ。2016春イベ感想 [GAME]

+ まだE-3甲でのんびりU-511/しおい掘りをしているけれど(*1)、まあ、消化試合なので……。

+ 最初の頃のゴタゴタは擁護のしようもない。あの状態でリリースされたということは、Quality Assuaranceという発想なんか全くないことになるだろう。数百万のユーザーを抱えるにしては体制が脆弱すぎるのではなかろうか。

+ ただ、緊急メンテ後の状態は、「ちゃんと攻略できる」ので、ゲームとしての設計は悪くなかった気はする。基地航空隊を理解して正しく運用すれば、(もちろん乱数の神には祈る必要があるけれども)E-6はすごく楽に行けるし、E-7も楽になる(ことがある)ようになっている。また、道中の事故要因はルートさえ選べば少なくなるようになっていて、試行回数は増やせるようにもなっている。全てがうまく噛み合えばすっと抜けられる。最初からこうだったらそう文句は出なかったのではなかろうか。

+ まあ、それでも文句がないでもない。基地航空隊は仕様的な洗練が足りない。道中でも指定の箇所で必ずやって来るので、何らかの制約を課したかったのは判るのだが、試行錯誤がしづらい仕様になってしまっているのが惜しい。ボス戦でガンガン基地航空隊が来援しては削ってくれるのは面白かったから、もう少し単純化とよくなるのではなかろうか。

+ まあ連合艦隊も最初は荒かったから、基地航空隊も次回以降に出番があるならば、少し洗練されるんだろうと思いたいね。
