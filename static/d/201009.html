Sep.2,2010 (Thu)

■きょうのよしなしごと [MISC]

+ なんとなく買った「はつきあい(1)」(カザマアヤミ)【 amazon / bk1 / 楽天 / 7netshopping 】を読んだ。実にあまずっぱくてよろしい。

■「君の名残は静かに揺れて」(ユニゾンシフト) [GAME]

+ 話としては、実にユニシフブロッサムらしい「大切な何かとひきかえに大切な何かを得る」話であった。ここのゲームはこれでいいんだよね、そういう作風ではバカ売れしないだろうが……。あと、茉百合さんの境遇がここまでヘビーとは「Flyable Heart」プレイ時には思いもよらなかったね、という感じ。

+ 晶くんが「あっち」に飛ばされないので、結衣とすずのが一切出てなくてちょいと淋しい。でもかわりに天音がかわいかったので許そう。そういえば、今回は魔百合さんモードがすぐに腰くだけになるのが面白い。やはり押し切られると弱いぜ茉百合さん。

+ ま、そこそこ面白かったと言えよう。でも、もう一度やるかどうかと言われると……うーん。
