■2005年えろげまとめ [GAME]

+ セーブデータのバックアップ数からするとプレイした本数は63本ほど。忙しいかったわりには週に1本以上やってたんだなー。まぁ、途中で投げ出したゲームも数多いけどな :-)

+ では俺的に印象に残ったものたちを。

+ 「ゆりね〜おねえさまがおしえてくれた〜」(CAGE)。女装主人公ゲーと言うと世間一般ではほぼ同時期の「処女はお姉さまに恋してる」(キャラメルBOX)が挙がるんだろうが、俺としてはやっぱり「ゆりね」なのです。頼りない奈月がエンディングでちょこっと凛々しくなるのがよいのです。紗耶香お姉さまの奈月へのべた惚れっぷりも印象的。

+ 「お嬢様組曲」(Symphony)。やはり小雪ラブでしょう。2chでの評価の動きて見てると、最初っから修正パッチ後の品質だったらもうちょっと評価されたような気がする。

+ 「神様のいうとおりッ 〜くじびきAI-BIKIスクランブル〜」(RUNE)。素直になれないたすくんと亜鈴ちゃんがよいのですよー。「キャラはすげーいいのになんかエピソードが足りてないぜ!」てな「リトルモニカ物語」にもあった物足りなさはあるのだが、好きなものは好きなんだからしょうがない。

+ 「サナララ」(ねこねこソフト)。小品として作られ小品として成功したゲームかな。ゲーム全体を流れる雰囲気がやさしげでよい上に、希未(CV:夏野こおり)と由梨子(CV:まきいづみ)が大好きなのです。おどおどな夏野こおりは「ナイショのよりみち」(CAGE)でも見られますが、あっちは男の子だしなー。俺の所にもかわいいナビゲーターが来てくれませんかね?(ぉぃ

+ 「せんせいがおしえてあげる 桜待坂Stories Vol.2」(ivory)。発売された事は仕事ですごく疲れてたんだけど、迅と4人のせんせい達の関係が心地よくて癒されました。俺の所にもかわいい4人のせんせいたちが来てくれませんか?(ぉぃ

+ 「さくらむすび」(CUFFS)。やはり紅葉ですよ紅葉。すがすがしいまでのバカップルエンディングに拍手。あとピアノの音色に激ヨワな俺としては全編ピアノのBGMもポイント高し。

+ 「はぴねす!」(ういんどみる)。次点級のゲームよりも「俺にとってよい所」がちょこっと多かったので格上げ。やっぱ俺はこーちゃ絵とえっちなヒロインが好きなんだよ!(ぉぃ あと、重さを感じさせずにスカスカ動くシステムもよかったと思うのです。

+ 「仰せのままに★ご主人様!」(桜月)。声萌え的にも、お話的にも俺好みでありました。ハーレムがさほどなげやりじゃないのもよい所。次回作はシステム的な向上をお願いしたいかな。

+ 「To Heart2 XRATED」(Leaf)。いろいろ食い足りないのだが、いいんちょと由真が気にいってしまった以上挙げざるを得ないね。ところで"愛佳"がなかなか"まなか"と読めない俺なのでありました。

+ 以下ちょっと惜しかったたね級。

  • 「処女はお姉さまに恋してる」(キャラメルBOX)。やってた当時はわりと好きだったのだが、今振り返ると話としてちょっと食いたりない。
  • 「プリンセスうぃっちぃず」(ぱじゃまソフト)。最後、ハーレムでまとめたらよかったのにー(ぉぃ ゲームとしてはよく動いてたので○。
  • 「夜明け前より瑠璃色な」(オーガスト)。菜月と麻衣は好きだし他の娘も悪くないんだけどねぇ。分岐前のつまんなさと、最後の「夜明け前より瑠璃色な」の食い足りなさで評価上がらず。
  • 「魔女っ娘アラモードDVD EDITION」(F&C/FC01)。あ、日記内に言及がない。コンシューマ版から追加されたホルンがキャラ的にも中の人的にも好みだったのでー。
  • 「思春期」(RUNE)。……うまく誉められないんだけど、好きなんです。

+ 世間一般の評価とすんごくずれてる予感。普通「ゆりね」とか「神様のいうとおりッ」とか挙げませんからっ。ま、例年に比べるとちょっと不作気味でしたかな今年は。

+ ……俺的イチオシは「サナララ」かな。

Jan.14,2006 (Sat)

■「あえかなる世界の終わりに」(キャラメルBOX) [GAME]

+ ……Rippleと言われると戯画のソレを思い出すよねやはり。

+ 千冬→柚子→ひまわり→ナギ→リップルで終了。

+ 個別エンドはあるけど基本的にお話は一本道ってことで、シナリオライターは違うけど、キャラメルBOXの2作目「めぐり、ひとひら」を思い出したりもした。進めるごとにスキップだらけになってしまうのも同じだし :-)

+ 最初は千冬。OP.に行く前に終わってしまうってのはワナのような娘。まぁ、主人公のなつめとの過去の関わりの少なさを思うと、柚子よりさらにグダグダ感溢れると思うのであそこで終わらせて正解のような気もするけど。

+ 次に柚子。中盤以降なつめといっしょにいる女の子はリップルとひまわりなのに、突然柚子エンドと言われてもちょっと困るよね。なつめが柚子のこと思い浮かべるシーンをもっと増やすとよかったような。

+ ひまわり。ほしまるシナリオでCVが海原エレナとくれば、BLUEのなずな・シャマナシャマナのサラに倣って最後は「お兄ちゃん」か?と思ったらちょっとだけひねってあった\。無表情な娘がちょっとづつほどけていって、でも完全にほどけきらないまま終わるってのはオレ好みだった。にゃん。

+ ナギ。……リップルのスカ扱いですかな、この娘のエンディングは。エンディングなしでもいいような気がするけど、「シャマナシャマナ」のラビみたいに変な文句が出るのを嫌ったのかな。

+ 最後にリップル。や、強気っぽくて、なつめを意地悪くからかったりするけど、実は素直になれないはずかしがり屋だったりしてとてもいい娘ですな。あとCVの青山ゆかりがすごくよかった。ラストどう締めると思ってたらなるほどたしかにソレがあったね、という感じ\。ところでリップルとえちしてるシーンは端から見たらきっとすごく間抜け(ぉ

+ さて。中盤までの謎がいろいろ増えていく感じとか、キモの部分の謎が最後収束してる感じとか(*1)、お話そのものは悪くないですな。俺好み。ただ、共通部分があまりにも多すぎてマルチエンディングのゲームとしてはイマイチでしょう。エンディングがあるのはリップルとひまわりだけにして中盤の道筋を二人それぞれ変えるか、柚子シナリオをもうちょっと頑張って描写してバランスを取るかしたほうがよかったんじゃないかな。

+ ま、文句はあるけど、最後までやる気が継続したし、なんか楽しかったってことで評価は8点。ほしまるシナリオのキャラメルBOXのゲームは肌に合うので、次も出たら買うんだろうな。もっともキャラメルBOXの今までのパターンからすると次回作は違う人がシナリオ担当だろうけど。

*1: よくわからんことも多いけどな。
