■艦これ。 [GAME]

+ 先制爆雷。五十鈴以外は装備込み(しかし改修値なし)で対潜値100以上が必要と。

Jul.10,2016 (Sun)

■HTC Vive [COMP]

+ Intel Core i7-4790K/GeForce GTX 970/メモリ16GBのPCに接続した。HDMIはTVへの出力で埋まっているので、DisplayPortをコレでHDMIに変換してつないだ。GPU性能的にはまあ推奨の下限に近いんだろうか。CM3D2でメイド1人を見てるだけなら特に問題なしな感じではある。

+ 設置場所はメインPCの置いてあるリビングである。ベースステーション2つは部屋の上部にコレの押しピンで固定した。賃貸ではないから壁に大穴開けてもべつにいいんだけどね。ベースステーション×2とコントロールボックスでAC電源が3つ必要なので、なかなか電源まわりの配置が苦しい。

+ スペース的にはイスとテーブルを退避すれば2m×2mを確保できるので、ルームスケールを使うゲームでもそこそこ対応できるはず。

+ で、購入特典でもらえたThe Labをやってみると……いやあ、可能性を感じるねこれは。

+ あと、CM3D2。自分で作った嫁イドが目の前で踊ってるの見るだけで感慨深い。基本こちらを見つめてくるので、なかなかくすぐったい。あとはこちらが近づきすぎたら避けたりしてくれると面白かろう。これも可能性は感じる。

+ とりあえずさらにGPUパワーが欲しくなった。GeForce GTX1070を買おうかな。

■艦これ。 [GAME]

Jul.24,2016 (Sun)

■Surface Pro 4 [COMP]

+ なんとなく勢いでSofmap.comで中古のSurface Pro 4(Core i5/RAM8GB/SSD256GB)を買った。先日の値下げの影響で中古も値下がりしてるし、Sofmapはポイントの蓄積もあったので、案外安く済んだ。

+ 画面が3:2なのがよい。解像度が高いのもいい。Windows Helloもよい。ペンの感じはVAIO Duo 11よりよい、と今のところ特に文句はない。

+ Type Coverはとりあえず買わずに、うちで余剰していたThinkPad Bluetooth ワイヤレス・トラックポイント・キーボードと組み合わせて使っているが悪くない。持ち運び時に悩みそうだけれど……。

+ 暁が先制爆雷攻撃できるようになったし、そろそろ放置していた「海上突入部隊、進発せよ！」任務をやってもいいかな。比叡改二Lv143/霧島改二Lv143/長良改Lv110/暁改二Lv94/雷改Lv80/電改Lv81だし、まあ、駄目なことはあるまい。
