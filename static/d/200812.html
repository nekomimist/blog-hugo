Dec.6,2008 (Sat)

■Realforce 91UDK-G [COMP]

+ ここ1年ほど、Microsoft Natural Wireless Ergonomic Desktop 7000をキーボード&マウスとして使ってきていたのだが、BlueTrackを試したくてマウスをMicrosoft Explorer Mouseに交換してしまってマウス部がお荷物になってしまったのと、キーボードの刻印がハゲたりパームレストにひび割れが入ったりして見た目みすぼらしくなってしまったので、かわりにRealforce 91UDK-Gを買ってみた。

+ Realforce 89とRealforce 91UBKに続いて3台目のRealforce購入だ。会社のPCのキーボードもずっとRealforce 91Uだし、どんだけRealforce好きなんだ俺……。

+ さて、キー刻印が茶のレーザー印刷だったRealforce 91UBKに対し、Realforce 91UDK-Gは黒の昇華印刷なので、見た目がさらにストイックだ(*1)。Realforce 91UBKのキートップは使い込むにつれどんどん薄くなっていって悲しかったけど、昇華印刷ならもうちょっと保つにちがいない。

+ キーの押し下げ圧は全て45gなので、小指付近の負荷が普通のRealforceより高い。が、実はRealforceの30gキー部分は軽すぎると思っていたので、この方がむしろよい。ソフトの助けを借りずに左CrtlとCapsLockを入れかえられて、かつ交換用のキートップまでついているのも実によい。

+ 一方、Microsoft Natural Wireless Ergonomic Desktop 7000が持っていたパームリフト+パームレストの快適さがないのが残念だ。とりあえずパームレストは別途手配しよう。

*1: ぱっと見無刻印に見える。

Dec.15,2008 (Mon)

■iTunesでのタグのつけかたの話とか [COMP][AUDIO]

+ ふとやる気になったので、合唱関連CDもiTunesに全て取りこんでみた。もちろんこんなものをmp3やAACにするわけはないので、Apple Losslessで入れる。これで、現在iTunes管理下にある曲は8045曲84.72GB。Apple Losslessがわりと多いのでけっこうデカいね。

+ さておき。Classicもそうなのだが、合唱モノはArtistフィールドに何を入れるか大変悩むよな。Classic系は同じ曲でも指揮者・演奏者・オケの差が大変重要なので、Artistフィールドに「指揮者; ソリスト; … ; オケ」って感じで入れてるのだが、合唱系は指揮者をそれほど気にせず聴くので、これだとイマイチなのだな。結局作曲家をArtistフィールドに入れる俺。これはこれでよいけど、なんか不徹底なのがなー。

+ さて、そろそろまたClassicや合唱のCDを買い漁るモードに入りますかな。残業規制モード絶賛継続中で月の収入が10万くらい落ちたけどそんなことで俺の趣味散財は減らないぜー(ぉ

Dec.20,2008 (Sat)

■最近やってたゲームたち [GAME]

+ 最近えろげの感想書いてないなーということでつらつらと。

+ 「FairlyLife」(HOOK)。設定的にはLikeLife 2nd Stageぽいけどノリは大分違う。さておき、ひまわりはいいコだけどヒロインとしては力不足だなとか、南は自己中心的すぎるので人を選びそうだとか(俺は嫌いじゃないが)、優の最初のえろシーンはいくらなんでも突然すぎとか、まぁ、いろいろあるけど、このゲームの95%は癒し系の未来と頼子さんでできていると個人的には思うので問題なし。まぁ、7点かな。

+ 「明日の七海と逢うために」(Purple Software)。みんな本編でこんなに楽しいヤツらだったろうかと思ってしまうほどに、どのキャラも本編よりキレが増してたな。さて、七海もさることながら、七海妹もよい。お話自体はハンパにシリアスなのがつまらんとも言えるかも。まぁ、7点か。

+ 「しゅぷれ〜むキャンディ」(枕)。ジャコがCVの金松由花含めて大変好みであった。あと、弓音さんのお姉さんっぷりとマニアっぷりが楽しい。CVのかわしまりのも好演。全体の感想としては、最後の2ルート\とそれ以外の落差が激しいのがアレかね。いきなりシャイニングトラペゾヘドロンとか言われても――それはそれで王道だけど――ちと困るよな :-) 評価は7点。

+ 「あいれぼ 〜IDOL☆REVOLUTION〜」(SIESTA)。SIESTAらしい実にどーでもいいお話だったのだが、「ぱすてる」よりはマシだったように思う。しかし、プロデューサーというならば、もっと仕事方面でキリっとせんかいと思うのは俺だけではあるまい。6点。

+ 「姪少女」(たぬきそふと)。この主人公はいろんな意味でダメすぎる。ウソでももうちょっと愛が欲しいよ俺は。まぁ、一番ダメなのはこんなゲームを買った俺だろう :-)

+ 「ENGAGE LINKS」(ALcot)。一番惹かれないのから倒すの法則でマリナをやってたら後半でイヤになって全部スキップしちまった。ぶっちゃけ分岐まではそう悪くもないのだが、分岐後はやたら陳腐な感じでちと耐え難い。「Triptych」もそうだったけど、ここのヒトたちはシリアスをやってはいかんのでは……。たぶん続きはやらない :-)

+ 「さかあがりハリケーン」(戯画)。学園祭開催決定までが物語上の一番のヤマであり、そこまでにある個別イベントが話の流れに介入しないものなので、「共通ルートが長い」と感じられる問題がある。とはいえ雰囲気はよいし、下手に長すぎず気持ちよく終われる利点があるとも言えるだろう。キャラ的には奈都希が好みで、涼が次点。まぁ、手放しですばらしいとは言わんけど、楽しく終わらせることができたゲームでした。評価は8点。
