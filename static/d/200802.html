Feb.16,2008 (Sat)

■「さくらシュトラッセ」(ぱれっと) [GAME]

+ ライトなノリでいい感じである。最近見た目に重そうなゲームは手が伸びにくくてねぇ。

+ かりん。本気で子供っぽい娘なわけだが、ちゃんと理由が話られるのが好感が持てる。話の展開もあまり見ないノリで面白い。結局かりんがちょっと大人になるだけで何もかもが解決する話なのだが、そこがわからなかったあたりもまたかりんらしくてよいと思う。CVの桜川未央もベストマッチでよい。

+ ルゥリィ。猫が嫌いな春美と猫なルゥリィのお話。クールでやたらとつれない娘なのだが、たまに甘えてくるところがあまりに可愛すぎて反則。最後の最後のオチはナニっちゃナニだが、たぶん狙ってやってんだろうなぁ。最後の絵がまたかわいい。CVの井村屋ほのかも実によい。

+ 優佳。テレ隠しにぽかぽか殴る娘ならいいんだけど、なんか理不尽にバキバキ殴る印象が強いのがにんとも。春美がプロポーズするシーンのマリーの魔法が印象的か。CVの風音はキャラに合ってるやね。

+ マリー。ダメな恋愛する娘だなぁ。ある事情で春美を一旦ふる事になるんだけど、その後の春美に対する態度がグダグダなあたり、某香澄ちゃんを思いうかべちゃうぞ。一方、終盤自虐モードに入ったマリーに、春美が料理で語るシーンはなかなかイイと思う。CVは佐本二厘で、これもなかなかハマってたように思う。にしてもマリールートのクラウディアは真っ当すぎて逆にヘンだよな :-)

+ その他の話。ゲーム的なデキやや演出の話はさておいて、セーブしたページを覚えてないとか、ゲーム中日時がセーブデータからはわからんとか(*1)、システムのユーザビリティがイマイチなのが残念だね。

+ ま、楽しくプレイできたよいゲームでした。8点。

*1: 個人的にはクリックしても次の音声が始まるまでは音声を残す設定が欲しいけど、これはない方が多いし好みの問題だとも思うので許せるかな。

■「ピリオド」の話 [GAME]

+ 「ピリオド」(Littlewitch)をリプレイしつづけていたりして。短めのゲームなので、手軽にリプレイできるのはよいよね :-)

+ エ作板のピリオドスレとかで一部の人間に朝姫がビッチ扱いされることがあって苦笑。他の人に告白していただけでビッチ扱いとは、独占厨の考えることはわけわからんぜ。他の人が好きだったヤツを振り向かせるのがいいんじゃないか(*1)。

+ 最後の光樹と朝姫のやりとりが好きだ。「……イタリア人?」と返す朝姫も素敵だが、それにノって返す光樹も大好きだ。やはりヒロインだけがかわいくてもダメで、ヒロインと主人公が似合ってないと恋愛モノとしてはダメだよね。

*1: まぁ、朝姫は自分にとってとって光樹が、雄一以上に特別だったことに気付くわけで、他の人を好きだったヤツを振り向かせるのとはちょっと違うのだが……。
