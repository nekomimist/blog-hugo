Jan.4,2008 (Fri)

■iPod touchと俺 [MISC]

+ 今回の帰省ではiPod touchを持って帰らずに、5G iPod(60GB)を持って帰った。初代GIGABEAT→3G iPod→5G iPodとHDD搭載プレーヤを選びつづけてきた俺は、手持ちの全ての曲が入っていない状態が好きじゃないということがよくわかった。

+ ま、うち5G iPodも60GBしかないから俺の手持ちの曲全ては入らないのだが。iTunesのライブラリは83.1GBある。そのうち53GBはApple Losslessのクラシックと合唱曲なので、これらを外してiPodに入れているのが現状。全部入れたくないわけじゃないので、iPod Classic(160GB)がちょっと欲しくなったりする。iPod touchを売りとばしてiPod Classic買うかなー。

Jan.13,2008 (Sun)

■きょうのよしなしごと [COMP]

+ 家のネットワークケーブルにマトモなものが少ない事にふと気付いたのでケーブル交換大会開催中。それにしてもうちのマンションは各部屋にネットワークの口があるのはよいのだが、100Base-TXなのがちょっと悲しい。せめてリビングとサーバ部屋の間は1000Base-Tで繋ぎたいよなぁ。

+ Mac mini (Core Duo 1.66GHz)が遊んでいるのがもったいないので、Leopardお遊び用と、「Vista上では動かないモノ」を全部押しつけるマシンにすることにした。FIENA上のVirtualPCで動かすよりは総合的には快適だろうし、ちんまいからリビングに置いても邪魔にならないのもよい。

+ ただ、うちのAQUOSの入力はFIENA用にDVI1入力と、PS3・XBOX360・AC-AR55用にHDMI3入力を使い切っているので、Mac mini用の入力端子がないのが問題だ。DVIセレクタやHDMIセレクタを買えばいいのだが、この際TA-DA5300ESを買っちゃおうかなーとか思いはじめていてちとヤバい :-)

Jan.18,2008 (Fri)

■きょうのよしなしごと [COMP]

+ ネットワークケーブルは順調に交換完了。ついでに家の幹線ハブの横にgigabitのハブを置いて、リビング-サーバ部屋間をgigabit化してみたらうまくいった。家全体をgigabit化してもよいかもな。

+ そしてMac mini(ASAHI)をリビングへ移動する。AQUOSのDVI入力を空けるために、XBOX360の出力をHDMIからD5に、FIENAの出力をDVIからHDMIにそれぞれ変更。最後にASAHIの出力をDVIでAQUOSに入れて対応終了。

■「ピリオド」(Littlewitch) [GAME]

+ "すこしだけふしぎ"な現代世界の中での優しい物語。俺はこういう話は大好きだ。

+ 葵。悪友タイプの女の子。話のパターン自体はわりと見るカタチなのだが、解決の仕方がちょいと面白かったかな。琴先生の仕込みでデートする時の姿がたいへんかわいくてよい。

+ つづみ。たいへん謎な娘だが、その謎について話の中でほぼ何も触れないのが却ってよいと思う。ヘタに踏み込むと陳腐になるし、陳腐にしないほど踏み込むのはこのゲームの雰囲気には合わないしね。ラストの姿はこのゲームであれば必然か :-) なんであの姿でのえちシーンがないのだ。

+ 美由。大変不運な目に合う娘だが、あのほそっこい腕にギプスつけて吊ってるのが実に愛いのだよな。光樹じゃなくとも世話してあげたくなるね。お話としては、ある意味王道的展開でこれはこれでよし。

+ 鈴。言動と中の人の関係で「神様のいうとおりッ!」(RUNE)の亜鈴を思い出した俺(*1)。二人のデートシーンのCGがすげー好きだね。あと、交換日記ネタで俺の過去を思い出してたいへんムニャムニャな気分になった :-)

+ 小羽。天使な人。羽があるのに飛べないのはシッポがないからですよ(←ネタが古いね)。「HoneyComing」(HOOK)のクレアに似たいい娘すぎて困るタイプやね。パンを配達する姿が実に愛いのだな。お話は最後の亜理紗ちゃんが印象的。いい娘や。

+ 幸奈。デコ先輩ラヴ。キャラ的には大好きだけど、お話的には印象が弱いかもしれん。いやもちろんアレ\と、その後のえちシーンは印象的だけどね。

+ 朝姫。真のヒロインっぽく過去の「ある人とある事」を二人でひきずる話。他の人を好きだった娘を振りむかせる話が大好きな俺としては、この話はすげー好きだ。なかなか素直になれないキャラ付けもよい。あと、個人的には金松由花の声がすげー合ってたと思う。バカって言われてニヤける俺がバカです :-)

+ 琴先生。一番年上で一番ちんまい人で中の人がまきいづみってどんな罠だい。おまけっぽく、深刻な事がないお話でこれはこれで悪かない。

+ 総評。お話は重くなりすぎず軽くなりすぎず俺的にはわりとよい感じ。演出は今までのより「普通」になってるとどこぞで見たが、それでも水準以上やね。あと、音楽が好みだったのだが、サントラ一般販売マダー?って気分。

+ ま、やさしい気分でプレイできるよいゲームでした。8点。よくできました。

*1: あのゲームのシナリオ担当は両人ともこのゲームにかかわってること考えると案外面白い。

Jan.30,2008 (Wed)

■最近のえろげのこと [GAME]

+ 忙しくて精神的余裕が足りないせいか、新作ゲームよりもプレイ済みゲームをリプレイする方が多い。そんなわけで最近は「ピリオド」(Littlewitch)をふにふにつまみ食いでリプレイしている。ヒロイン全員大好きだが、やはり朝姫が好きすぎるなぁ。

+ 一応新作もちまちまプレイしている。「FORTUNE ARTERIAL」(オーガスト)は完了。感想は別途書くかもしれないが、ちょいとこぢんまりとした印象かね。Artemis☆さんの「広げた風呂敷は一応畳んであるが、風呂敷のサイズがハンカチくらいだった」って、まさにそんな感じ。

+ んで、今は「ご主人様だーいすき」(SCORE)をちまちまプレイ中。主人公は変態だが案外よいヤツだ。「Garden」(CUFFS)は最終的なパッチが出るまで待ち。「さくらシュトラッセ」(ぱれっと)は次に手をつける方向で。

+ 去年のゲームを大分積み残している。「こいびとどうしてすることぜんぶ」(Sirius)が中断中。これは「さくらシュトラッセ」に手をつける前にやろうかな。「おしえて!唯子先生【エッチ】を覚える大人の性教育レッスン!!」(SQUEEZ)も中断中。前作の唯子先生の方がかーいかった気がする……。

+ 「FairChild」(ALcot)とか「Clover Point」(Meteor)とか終わらせているのだが、 すごく悪くはないが特に良くもないデキで感想を書く元気がない。「世界で一番NGな恋」(HERMIT)は面白かったので別途感想を書きたいところ。
