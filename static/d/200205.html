■「水月」(F&C) 8thプレイ / 総合評価 [GAME]

+ ラストの那波さん。

+ あんまり好きじゃないとか思ってたけど、やってみるとけっこうやられたなぁ (^^;

+ シナリオ自体は…単純に受けとってよいのだろうな。何が「正しい」とか考え出すと俺の気分のおさまりが悪い :-(

+ 全部終わったので評価を。

+ システム。安定度その他は問題なし。問題点としてはスキップが遅いのと(*2)バックログが少ないのが筆頭だろうか。

+ 絵。弓道がらみの絵は正直「?」と思わんでもないけど、それ以外の絵を見れば☆画野朗を原画に選んだのは大正解。なんつーかどれもこれも反則級にかわいくて、かつ微妙にえっちい香りがしてすばらしい。塗りも良し。

+ 話。全般に不思議な感じの話。那波と雪さんのエンディングは微妙にスカっとしない。まぁそれもきっと狙いなんだろう。残りの話はわりと明解か。話としては花梨の話が好み。あれはよいものだ。

+ 文章。気にならなかったのでたぶん「普通」程度なのではないか。誤字も特には気付かなかったな。ちょっとセリフが長めだが、音声なしなのでさほど問題にはならず。

+ 音楽。あまり印象に残ってない (^^; 合っていたとは思うが。

+ 総合評価としては…佳作かな。よくできているけど「傑作」と言うには何かが足りない。結局はキャラを気に入るかがカギだな。

*1: いや、水月も台詞はかなり長いんだが。
*2: スキップ中は立ち絵エフェクトをキャンセルすべきだろう。

+ さて、Willamette-1.9GHz(FSB400)+i845とNorthwood-2.53GHz(FSB533)+i845G、どれくらい差があるかに興味があるが今日はもう寝る :-)
