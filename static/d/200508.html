Aug.9,2005 (Tue)

■「さくらむすび」(CUFFS) [GAME]

+ 可憐の後、桜の2つのエンドを見て、最後に紅葉。

+ ……なんというか、紅葉シナリオの印象に全てが塗り潰されてしまった。可憐シナリオも桜シナリオも「理不尽な何か」をわざと残して終わらせるくせに、紅葉シナリオの圭吾たちは、そういう「理不尽な化け物」をぺろりと食らってしまうほどにいちゃいちゃバカップルでそして前向きで強固な二人で終わるのだ。脱帽。

+ 桜の無邪気さはかなり怖いし、可憐ちゃんはよい娘すぎて影が薄いので、やっぱ紅葉やね。いつもにこにこで、主人公に大して世話やきたがりで触れたがりで、でも、両親やクラスメイトにからかわれると激しくのぼせて赤面しちゃう、そんな娘。花梨@「水月」のようにもうちょっと成長しているほうがより俺好みだったのだが、まぁ、そこは瑣末な問題だろう。

+ 結論からすると「水月」(F&C)コンビらしいゲームだったなとしか言えんな。俺は好きですよ。\

Aug.14,2005 (Sun)

■HIMIKO Rev.7 [COMP]

+ 散財物(Athlon64x2 4600+その他)が届いたので、さくさくとHIMIKOを更新。

+ で、ASUS A8V Deluxe(VIA K8T800Pro)にしてRADEON X800Proを変えずに済ませたとはいえ、電源を変えたり、メモリを買い足したりしたので、結局十数万円コースになった。ひさびさに大き目の散財だ。

+ んで、CPUクーラーのThermaltake BigTyphoonだけ在庫切れて届くのが遅れることになったので、暫定的にリテールクーラーを使ってるのだが、これがQ-Fanの配下に入れても実にうるさくて困りもの。ああ、BigTyphoonはいつ届くのかなぁ……。

+ あまりにやかましいので、Athlon64 3000+の時に使っていたPAL8150Tに変えてみた。つけてるファンがPanafloの2000rpmと低回転なので、能力的にリテールのそれより少しだけ劣りそうだが、まぁ、BigTyphoonが来るくらいまではこれで行こう。

Aug.19,2005 (Fri)

■HIMIKO Rev.7その後 [COMP]

+ HIMIKOのサウンドまわりがどうも安定しない。

+ まずM-Audio Firewire Audiophileが安定しなかったので、あきらめてTerraTec DMX6Fire LTに戻してみた。こいつはこいつでPentium4 with HyperThreadingで安定しなかったので、Athlon64x2で安定するかどうか不安。まぁ、iTunesやfoobar2000でしか使わないから、ハードウェアアクセラレータの設定を2段落として様子を見てみよう……。

+ ゲーム用のSoundBlaster Audigy2も安定しない。ハードウェアアクセラレータを1段落とすとわりと安定するが、これだとEAXが使えないのでAudigy2の意味なし :-(

+ 昔PentiumIIやPenitumIIIでDualやってたときもサウンドカードには悩まされたもんだけど、もうちょっとなんとかならんかねぇ。

+ あと、やっぱりAthlon64x2 4600+は熱い。負荷かけると70度超えちゃいますよ。やっぱりPAL8150T+静音ファンでは力不足かー。BigTyphoonはいつ届くんだろう。

Aug.21,2005 (Sun)

■きょうのよしなしごと [MISC][COMP]

Aug.26,2005 (Fri)

■きょうのよしなしごと [COMP]

■「ちょっと素直にどんぶり感情」(ういんどみる) [GAME]

+ 全部終わった。前作「魔法とHのカンケイ」が税込6090円だったのに、これが税込9240円っつーのはなんか納得いかなかったりするのです。

+ とまぁ、そんな中でお気に入りは珠美さん(CV:北都南)。ことり・ヒナのおかーさんなくせにちんまくて外見幼くてしかし巨乳。キャラ的には日に好みなので、もっとえろえろなシーンが欲しかったなーと思わないでもない(*1)。次点は雪音(CV:木村あやか)か。
