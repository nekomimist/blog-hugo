Oct.5,2007 (Fri)

■きょうのよしなしごと [COMP][AUDIO]

+ Majestouch Wirelessが修理から帰ってきた。コントローラを交換したらしい。これで安定度が上がるといいな。

+ STAX SR-007のヘッドバンドの部分がぼろぼろになってきた。もう買って7年半だしなー。SRM-007tもそろそろ球交換すべきかなって気もするし、STAXに送って両方ともオーバーホールしてもらおうかな。……となると、とりあえず、代替ヘッドフォンを買わなきゃな。

■「ほしフル」(F&C) [GAME]

+ 最初に陽(ひなた)。あまりしゃべらない娘なので、彼女の行動の意図がわかりづらくはあったとしても、さすがに主人公の行動が納得いかんなぁ\。

+ 次に水希。主人公の親友・惑の悪い面が炸裂するわけだが、じゃあ主人公と水希の言動が誉められるほどマトモだったかというとそうでもない :-) 結局友人4人でズブズブ悪循環モードって感じのお話だったな。

+ 次に琴姉。惑の悪い面炸裂パート2なお話なのだが、つっぱしってヒドい事をやったあげく後は一人でスネてるだけなので、水希シナリオよりましな感じ。むしろ、惑より琴姉の自分勝手さの方が気になるシナリオであったことよ。

+ 流史先輩。正直期待していなかったのだが、一番イライラ度の小さいシナリオだった。へたれまくりの琴姉にイラっときそうになるけれど、流史先輩がびしっと決めてくれるし。あと、二人の距離が近くなってからのムズムズ感がよい感じ。

+ 最後に瑠歌。中盤の、瑠歌にされたことをなんでも耐える主人公とそれを見て暗い喜びに笑う瑠歌が壮絶だ。そんな二人だったのに急転直下で一晩でベタ甘な二人に。翌日の惑のオドロキも当然だ :-)

+ 総評……正直水希・琴姉あたりは惑の事を抜きにしても面白くなくて、陽はまぁ普通で、流史と瑠歌はまあまあ面白かった。そんなわけで評価は「普通」の6点。どーもいかんねぇ、F&Cは。

Oct.20,2007 (Sat)

■「ソルティアンジュ魔法倶楽部」(micro) [GAME]

+ 真央→アスカ→冴子→一葉→真琥で終了。魔法倶楽部とかいいつつ後半のノリは魔砲バトルだったりするのはよいのかわるいのか。前半の冴子先輩を中心としたドタバタが楽しかった俺としては微妙なところ。

+ 真央(CV:根津まお)。妹的立場な娘。一人だけピー音修正があるのが謎だ。

+ アスカ(CV:佐本二厘)。強気な幼なじみに見えるくせに、ゲーム中は終始ヘタれてるのはちょっと変わってて面白いかもしれんな。

+ 冴子(CV:ヒマリ)。「天が呼ぶ、地が呼ぶ、人が呼ぶ! 悪を倒せと我を呼ぶ!」……と、性格的にも魔法的にも最強で大迷惑な人 :-) まこばるさんの次に好きだ。あとメタちゃんがムダにかわいい(ぉ

+ 一葉(CV:香澄りょう)。優等生っぽいくせに無駄にえろい所が好みだったなぁ。

+ 真琥(CV:理多)。最初は敵だが途中で合流する娘。一人だけ修正が必要になるようなセリフを全く言わないのが謎。この娘だけをやったシナリオ担当がいるってことかね。なんにせよ、意地をはるまこばるさんも、デレるまこばるさんもとてもかわいくてよし。

+ 基本的にどのヒロインでも話の流れが同じなのが厳しいかなー(*1)。ま、文句はあるけど、そこそこ楽しかったかな。ただ高い評価をあげるのにはもう一歩二歩の突き抜けがほしいところではある。7点。
