■Emacs 23への道 [COMP]

+ Emacs 23.0.95を試している。以前Emacs 23で全角ハイフンの幅がおかしいって思ってた話だが、どうも下記のような話のようだ。

  • skkで全角ハイフンを入力すると、バッファ内でEM DASHになり表示が半角幅になる。
  • skkime/MS-IMEで全角ハイフンを入力すると、バッファ内でHORIZONTAL BARになり表示が全角幅になる。

+ SKK内部は辞書サーバとの通信含めてeuc-jpが基本。Emacsでは(*1)「euc-jpの全角ハイフン」=「EM DASH」だから、EM DASHとして入力されるのだろう。一方、IME入力やコピペで全角ハイフンをEmacsに持ち込むと、Windowsによりutf-16le-dos(?)のHORIZONTAL BARとして入力されるようだ。

+ EM DASHが半角表示される件は、いわゆる"East Asian ambigious character width"問題の1つなんだろうが、Emacs 23でどう逃げればいいかはまるでわからないので放置する :-) どうせ見た目だけの問題だ。

+ 一方で、shift_jisやeuc-jpのバッファにHORIZONTAL BARやFULLWIDTH TILDEを入れちゃったり、cp932やeucjp-msのバッファにEM DASHやWAVE DASHなんかを持ちこんでしまうと、外に出すとき(セーブ時とかな)にいろいろ問題が出るのでこれはなんとかしたい。

+ コピペやIME経由でshift_jisにない文字が持ち込まれやすい以上、Windows世界内のEmacsはshift-jis系のcoding-systemにはcp932を使うべきなんだろう。

+ mule-jaの半田さんのこのメールを見ると、将来Windows向けにEmacsのこのあたりの設定が変わる可能性があるかもしれないが、とりあえず現状は自前で対策する。まず、coding-system-priority-listにjapanese-cp932が入ってないので入れる。これは(prefer-coding-system 'japanese-cp932)を実行すると勝手に最上位に入る。直後に(prefer-coding-system 'utf-8)等としても、coding-system-priority-list上でjapanese-cp932はjapanese-shift-jisより上のままなので問題ない。あとは、何かencodingをjanapese-shift-jisに設定しているものがあれば、ほいほいとjapanese-cp932に置きかえてゆく。

+ 次にSKK。SKKで入力する場合もWindowsならHORIZONTAL BAR優先がいい。でも、euc-jpやutf-8のバッファを使うかもしれないので、両方とも入力できるべきだ。つまりは辞書にEM DASHとHORIZONTAL BARの双方が入るといいのだが、skkservなどの辞書サーバはeuc-jp固定なのでこれは無理だろう(*2)。とりあえず(setq skk-jisyo-code 'utf-8-dos)として、.skk-jisyoをutf-8にして逃げる。ついでにskk-bayesian.elもbskkもeuc-jp決め打ちなのでこちらも変更する。

+ とりあえずこんなところか。Windows環境の話だが、非Windows環境でも実際のところ「shift-jisっぽいもの」は実際にはcp932で取り扱った方が幸せな事が多いと思うので、上記のことはやってしまってもいいかもしれないな。

*1: 一般にも。
*2: それともどこかにutf-8を取り扱えるskkserv亜種はあるのかな……。

■「Primary 〜Magical★Trouble★Scramble〜」(SkyFish)やりました。 [GAME]

+ プリム→ライム→マリア→ミンティア先輩→チャイム→日奈々の順で終了。

+ ライムは日奈々はなんか怖い。マリアはシナリオ的に弱い。チャイムは好きだけど主人公の琥珀がちょっとかわいそう。ということでプリムとミンティア先輩が好みであった。特にプリムは○○\化した時の甘えっぷりが防御不能。プリムもミンティア先輩もそうだけど、強気な娘が甘えてくるのは破壊力高いよなー。

+ お話としては「WIZ ANNIVERSARY」(Favorite)あたりと比べると「魔法のある世界」の描写が弱いと思う。ぶっちゃけ「はぴねす!」(ういんどみる)程度で全体に描写が軽い。そしてラスボスの底知れなさが足りない。そんなわけで軽い感じの印象の弱いゲームになってしまっているように思う。

+ 絵。いつものようにクセの強い絵ではあるが、俺は好きなんでまるで問題ないぜ。無個性な絵でないというのはプラスだと思うんだがどうだろうか。

+ システムまわり。今どきファイルを大量にばらまくシステムはどうだろうな。アーカイブできない深い理由があるのだろうか。メッセージ表示が瞬間表示のみなのは妙な割り切りだ(嫌いじゃないが)。あと、えちシーンの効果音とそれ以外の効果音は別のボリューム調整できるべきだと思うが、そうなっていないので、えちシーンの効果音がウザい。

+ ま、軽さは気になるが、きっと軽く楽しめばよいのだろう。プリムとミンティア先輩に免じて7点。
