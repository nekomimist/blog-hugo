Jan.8,2014 (Wed)

■艦これ。アルペジオコラボイベント終了 / まるゆ [GAME]

+ アルペジオコラボイベントが終わった。原作は読んでたので、キャラ自体への前知識があってまあ楽しめたかな。長波と予備のイクが手に入ったしね。

+ さて。通常海域は5-3まで終わってるから未踏破海域はない。通常建造で出てくる娘はもう全員うちにいる。未改造の娘は長波のみ。……装備開発と大型開発以外にやることがあまりなくなってしまった。

+ とりあえず大鳳狙いで大型開発やって、出たら46cm砲と電探と艦載機とソナーを充実させたいと思っているのだが、7回程度では大鳳は出てくれないようだ。まぁ確率10%なら7回では50%出ないし、5%だったら70%出ないしねぇ……。

+ ウィークリー任務のおかげで大型建造やっても全資源30kを上回る状態になったので、大型建造8回目。4000-2000-5000-5500/20。結果はまるゆ。……8回目にして初の新艦娘が来たが、嬉しくもあり、悲しくもあり。

+ まるゆ旗艦にしてちょろちょろ出撃してみた。なんだろう、この「守ってあげたい」感。

■艦これ。[GAME]

Jan.11,2014 (Sat)

■艦これ。/ 戦艦空母雷巡潜水艦抜きの4-2巡回 [GAME]

+ 大型艦建造9回目。レシピは4000/2000/5000/5000/20。2:40……祥鳳or瑞鳳か orz

+ 4-2巡回で重巡・軽巡・駆逐のレベル上げと戦果稼ぎ中。駆逐×2重巡×2軽巡×2で行ってる。下記のような、夜戦連撃を確保しつつ対空マシマシ対潜マシくらいな感じで行く。

  • 重巡は(20.3cm連装砲 or 15.5cm三連装副砲)×2 + 三式弾or対空電探×2
  • 軽巡は(20.3cm連装砲 or 15.5cm三連装副砲) + 10cm連装高角砲 + 対潜装備
  • 駆逐は10cm連装高角砲×2 + 対潜装備

+ ボス前の空母マスは輪形陣。陣形の対空補正と艦隊ボーナスの高い対空装備(三式弾・対空電探・高角砲)でほぼカスダメのみで済む。まぁこっちの砲撃もカスダメ祭りになりやすい陣形だけど、B勝利はできる。

+ ボスに行ったら昼は単縦陣で戦艦空母以外をなるべく落とし、夜戦で二人葬って終わり。昼戦だけだと勝利確定できなさそうなことが多いので、まず夜戦はしている。

+ 4-2は潜水艦×5で行く手もあるようだけど、潜水艦×6が出てくるとD敗北だし、高レベル重巡・軽巡・駆逐がいるほうが今後の海域党派やイベント攻略に役に立つはず。そもそもオリョールだけで潜水艦は十分育つしね。

+ 軽巡は五十鈴Lv80・長良Lv70・球磨Lv64・夕張Lv62・矢矧Lv59・多摩Lv51・阿武隈Lv45・阿賀野Lv40・能代Lv40あたりが戦力。那珂・神通・川内・鬼怒がLv30〜32なのでもうちょっと上げたいところ。

+ 駆逐はヴェールヌイLv82・島風Lv81・夕立Lv80・時雨Lv80・綾波Lv58・雪風Lv56がトップ6。鳥海さんに負けないくらい綾波が大好きな事に気付いて今さら雪風と共にレベル上げ中。敷波Lv36もかわいいのでもっと上げたいな。
