Jun.1,2003 (Sun)

■「夏少女」(PULLTOP) [GAME]

+ 明希→京→絢水の順で終了。

+ キャラとしては一番ちんまいくせに一番言動が大人びてる絢水ちゃんが好みなのだけど、話としては「二人の関係」が主軸かつ直球だった明希シナリオがよかったなーと。

+ 絵も音楽もシステムも大きなアラなくよくできているのだけれど、ちょっと地味かな? でも俺は好き。評価は7点で「佳作」。

Jun.2,2003 (Mon)

■RADEON9600PRO [COMP]

+ SPECTRA WX25の代替としてRADEON9600PROのカードを買った。DirectX9的視点ならともかくDirectX8的視点だと性能ダウンなのだが、まぁ、十分速いしOKOK。RADEON9800PROならDirectX8的にもDirectX9的にも性能アップだったと思うが値段と発熱の問題で回避。

+ ひさびさにATi系のカードを買ったのだが特に不具合もなく使えてたりしてちとつまらない。RAGE128や無印RADEONのトラブルぶりがなつかしいなぁ…。

+ ゆめりあベンチは"それなり/VGA"で17000くらいだったかな。細かい数値は忘れた :-)

■「夏少女」ぐるぐる [GAME]

+ 明希はキャラとシナリオ双方がよいですよねー。キャラ自体はむしろ苦手な方向なんですが、ゲーム内期間の3年でなんか好きにさせられました (^^;

+ 絢水・京シナリオも、明希シナリオっぽい彼女たちらしい成長&恋愛物語にしてくれてたらマジで俺的傑作級評価でしたねぇ。ちょっと惜しいです。

Jun.4,2003 (Wed)

■きょうのよしなしごと [COMP]

+ 同意見多数だろうがPainter8日本語版の値段がアレだな。英語版のPainter8は安くなったのに日本語版の値段は安くないんだねぇ。ちっ。

+ まぁ、英語版Painter8は現状バグバグらしいし、いろんな変更も賛否両論だし、日本語版アップグレードをもし買うにしてもしばらくは様子見だなぁ。

鈴 : 「だいだい猫くんは最近絵を描いてない!」
猫 : 「いや、まったくもってごもっともな指摘で…」
栞 : 「少しくらい否定してみればいいのに」

+ んで、Painter7Jのユーザ登録葉書をまだ出してなかったことに今気付いた。「夏少女」と「巫女みこナース」の葉書を出すついでにいっしょに出しておこっと(笑)

Jun.5,2003 (Thu)

■日本語CDDB問題 [COMP]

+ その昔似たような問題にブチ当たって、結局CD2WAV32を捨ててCDexに乗りかえました。CDexは「WINAMPがCDDBからgetしてきた情報」を読めますゆえ(*1)。ちなみにCDexについて知るには2chソフトウェア板のCDexスレを読むのが一番てっとり早いかと。

+ あと、有志による日本語のfreedbサーバもどっかにあったような…と、ここですね。

*1: CDDB2に直接アクセスしてるわけじゃないです。

Jun.8,2003 (Sun)

■「CANDY TOYS」(ivory/jANIS) [GAME]

+ 「シナリオ:嘘屋 佐々木酒人」。これだけで女の子の独白が変態おバカ風味なの確定だと思ってたら、期待はまったく裏切られなかった (^^;

+ …まぁ、「誕生日〜通い妻(自称)日記〜」のCV:浅見ちほ(青山ゆかり)による変態セリフ独演会ぶりにくらべるとだいぶパワーは落ちているのだが、エロゲ的にはこの方が正しいだろう(笑)

+ 馴れてないご主人様と奴隷sでらぶらぶ変態調教って感じ。なにやっても最終的にはかなたもレニーも明るい雰囲気(←重要)でよろこんじゃうんしー。

+ すばらしいとは言わんけど、なんとも楽しいゲームでございました。7点で佳作かな。

Jun.14,2003 (Sat)

■土曜日にぽしごと [LIFE]

+ 隣の技術部のヘルプとして、朝から試作機の改造にかり出されてみたり。

Jun.16,2003 (Mon)

■「Orange Pocket」(HOOK) その1 [GAME]

+ 最初はふらふらとツリ目でちんまいの羽弥先輩方面に。……なんかジェットコースター風味。とにかく「嘘くせぇ」話だったなぁ。キャラは好きなんだが。しかしプレミアH\の脈絡のなさに乾杯。着替えるかよあのタイミングで(笑)

+ 2回目は桐子方面に。…ぐはっ、激しくかわいいなこの娘 (^^; 羽弥シナリオと違ってむちゃくちゃじゃないし(最後青くさいけど)、すずもじいちゃんはいい味出してるし、よいねー。プレミアH\は……そういう状況で着るか?(笑)

+ 次は誰やろうかな。

■「Orange Pocket」(HOOK) その2 [GAME]

+ 平日の昼間っからエロゲ。健康的だ(ぉぃ

+ 3回目は妹分の恵留(エル)。いい妹ではあるのだが、エルよりむしろ美典に惹かれた (^^; 話は…主人公も最初から知っていて二人で乗りこえる展開の方が俺には好みだったかな。てゆーか逃げんな＞主人公。

+ 4回目は美典。何も事件が起きない話(*1)なのだけれど、関係の積み重ね様がなんか好きだったなー。しかし、この娘との最後のえちシーンは、プレミアであってもなくても奇妙だ(笑)

+ 5回目は円。わりとどーでもよかったのだが、くっつく直前あたりから俺のツボをぐっと突くかわいさが。やっぱ元気で強気な娘がこういうふうになるのは良い。しかし、最終盤プレイヤーを放り投げ気味なのが難点だが、最後の一枚絵ないい感じなので許す。

+ しかし円は音声の音割れがヒドい。収録時のレベルオーバーかのう。

*1: 美典自身が乗りこえなければいけない「ある事」はあるのだが。

Jun.18,2003 (Wed)

■きょうのよしなしごと [COMP]

+ 「ショコラ」の話、イベントを見るために必要となる空振りの量が「Ripple」よりぐっと増えてますんで、狙ってやっている可能性はありますね。嬉しくもなんともないですが :-(

+ 一応書いとこ。「こころナビ」(Q-X)は買う気満々ですでに予約済みっす。でも、日記には「終わった。面白かった」って感じの記述しかされない可能性も高いです (^^;

■初夏の物欲 [AUDIO]

+ 今は携帯音楽プレーヤとしてGIGABEATを使っている。容量が5GBしかないので、"lame --alt-preset standard"でエンコードしたMP3を96kbps WMAに変換してなんとか押しこんでいたのだが、ついにめでたく5GBにおさまらなくなった :-(

+ やっぱiPod買うかなー。今あるMP3を詰め込むだけで10GB近くあって15GBモデルでは心許無いから30GBモデルかな。厚くて重いっつーてもGIGABEATよりは薄くて軽いし :-) 左右反転バグが直ったら買おう。

■椅子 [MISC]

+ 部屋の椅子が壊れつつある。寮の各部屋にdefaultで置いてあるイスなので、もともとロクでもないのだが、壊れたせいでより無理な姿勢になってしまうのか、最近肩と腰の調子が怪しい。

+ 管理長に壊れた旨を伝えたので代わりがくるのかもしれないが、やはり毎日使うモノだし、ケチらずに予算10万〜15万あたりでよい椅子を買うかのう。

+ と、Aeron Chairとか、Leapとか、Pleaseとかの資料を漁る。結局は座ってみないとわからんのだけどな。

Jun.24,2003 (Tue)

■iPod、届く [AUDIO]

+ 2時間ほど残業した後に帰宅すると、iPodが届いてたので、ファームを2.01にアップデートして、データを転送してみる。

+ そしたら転送中に電池が切れて(*1)、俺も切れて(笑)、HIMIKO用のIEEE1394カードを発注してしまった (^^; USB2.0ケーブルを買えばほんのちょっとだけ安く済むけど、クレードルのしっぽが2本になるのも美しくないしね。

+ ついでに大き目の傷をつけて泣いてみたりもした (;_;)

*1: うちにあるIEEE1394つきのWindowsマシンはMACHIKAだけで、こいつはiPodに給電できないんだよね。

Jun.28,2003 (Sat)

■「こころナビ」(Q-X)を始めてみる [GAME]

+ AlphaROMか。でも、チェックするのはインストール時だけみたいなので、許せないこともない。うちの環境はどのプロテクトでも誤爆くらったことないしねぇ。

+ どーもクリックに対する反応が微妙に鈍くてイラつく。直前に「初恋」(RUNE)のリプレイをしてたせいで落差が大きいなぁ。RUNEのシステムが万全だとは思わんけど、なんか俺と相性がいいんだよなぁアレは。

+ さて、誰から行こうかな。

■「こころナビ」(Q-X)さいしょ [GAME]

+ 1stプレイはアイノ狙いで進行。ちんまくてよい(ぉぃ

+ んー、共通部分(フラグ立て部分)が長いかなぁ。そのせいか後半(個別部)がちょっとアンバランスかも。あと、パートボイスにもほどがあるくらいに声がない。まぁ、デフォルトでOFFになってるあたりからしても「おまけ」ということなのだろう。

+ さておきアイノ。特に何があるという話ではないのだが\、雰囲気は悪くないねー。

+ 2ndプレイは夢。二人のラブラブ度が高いかな。ちょっと片のつきかたが御都合主義的すぎな気もするが。あと、俺はあっちの姿の方が好きだ(ぉぃ

Jun.29,2003 (Sun)

■「こころナビ」(Q-X) さいご / 軽く総評 [GAME]

+ 3rdプレイはみまりさん。実にいい感じのお姉さんでございました。

+ 4thプレイは凛子。言動といい容姿といい実にいい実妹。やはり21世紀は実妹の世紀なのかっ(ぉぃ

+ 5thプレイは小春。黒髪ショートでちんまいくせに巨乳。実にオレ好みだ。ちなみにラウンダーの方も一番好み\。終盤のあの展開はもっとひっぱるべきだよなー。

+ 最後にルファナ。真相はこの娘の話で明かされるわけだが……このテの性格の娘が苦手せいで、ちょっとテンションが低くなって困った。

+ これにて終了。イチオシキャラはコナさん……って、「たいせつなうた」の真千香ちゃんに続いてまた脇役がイチオシかよ＞俺 (^^; あとは小春=凛子>みまり=アイノ>ルファナ>夢な感じ。夢のラウンダーは好きなんだがなー。

+ 「総じてシステムがイマイチ」とか「質的にも量的にも中途半端なボイスってどうよ?」とか、文句はいろいろあるけれど、そこそこ楽しめたからまあいいか。7点で佳作かな。
