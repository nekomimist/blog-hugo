May.1,2006 (Mon)

■最近やったゲーム [GAME]

+ 日記に書いてなかったけど先月あたりにやってたモノをまとめて。

+ 「ボーイミーツガール」(FrontWing)。美羽と七海を残してなんとなく停止中。イチオシは圧倒的に真央(CV:草柳順子)やね。草柳順子が声アテてるキャラにしてはすっげーひさびさにイチオシ :-) あと、陽奈が大人しそうなくせに激しくエロい幼なじみなのがよい感じ。観村咲子がここまでえろいキャラに声アテてた記憶はなかったりする。しかし残り二人を倒す日がくるかどうか不明だ。むんっ。

+ 「ときたまふぁんたずむ」(あてゅ・わぁくす)。これまた幼なじみの美杏が見た目に反してえろい娘で良し。しかし俺にとっては会長がイチオシなのだった。……会長の話がどんなだったかなんてまったく覚えてないけどな(ぉ ま、なんというか、とても"えろゲ的フツー"なゲームだった感じ。

+ 「青空の見える丘」(feng)。fengなのに存外まともでちょっと期待外れ(ぉ 主人公が「(誰にとっても)言ってほしいことを言いすぎ」であまりにも作り物臭いというあたりと、シナリオの締め方が全般に唐突気味なあたりが(*1)難点ではあるが。えちシーンは数はあるけどなんか軽いので、1回減らしててもねっちり描写してほしかったかも。キャラのイチオシは……ほぼ横一線ではあるのものハナ差ですいすいかな。

+ 「ちっちゃいメイドさん」(ハートブリング)。ここまでタイトルと絵から想像できる話そのまんまなゲームだといっそすがすがしい。値段に比して分量的なものたりなさが相当あるのだけれど、俺的にはちまりさんがかわいかったのでなんか許す(ぉぃ

+ で、現在「WIZ ANNIVERSARY」(CROSSNET)進行中。今んとこ、なかなか面白いデスよ。

May.2,2006 (Tue)

■「WIZ ANNIVERSARY」(CROSSNET) [GAME]

+ CROSSNET/Favoriteってことは、前作は2004年6月の「Angel Wish」。ほぼ2年ぶりですかな。

+ さて、最初はローラ先輩・ソフィア方面に進んでみようといいつつ開始。物語的な「展開」が始まる大賢者騒動までがちとタル目な感じだな。

+ で、まず先輩エンド。……いかにもな人が黒幕で苦笑。そして奴の正体がアレなのもなかなか\。このルート、先輩が途中退場気味かつ、後半はソフィアがいい感じに印象的で行動的なせいでソフィアの方に心が動いてしまって困った。オチはまぁお約束\だが、幸せそうなのでまぁ良し。

+ 続いて同ルートをソフィア寄りの選択肢で進めてソフィアエンド。ソフィアは真面目で積極的でえろすでいい娘でした。しかしまぁ、さすがにローラ先輩がかわいそうですな、この展開だと。デュオは\直後だっつーのにいくらなんでもふっきれすぎだし。まぁ、落ち込むデュオと励ますソフィアなんてのを入れてもテンポ悪くなるだけなので、これはこれでアリかもな。

+ 次はアリス方面。……と思ったらアリスルート自体は順序制御で閉じてるっぽい。そんなわけで、アリス方面から派生するディアナ先生から。ドジな新米教師なんて正直どうでもいいと思っていたのだが、そこに驚愕の真相が! うん、そっちの方が俺好みです。続いて同じ所からの分岐でグラニテ。こちらも正直どうでもいいと思ってたのだがとても初心で存外かわいかった。しかし、デュオも評してたけどこの娘はわんこだよね。この二人、話としてはとても軽いので息抜きにやるのがよい感じ。

+ アリスルート解放は多分最後だろうから、先にシャル方面を進めよう。「這い寄る"赫"」っつーのはあっちネタリスペクトだろうか。過去の事件と、その流れで語られるシャルとエンフィの関係。……こりゃ、エンフィがシャルと対立しやすいのも当然だわなー\。話の流れ自体はむちゃくちゃ奇抜ってわけではないのだが、いろいろ語られるバックグラウンドがわりときちっと作ってあって軽くなりずぎないようになっているのはよかったと思うね。

+ さておき、シャルは素直になれない系で実にかわいい。"影"との対決後の不安定なシャル→ゴンドラレースでの約束→その後の流れの中のシャルとか、最後の対決前のシャルとか、ものすごくかわいくて良し。最後は"らしく"終わってるのも良し。

+ 続いて同じルートをエンフィ寄りの選択をしつづけてエンフィエンド。……二人が対になりすぎててシャルルートとえちシーン以外ほとんど同じだなー。しかし制服姿はよいねー。あと、エンディングのバカップルぶりがすばらしい :-)

+ さて最後はアリス。なんとなくだがアリスの最終分岐に至るまででアリスとデュオの後は見えてるんだけど……。ふむ、やはりそういう展開だ。そして王道といえば王道の結末に。もうちょっと直截的にハッピーな方が俺の嗜好に合うのだが、これはこれで悪くなかろう。この終わり方を「鬱」とか言う人には「違うだろ」と言いたい。……ただまぁ、アリスのキャラ作り自体は苦手な方向なので、イマイチお話に入り込めなかった感はあるな。

+ で、あとは妄想な分岐など回収してこれにて終了。

+ さて感想。前作「Angel Wish」は絵はともかく他が微妙に残念なゲームだったけれど、今作「WIZ ANNIVERSARY」は今回えろ分もおろそかにせず、かつ話もきちんと作ったって感じがするね。

+ 塗りは独特な風合いだが、絵と話には合っていたと思う。あいかわらずキャラがかわいいしのがよい。CVも俺はよく知らない人が多めなのだが悪くないと思う。

+ 前回も絵の拡大縮小を多用していたけれど、今回は背景と立ち絵とイベント絵の拡大縮小回転をテンポよく多用している。重くもじれったくもないのでよい感じだったと思う。CPUとGPUが非力な環境で重くもじれったくもないかは知らんが :-)

+ キャラ的に好みなのはシャル・エンフィ・ソフィアの3人は甲乙つけがたく好み。グラニテ・ディアナ先生は普通な感じ。アリスとローラ先輩はちと苦手な系統。

+ 評価は「9点にはちょっと届かない8点」ってところか。ま、俺にとってとてもよいゲームでした。まる。

May.7,2006 (Sun)

■「Triptych」(ALcot) [GAME]

+ 順路通り\で終了。

+ 前作「Clover Heart's」の時よりヒラメ顔に感じるなー(ぉぃ 絵柄の方向性自体は悪くないと思うんだがなぁ……。一方服装なんかは見慣れない感じで悪くない。CVは夏野こおりとみると佐々留美子と俺の好きな人達なので特に文句はない。

+ 話としては……やりたかった事に何もかもが追いついてないんだろうか。緊迫してたり、気味悪かったり、血腥かったり、絶望的だったりしてる状況のはずなのに、どうにもそれが伝わってこない。演出も主人公やヒロイン達の行動も軽く見える。おかげでプレイ中醒めっぱなしだった。このテの話であれば、もっともっと「重苦しさ」を感じさせてくれないとダメだろう。例えばサイオンなんぞは存在自体がいらん。脇役とのえちシーンもAppendixだけに留めたほうがよかったのでは、と思う。

+ 総評としては……陳腐な話だったなぁとしか言えん。ま、絵と声にピンと来ちゃったら買ってもいいんじゃない?という程度だな。評価は6点。

May.28,2006 (Sun)

■「七彩かなた」(千世) [GAME]

+ 攻略順はほぼ固定っぽい。最初(ひまわり)→ここたん→祭→夕日→菫→ひまわり→ラストと進めて、最後に全員のここたん、祭、夕日、菫の別エンドを回収して終了。

+ いわゆるひとつのループ物だが、現代と平安を行き来するあたりが特徴か。前作「六ツ星きらり」は輝夜姫がネタになったわけだけれど、今回は後の紫式部たる菫姫がネタ。言葉が現代にそまってゆくのが実に愛いですよ菫。

+ 話の中に前作「六ツ星きらり」をやってるとニヤリとできるシーンがいくつか。祭が歌った歌とか、着メロとか、ここたん終盤の移動マップとか、天文部の話とか。

+ あとはつらつらと。

  • ここたん。内面はけっこうアレ\なんですな。しかしこの\ネタ、どっかで類似の物を見たような……?
  • 祭方面は半端に涼一・夕日のイベントが出てきて"ごちゃついた"感じがした。ハナから国丸視点でよかったんじゃないか、という気がする。
  • 夕日。不器用で意地っぱりで実によい。しかし、捨てセリフ発して逃げ去る姿はToHeart2の由真を彷彿とさせる……。
  • 菫姫。わがまま姫様ラブ。洋服姿も実に愛いし、「にいさま」と呼んでくれるようになるのもとてもよい。今作の俺的イチオシかな。ひまわりと夕日も捨て難いのだが。
  • ひまわり。ゲーム中、ずっとずっと不憫な目にあう娘なので、最後に幸せになれてよかったなーという感じが。
  • 何度も何度もよく落ちるシステムであった。使用感は悪くないんだけどねー。
  • イベントCGは絵柄的には一般受けする方向にシフトしつつも全般的にあかぬけない感じがする。まぁ、これも味だよな(←こういうのを信者補正と言うのだろう)。
  • 最後に向かうにつれ、俺の気分がダレていっちゃったのは何なんだろう。核心から解決に至るまでの道筋はもっとガチっと描写しちゃってよかったかも(*1)。そのほうが最後までハラハラした気分でプレイできたような気がする。
  • というか登場キャラ全員に緊張感が足りないよな。
  • シナリオの不整合もちょっと目立つかな。知らないはずのことを知ってたりとか。

+ ま、総合的に言えば、不満な点も多いけれど、全般的に見て「千世らしい」感じがしてたのでまあいいかという感じだろうか。評価は7点で「佳作」かな。「六ツ星きらり」越えはならず、という感じ。

■SoundBlaster X-Fi死亡? [COMP]

+ HIMIKOで「七彩かなた」プレイ中、突然音が鳴らなくなったのでリブートしてみたら、その後WindowsからSoundBlaster X-Fiを正しく認識できなくなった。抜き差ししてもドライバを入れなおしてもだめ。……壊れたか? 別のWindows環境で確認してみたいところだが、残念ながらうちには「PCIスロットを持つWindowsマシン」は1台しかなかったりするのだった。

+ 死亡確認は後まわしにして、とりあえずHIMIKOはオンボードの音源で逃げることにする。幸いデジタル出力を持っているM/Bなので、LavryBlack経由で聴けるので(44.1KHz→48KHzのSRCを通ってしまうものの)ノイズとは無縁だ。まぁ、えろゲ用なら許せるかな。

*1: 菫方面はちょっとぼかす必要はありそうだが。
