Jan.16,2016 (Sat)

■モバイルPC購入に悩む [COMP]

+ さすがにIvyBridge時代のVAIO Duo 11。使い勝手はそう悪くないのだが、古くもなってきたので、新しいものが欲しい。

  • タッチパネルは捨てられない
  • 画面比16:9はイヤだ (VAIO Duo 11でイヤな点その1)
  • ファンは静かなのがいい (VAIO Duo 11でイヤな点その2)。でもサーマルスロットリングは勘弁な
  • Core i7-3517Uより遅いのはイヤだな
  • Windows Helloでカメラでログインしたい!
  • ペンは欲しい
  • タブレットモードも欲しい

+ という感じだが、

  • Surface Pro4 : i7モデルなら×は一切なしと思われる。
  • Surface Book : タブレット単体の使用性がSurface Pro4より悪そうなのが×。
  • VAIO Z Canvas : 性能は文句なし。ちょっと重い。Helloがないのが×。
  • Thinkpad X1 Tablet : Core m7はどうだろう?? Helloが×。Lenovoはなあ。
  • Thinkpad X1 Yoga : 14型なのは○だが16:9なので×。Helloが×。Lenovoはなあ。
  • Dell 新XPS 12 : 液晶が12型で16:9で×。Helloも×かな。
  • HP Spectre13 : 液晶が13型でFullHDで×。Helloも×かな。

+ Surface Pro4一択だなあやはり。

Jan.24,2016 (Sun)

■艦これ。12月作戦褒賞 [GAME]

+ 12月作戦も500位以内に入ったので15.2cm連装砲改と16inch三連装砲Mk.7をもらった。

+ もらってうれしい15.2cm連装砲改だが、これで5本目だ。手持ちの4本の改修状況は+7/+7/+6/+1。イマイチ進んでないが、PT小鬼対策でもあるし、頑張って改修を進めておきますかね。

+ そして16inch三連装砲Mk.7。米国最後の戦艦砲のアイオワ級搭載砲やね。艦娘の登場も春と指定されたわけで、4年目の艦これは多国籍化が進むのかな。英艦も仏艦や独艦や伊艦も頼むぜ!
