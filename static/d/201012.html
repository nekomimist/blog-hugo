Dec.6,2010 (Mon)

■自転車のこと [BIKE]

+ 通勤用のAbios Tourerは現状50-36×13-26(8s)だが、実際のところ、通勤中にフロントをインナーに入れることはないし、リアを13に入れる事も少ない。

+ そんなわけでチェーンリングが痛んできた今、44x12-25(10s)くらいに交換するとよいのかもしれない。ブレーキレバー(LR)、10sシフトレバー(R)、チェーンリング、スプロケットの交換になる。ついでに痛んできたクランク・ペダルも交換できればなおよし。

■電話機のこと [MISC]

+ Find My iPhoneの無料範囲に「iPhone3GSだけ使っている人」が入れないのはあんまりだし、日常的にiPhone3GSの電波の掴みの弱さに悩んでいるのもあるし、やはり端末を変えることを本気で考えようか。

+ iPhone3GSは2年縛り残り分だけ維持して、追加でAndroid端末を買うのがいいのかな。SoftbankはiPhone3GSと同じだからイマイチで、DoCoMoのシャープ製Android端末(SH-03C)がデザイン的に気に入らないので、消去法でAUのIS03あたりにだろうか。自社製に拘らなければもうちょい選択肢があるんだが……。

Dec.12,2010 (Sun)

■きょうのよしなしごと [COMP][MISC]

+ なんとなく、SugarSyncの招待リンクを貼ってみる。

+ 元気不足なのだが、家の中にホコリが積もってきたので仕方なく家中を掃除して、一部フローリングをワックス拭きしてみた。……フローリングをもうちょっとマジメにきれいにしてからワックスかけないとあまり意味がないな。

+ 11時間以上寝たのだが、なんかまだ寝たりない。正直今年5月から仕事が山場続きなので精神的な疲れが抜けない。俺の最高年収は一昨年なのだが、ボーナスが一昨年より大幅に少ない今年もその金額に並んでしまいそうだ。

Dec.25,2010 (Sat)

■バックアップの話 [COMP]

+ 現状自宅PCは全てWindows Home Serverにバックアップされているのだが、これだとうちのマンションが焼けたり崩壊したりしたら全滅する運命だ。できればリモートにもデータを退避先が欲しいのだが、TrendMicroのSafeSyncはどんなもんだろうな。

+ そういえば、最近XmarksがLastpassに買収されたわけだが、LastpassもXmarksも愛用する俺としては、サポートのためにプレミアム会員になってみるのもよいかもな。合わせて年間$20程度だし。これもある意味バックアップ。

■散財したい -- PC編 [COMP]

+ ボーナス月だし残業80時間やったしで今月の収入が多かったので、いっちょ散財したい。

+ PC系はSandyBridge待ちなので発動は来月以降だが、Core i7 2600K+M/B+DDR2 SDRAM 16GBをたぶん買う。SSD(Intel X25-M G2 160GB)とグラフィックカード(RADEON HD5850)とサウンドカード(XONAR D2X)などは特に不満はないのでとりあえずは現状のまま。

+ 現状のCore2 Extreme QX9650+Rampage Formula+メモリ8GBな構成は、そのままKONAさんの中身になってもらおう。現在のKONAさんはAthlon X2 5050eなので、かなりのCPU強化になるので、ZFS+iSCSIの性能が上がればいいな。

+ 余ったML115G1のAthlon X2 5050eマシンは使い道がない。実家に置いてバックアップマシンにするのもよいかもしれんがそのままだと爆音なので手におえない。安定動作はしているので捨てるのも偲びない。うーん。

+ ノートのDELL XPS M1330もそろそろ変えたいが、これもSandyBridge待ちだろう。Sandyなら内蔵GPUで大丈夫そうなので選択肢が広がりそうだ。まぁ、こいつは来年の夏ボあたりが買い変えポイントだろうから、しばらくこのままでよろしい。

■きょうのよしなしごと [MISC]

+ ふるかえってみると、今年はやけに土曜出勤が多かった。そして最後の土曜日も出勤だった。ここまで働いても一昨年の年収を超えられないんだから、ボーナスの差は生め難い。

Dec.31,2010 (Fri)

■2010年も終わりですな [MISC][COMP][BIKE]

+ いや、なんつーか忙しい年だった。ほとんど仕事漬け。そして来年も1/2から仕事でデバッグ。いい年になりそうだなぁ orz。

+ 忙しかったのでゲームその他の消化状況は大変低調だった。気になるラノベを買っても消化できないので買うのを絞ったりもした。CD買っても積みっぱなしとかなー。

+ 買ったものリストを眺めると自転車関係が目立つ。RNC7フレームにアルテ一色。工具。ケミカル。ウェア。ヘルメット等々。いろいろ揃えたからには、来年もうちょっと走りに行きたいもんだ。

+ PCがらみの出費は低調かと思ったが、メインマシンのFIENAの構成を変えてないだけで、サブのFreeBSDマシンKONAにはRocketRAID 2314とSeagate Barracuda LP ST32000542AS×4と追加してたので金額的にはそこそこか。でも機能的にはHDDが増えただけだしなー。

+ ま、来年もいろいろ仕事が目白押しである。腕と知識を磨いてそれなりに役に立つ存在であり続ける事と、次の人材を育てることあたりがテーマかね。
