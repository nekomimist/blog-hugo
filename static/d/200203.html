■「僕夏」のOPとMP3と携帯プレーヤ [GAME][AUDIO]

+ 「そういや『僕夏』のOPの曲をMP3化してなかったなぁ」ということでやる。最初に「lame --alt-preset 128 boku.wav」って感じでencode。ちなみにlameはversion 3.91。

+ 曲の感じから予想はしていたが、DeltaDiO2496のanalog out→STAX Lambda Nova Signature(*1)で聴くとダメさが際立つ。いきなりMPIOで聴いてたら気にしなかったかもしれんが、知ってしまった以上もうダメだ(ぉ

+ 結局「lame --alt-preset standard boku.wav」で逃げる。これで171kbps。結局のところMP3は192kbps近辺からが実用域だなぁと再確認した次第。

+ …ああ、容量のデカい携帯プレーヤが欲しい。

*1: SR-007は未修理なのだ。はよやらんとな…。

■Windowsメイン環境をAKANEからHIMIKOに移す (実行編) [COMP]

+ 通常使用時とゲーム時はユーザーを分けるとする。

+ さて、ここで問題「WinXP対応ゲームはAdmin権限なしでインストール/実行できるか?」。。試してはいないが、まともに動かないものが存在しそうなので、ゲーム用のユーザーはAdminグループに入れておく。一方、それ以外の作業用の方はPower Usersグループに入れる。

+ で、AKANEからががっといろいろコピーして、さっきの権限分けを前提に各所のディレクトリのアクセス権限を弄りまくってとりあえず作業終了。にしてもNTFSのアクセス権設定はややこしい。

+ あとはAKANEからCD-RやらDeltaDiO2496やらSSH-Type.Bやらをひっこ抜いてHIMIKOに移して、AKANEにFreeBSD-4.5Rを入れれば当初の予定は完了かな。

■Windowsメイン環境をAKANEからHIMIKOに移す (ハードウェア編) [COMP]

+ まずはAKANEから。

+ SPECTRA8800を外してSSH Type.BをSSH Type.Dに交換して戻す。SPECTRA5400PEの頃から延々使いまわされるSSH Type.BとBB-75…実はC/Pは高かったのか?

+ さらに、M-AUDIO DeltaDiO2496とPLEXTOR PX-W2410Aを外す。キューブケースだっつーにずいぶん5inchベイを占有するデバイスが減っちまったなぁ…ってことで、昔外したTR4ドライブをつっこむ。非圧縮4GB/圧縮8GBとはいえ、そこそこ使えるやろ。

+ 起動、SCSIカードがTR4ドライブを認識してくれない…と、よく見るとMOドライブも×でnarrowデバイス全滅。んー、ケーブルまわりかのう。まぁ、即必要なものではないし、HDDとはSCSIチャネルが違うので気にしないでしばらく放置決定 :-)

+ 続いてHIMIKO。

+ SPECTRA X20のSSH Type.DをSSH Type.Bに交換。SSH Type.Bはメインマシンの証(ぉ

+ M-AUDIO DeltaDiO2496ととPLEXTOR PX-W2410Aを移動。フロントの5inchベイ×4が埋まっちまったなぁ…電源足りるのか微妙に不安に。電源ファンがうるさいし大容量かつ静かな電源にとりかえるかなぁ。

+ 起動。む、M-AUDIO DeltaDiO2496は認識するが、差してあったLive!をみつけてくれない。PCIカードの場所をとっかえひっかえするとなんとな両方認識。あぶないあぶない。

+ これにて終了。今後のToDoは以下のとおり。

  • AKANEのIDEとNarrowSCSIのケーブル交換。
  • HIMIKOの電源交換。

+ また散財だな (^^;

■数式エディタの話 [COMP]

+ Officeなどに標準で入ってる「数式エディタ」は「一般のビジネス文書に入る数式程度ならこれでよかろう」という感じで入ってるだけじゃないかな。で、あれはMathTypeの「とってもサブセット」版なはずなので、複雑な数式を書くときはMathTypeを買って使うのが正しい道かと。

+ MathTypeどころか「数式エディタ」もロクに使ったことはないので、MathTypeがどれだけ「まとも」かは知らないし、俺が知ってるくらいの話なので既知な可能性が高いとは思うけれど。
