Feb.6,2014 (Thu)

■艦これ。 [GAME]

+ 衣笠さんがLv順1ページ目に入った。レベル上位陣に雷巡がいないのが特徴だろうか。長門型が上位にいないのもめずらしいかもしれん。

+ 衣笠さんも、鳥海さんLv99と同じ20.3cm(3号)砲+15.5三連装副砲×2+14号電探にしたいし、伊勢型・扶桑型・金剛型・長門型・大和型には46cm砲を2個づつ積みたい。ということで、大型建造はしばらくお休みしてデイリーで電探と46cm砲開発をすることにした。まあ14号も32号もなかなか出ないんだけどさ……。

Feb.9,2014 (Sun)

■艦これ。重巡装備 [GAME]

+ こんなこと書いたけど、20.3cm(3号)連装砲+15.5cm三連装副砲×3がイイという話を聞いた。

  • 20.3cm(3号)連装砲+15.5三連装副砲×3 = 火力+30/命中+6/対空+14/対空艦隊ボーナス2.6
  • 20.3cm(3号)連装砲+15.5三連装副砲×2+14号電探 = 火力+23/命中+8/対空+17/対空艦隊ボーナス3.5
  • 20.3cm(3号)連装砲+15.5三連装副砲×2+三式弾 = 火力+23/命中+4/対空+16/対空艦隊ボーナス4.1

+ 火力+30が乗ると、鳥海摩耶は火力108、高雄愛宕は火力107、衣笠は火力104になる。エリルとフラヲの装甲を抜けるケースが増えそうではあるなぁ。

+ で、Lv96の衣笠でトライしてみたら、たしかに案外悪くないような気がする。主力重巡は鳥海Lv99・衣笠Lv96・摩耶Lv94・愛宕Lv78・高雄Lv76なので、15.5cm三連装副砲が15本いることになる。軽巡雷巡から奪えばそれくらいあるが、まあ、素直に副砲を増産しよう。

Feb.14,2014 (Fri)

■艦これ。ケッコンカッコカリ [GAME]

+ さて、ケッコンカッコカリ実装日である。

+ 最終準備任務はLv99鳥海・Lv97衣笠・Lv51古鷹・Lv50青葉・Lv42加古・Lv35天龍で完了させてみた。ちなみに新三川任務完了は2013/10/1。Lv47鳥海・Lv27衣笠・Lv25古鷹・Lv25青葉・Lv25加古・Lv23天龍であった。

+ そして鳥海さんがLv100へ。イベントボイスといい、母港ボイスといい、たいへんよいねぇ。ついでに耐久が7増えて64になり、運が5増えて15になった。

+ そして「二人でする初めての任務」で4-3ボスS勝利を目指すが、羅針盤とフラ重が荒ぶって無理だった。まあ、のんびりやるか……。

Feb.15,2014 (Sat)

■艦これ、約半年 [GAME]

+ リンガ泊地サーバの開設は2013/8/17なので、そろそろ開設半年記念である。

  • 空母 赤城Lv97/加賀Lv96/瑞鶴Lv94/蒼龍Lv92/翔鶴Lv91/飛龍Lv90/大鳳Lv87
  • 軽空 千歳航改二Lv89/千代田改二Lv89/飛鷹改Lv89/隼鷹改Lv89/以下略
  • 戦艦 金剛改二Lv97/霧島改Lv96/比叡改二Lv95/榛名改Lv94/陸奥改Lv83/武蔵Lv73/長門68
  • 航戦 伊勢改Lv97/日向改Lv96/扶桑Lv87/山城Lv87
  • 航巡 鈴谷改Lv52/最上改Lv51/熊野Lv45/三隈Lv44
  • 重巡 鳥海改Lv106/衣笠改二Lv98/摩耶改Lv95/愛宕改Lv80/高雄改Lv77/妙高Lv61/利根Lv61/古鷹Lv51/青葉Lv50/以下略
  • 雷巡 大井改二Lv92/北上改二Lv92/木曾改二Lv88
  • 軽巡 五十鈴改二Lv85/長良改Lv78/球磨改Lv74/由良Lv73/夕張改Lv73/矢矧Lv67/以下略
  • 駆逐 ヴェールヌイLv89/島風改Lv88/夕立改二Lv87/時雨改二Lv86/雪風改Lv72/綾波Lv71/夕雲Lv67
  • 潜水 伊168改Lv80/伊58改Lv77/伊19改Lv74/伊8改Lv72/伊401改Lv66/まるゆLv25
  • その他 あきつ丸Lv45/千代田甲Lv38/千歳Lv36

+ 2ヶ月前の状況はこちら。

+ 空母の上位はLvが1〜2しか上がってないのは、下位のレベル向上に励んだせいである。2ヶ月前には存在しなかった大鳳がLv87になっている。とりあえず加賀と大鳳はケッコンカッコカリで課金する気満々である。

+ 軽空母は対潜が強化されたので4-2巡回に投入してレベルがけっこう上がったなと。鉄火場には入れないけど、祥鳳・龍驤・瑞鳳・鳳翔もけっこうレベルが上がっている。コスト低くて便利。

+ 戦艦の上位もLvが1しか上がっていないのは、あまり出撃しなかったからだなぁ。やっと建造できた長門の育成はしたけど、Lv68なのであまり育ってはいない。

+ 航戦の上位もLvが1しか上がっていない。扶桑山城はちくちく育成していたので、Lv99までの50%の経験値を全員稼いでいることになる。航戦、好きです。潜水艦の演習も航戦出したばっかだしー。

+ 重巡。俺は重巡が大好きだ! 鳥海はケッコンカッコカリしたので一人だけLv99超えのLv106である。衣笠は次のケッコンカッコカリ相手の予定だが、Lv98→Lv99はマジつらい。摩耶ともいつかケッコンカッコカリしたい.それ以下は、まあ、地道に育成中。

+ 航巡は不遇なままだなぁ。2-2での南西任務には投入してるのだが……。

+ 雷巡は木曾以外は隠居気味。強いのは判っているのだが、「あまり頼りたくない」気分が強い。まあ全員戦力になるレベルなので、いざって時は出撃させられるけど。

+ 軽巡はLv70超えがだいぶ増えたので、五十鈴長良以外は鉄火場海域に出せないという状況は脱したかな。ただ、五十鈴ですらLv99への必要経験値の50%稼いでないので、不遇ではあるかな。

+ 駆逐艦はわりと力を入れて育成したので、トップ4がなかよくLv10up。雪風がやっとLv70超え。性格に惚れた綾波・夕雲もかなりレベルが上がってきた。駆逐艦は好きな娘が多いので悩ましい……。

Feb.17,2014 (Mon)

■艦これ。大和出た [GAME]

+ 鳥海さんの次を狙う衣笠さんがLv99まであと経験値52000くらい。もうちょっと?と一瞬思ったが何か麻痺している気がする。

+ 鳥海さんの「初めての二人の任務」の随伴と、衣笠さん経験値稼ぎのために4-2/5-2巡回に投入していた加賀さんがLv97になった。このまま加賀さんは三人目にLv99になってもらおう。

+ 大和狙いレシピ(4000/6000/6000/3000/20)で回しはじめて6回目で大和が出た。ヘンな声も出た。これで残りは卯月のみ。衣笠さんの経験値稼ぎを兼ねて4-4か5-1へ掘りに行くか……。

Feb.19,2014 (Wed)

■艦これ。 [GAME]

+ 大和を演習でレベル上げ中。改Lvが60なのでちょっと遠い。

+ 衣笠さんも無事Lv99になったのでケッコンカッコカリした。次は加賀さんLv97になりそうだが、まあ、無理してレベルを上げるのは止めといて、他の娘のレベル上げをしよう。

+ 2/26のメンテで新海域解放とのことだけど、育ってないのは航巡だけって感じなので、これといってやらなきゃいけないことはない。というわけで、デイリー開発+αで46cm砲レシピを回しているが40回連続でゴミしか出てこなくてちょっと悲しい。

Feb.21,2014 (Fri)

■艦これ。重巡装備その2 [GAME]

+ 3-x/4-x/5-xでのデイリー・ウィークリー消化に鳥海改と衣笠改二を気兼ねなく入れられるのはありがたい。そういう意味でケッコンカッコカリは満足。

+ で、ここで書いた20.3cm(3号)連装砲+15.5cm三連装副砲×3の鳥海・衣笠を使う頻度が増えてこの装備の評価が上がった。もう重巡は全部この装備にしたいくらいが、20.3cm(3号)連装砲は4本しかない。衣笠改二と三隈改しかもってこないこの砲を量産するのは厳しい……。

+ 高レベル空母戦艦の経験値稼ぎも兼ねて5-2に通って三隈牧場するか、まるゆ・三隈狙いで、最低値で大型艦建造を定期的に回すかだろうか。
