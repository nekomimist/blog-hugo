May.1,2014 (Thu)

■艦これ。春イベントその4 [GAME]

+ E-5。4/29に11回出撃してゲージを削りきって、本日1回出撃で終了。

+ 下記の戦3空3の構成で12回出撃。12回ボス到達、7回ボス削り、5回ボス破壊。

  • 金剛改二 Lv107→Lv110 - 46cm三連装砲×2/零式水上偵察機/三式弾
  • 大和改 Lv93→Lv94 - 46cm三連装砲×2/零式水上観測機/三式弾
  • 武蔵改 Lv95 - 46cm三連装砲×2/零式水上観測機/三式弾
  • 加賀改 Lv116→Lv117 - 零式艦戦62型(爆戦)/烈風/烈風改/ダメコン
  • 赤城改 Lv113→Lv114 - 零式艦戦62型(爆戦)/烈風/烈風/ダメコン
  • 瑞鶴改 Lv106→Lv109 - 流星改/烈風/烈風/ダメコン

+ 支援は決戦支援だけ。最初は駆逐×2+航戦×2+軽空母×2のちょいエコ支援。最後は駆逐×2+航戦×2+戦艦×2の脳筋支援。タコヤキとフラ戦1人くらいしか落としくれなかった。

+ E-5開始時、油95k弾90k鉄126kボーキ96kで、終了時は78k弾78k鉄102kボーキ93k。……まあ、4/30に5-4で戦果稼ぎ数十回やったので、E-5だけでここまで減ったわけではないが、大和型の修理代は痛いなぁ(白目

+ 谷風が出なかったのでまだちょっとイベント海域につきあう予定。

May.3,2014 (Sat)

■艦これ。春イベントその5 [GAME]

+ 谷風を掘りにいかなければいけないのだが……やる気が出ないね。

+ E-2は夜戦がイヤだし、E-3は資源消費が痛く、E-4はルート固定するとダメコンが痛い。E-5? 冗談でしょ。

+ まあ、E-4はルート固定せずとも行けなくはないようなので、駆逐×2+雷巡×2+航戦+空母で挑んでみたら、2回連続で外れでボス前まで行けずorz。まあもうちょっとやってみるか。

+ 蒼龍がLv98になった。これでLv98が、夕雲・綾波・日向・霧島・比叡・榛名・蒼龍の7人体制に。はたして俺は7人×700円を払うのか！←きっと払う。

May.4,2014 (Sun)

■艦これ。春イベントその6 [GAME]

+ 谷風なんていなかったんや……。まだあきらめないけど! 結局羅針盤荒ぶるルート固定なしE-4よりもルート固定のE-3のほうがいいってことで、金剛型×4+空母×2でE-3を回す。思えば2013年も初風と阿賀野を求めてE-3を50回回したっけなぁ。

+ 明石さんがLv34にになって改造間近である。改造したら艦艇修理施設が1コ増えてさらにカスダメ修理が捗る。いいことだ。最近放置は明石さん旗艦ばかりだよ。

May.5,2014 (Mon)

■艦これ。春イベントその7 [GAME]

+ 戦果稼ぎに5-4を回していると衣笠さんが死ぬほどドロップするなぁ……。

+ 明石さんがLv35になったので改造した。改造前「提督も修理ですか？どこが壊れてます？」→改造後「提督も、少し修理したほうが良いみたいですね」。壊れている箇所が見てわかるほどに、提督の破損が進行したらしい(ぉ

+ E-3回しトータル4回目で谷風がドロップ。攻略時と同じ金剛型×4+空母×2で行ってたから(*1)、周回の資源消費が痛かったけど、わりとあっさり出てくれて助かった。

+ これでうちにいない娘はビスマルクだけに戻ったので、イベント海域からは撤収かな。

*1: 重巡×2+金剛型×2+空母×2で行ったらD敗北して泣いたので……。

May.10,2014 (Sat)

■艦これ。 [GAME]

+ 夕雲さん&綾波さんとケッコンカッコカリしたので、次は放置ボイスがかわいい敷波と、3-2突破時の主力だった磯波さんを育てるぜ。遠征レベルトップ2でもある二人だけど、まだLv60&Lv58なので育て甲斐あるなー。

+ 谷風さんの補給ボイス「かぁ! これで勝つる!」がクセになる。母港ボイスの1つ「なーにー? よしよしされたいの?」もいい。よしよしされたいぞ!(ぉ

+ 酒匂は「ねぇ、あたしも連れてってよ〜」がグっとくる。竣工時に阿賀野と能代は戦没していて、そして矢矧が沈んだ坊ノ岬沖海戦にも参加できなかった娘だもんね……。とりあえず改にして、姉たちに追いつかせねばね。

May.23,2014 (Fri)

■艦これ。[GAME]

+ 羽黒と龍驤が改二とのこと。鳥海さんの改二まだーっ。

+ うちの羽黒はLv57だったので未だ改二になれず。演習でチラ見する限りかなりかわいい系だ。重巡は鳥海Lv130・衣笠Lv126・摩耶Lv120・愛宕Lv97・高雄Lv97と層が厚いので、主力には入れるかどうかは悩ましいところではあるが、とりあえず改二にはしよう。

+ 一方、龍驤はLv79で改造可能レベルを超えてたのでサクっと改二にした。俺は改の変更グラフィックはそんなに嫌いじゃなかったのだが、改二は文句なくかわいいなー。

+ そして2-5に突入。重巡×3+航戦+空母×2や重巡×2+高速戦艦+航巡+空母×2で行ったが4戦コースばかりだった。3戦目でフラヲ改に加賀さんが一発大破させられて途中撤退したのが1回だけあったけど無事ゲージ破壊できた。

+ で、勲章1コ取得したら改装設計図が2つになったので、1つ使って利根を改二に改造した。……ずいぶん幼い感じになったな利根姉さん!
