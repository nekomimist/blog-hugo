Sep.5,2012 (Wed)

■「いろとりどりのヒカリ」 (Favorite) [GAME]

+ 軽いキモチで始めると後悔するねぇ、このゲーム。

+ 真紅以外の娘たちのセカイに残った課題をきっちり片づけつつ、「いろとりどりのセカイ」が終わってなお真紅の周辺に残ったお話上の"ひずみ"を回収しようとしている力作でありました。……そう、力作では、あるん、だけど。ね。

+ なんでここまで真紅はひどい目に合わなきゃいけなんだろう。そもそも真紅は本編の「いろとりどりのセカイ」でも幸と不幸の収支が赤字だったのに、FDであるはずの本作でさらに赤字を積み増すとはどういうことだ。赤字を積むのはうちの会社だけで沢山だぞ(ぉ。

+ 「いろとりどりのセカイ」と「いろとりどりのヒカリ」で前編後編と考えればこういう話になるのは必然なんだろうけど、真紅がもっともっと幸せになる話を期待していただけにちょっと肩すかしというか、たいへん気が重くなる展開なゆえに二度とリプレイしなさそうな、そんなゲームでありました。

+ よくデキてんだけどねー。うん。

Sep.17,2012 (Mon)

■「中の人などいない!」(ALcot) [GAME]

+ 先日終了。俺としてはわりと早めに終わった感じ?

+ キャラ的にはあやのんとルイさんが甲乙つけ難い感じだった。あやのんは、黒髪ですらっと背が高いくせに、おどおどかわいい娘で良し。CVの杏子御津も良し。ルイさんは金髪でちんまい年上ってだけで弱い上に、きりっとしている所もよいし甘えてくる所もよい。CVも車の人なので安心(何が)。

+ 優香(CV:桐谷華)もよいキャラだったけど一歩譲る感じがあった。お話がBO団の中に閉じすぎるのもちょっとマイナス。佐緒理は他の娘のシナリオの方が輝いていたように思う。「ゆるふわボディ」とか中の人のそよそよネタはなかなか面白いが :-)

+ 総じて「質感」は低くないのだが……一点気になる所がある。そろそろ「暗闇に光」系の心象を示す一枚絵は描き直したほうがいいし、システム系の効果音は変えてもよいように思う。いつから使い回してんねんって感じがしてしまう。

+ まあしかし、安定した感のあるよいゲームでした。「幼なじみは大統領」から続く一連のムチャな世界観の中で、それなりにシリアスとギャグのバランスをうまくとって楽しくしようとしている感じ。俺は嫌いじゃないよ。8点。

■「1/2 summer」(ALcotハニカム) [GAME]

+ ついでに7月末ころに終わらせてたこれの感想も書いとこう。

+ せせなやう原画と、CGと、キャラの性格づけはよかった。各ヒロインの話は……汐とか叶とか俺にはイマイチだったけど、まあ、そう許せなくもない気もする。

+ が、全員終わったあとに入る最終シナリオがツラかった。誰のための話だったのやら。クロハのためならこの作りにはならないだろうし……\

+ えろシーンが多めなのだが、どのキャラも後半に行くにつれパワーダウンを感じたように思える。ぶっちゃけ各キャラ数を1,2回減らしてその分ねっちりやったほうがテーマ的にはよかったように思える。

+ 総合的には6点。光る所はあったように思うだが、ね。

Sep.25,2012 (Tue)

■「イモウトノカタチ」 (Sphere) [GAME]

+ あやか。ふりかえってみると話的にはいちばんツッコミ所が少なかっただろうか。兄貴の言動がアホで底が浅いのが気になるくらいか。それにしても、最後に許すあやかが天使すぎだよね。

+ 美優樹(美幸)。キャラとしてはが一番好きで、話も告白のところまではよかった。が、どうも終盤の行動の整合性が取れてなさ\がこの娘らしくないのが気になるし、話の最後がどうも恋物語としてはすっきりしない。アフターシナリオを見てもなおすっきりしない。

+ 千毬。「どうにもウザい」という感想しかない。星メモの千波とかタユタマのアメリあたりはかわいいと思えた俺でも耐えられないくらいウザい。なんだろうあれ。

+ 最後に真結希(真幸)&ミータ。ええと、ミータだけでいいです。こういうエンドがありならば、美優樹(美幸)シナリオももっと幸せいっぱいに終わってよいじゃないかと思えてしまうよね。

+ 総評。どうしてこうなった……という感じがするが、まあ「ヨスガノソラ」もヘンなゲームだったのである意味正常進化なんだろうか。評価は6点。

Sep.26,2012 (Wed)

■日記とブログのスマホ対応のこと。[COMP]

+ 技術ネタだからblog側に書くべきな気もするが、まぁ、こっちの日記ネタなのでこっちで。

+ INFOBAR A01でこのページを見たらイマイチだったので、ちょっといじってスマホ対応にしてみた。基本的にはmetaにview-portでデバイス画素幅に表示幅にするようにして、CSS media queryで、表示幅が狭いときは1カラム表示に変えるようにしただけ。

  • Windows7上のIE9/Firefox15.0.1/Chrome22
  • INFOBAR A01(Android2.3)の標準ブラウザでのportrait/landscape表示
  • Xpera Tablet S(Android4.0)のFirefox/Chromeのlortrait/landscape表示
  • iPhone3GS(iOS6)のportrait表示

+ という手持ち環境では大きな問題はなさそうだった。iPhone3GSのlandscapeはちょっと文字デカすぎなのが悩ましいけど、まぁ、読めるしいいかな。
