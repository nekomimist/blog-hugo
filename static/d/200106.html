Jun.15,2001 (Fri)

■「Railway〜ここにある夢〜」(EMU) [GAME]

+ 疲れてるのでインストールするだけで終わりそうなそんな予感。あしたも仕事だしな。

+ 最小で2MB、通常で100MB、フルで900MB…人としてフルインストール、ぽちっとな。

+ 主題歌は「夏草の線路」(歌:KOTOKO TO AKI)。…なんかタイトル聞いた瞬間、遊佐未森のソレが頭をかけめるぐるのですが、もちろん似ても似つかない歌。遊佐未森の「夏草の線路」(「HOPE」所収)ももう11年も前の歌なんで、知らん人も多かろう。遊佐未森だし(ぉ

+ 640x480なんつー低解像度だと、フォントは下手にアンチエリアスをかけたものより、ビットマップ内蔵のフォントを使いたいが、もちろんフォントは選べない。死ね。…まぁ、前作「るなシーズン」よりはマシなシステムそうな、そんな予感。

■「プライベートナース」(AngelSmile) [GAME]

+ まりあと彩乃が終わったので、シナリオ内容には特に触れないで、ただ、やってて思ったことを。

+ 俺が時間軸強制演出が嫌いだ。そしてこのゲームはそれが多い。おまけにマウスの右や左のボタンをカチカチしても、メッセージスキップでも飛ばせない。コロス。

+ セーブの粒度が荒くてイラつく。1日がけっこう長いこともあるのにセーブしてロードするとその日の朝に戻る。これで既読スキップが速かったら許せるのだが遅いのだ。滅殺。

+ 原画の根須さんの絵は好きなのでOK。しかし、なんだか今回は胸。…好きだけどな(ぉ 塗りもよいね。CGには文句はない感じ。

+ 音楽。あまり印象に残ってない…とゆーのも、時間軸強制イベントにイライラしたせいで、メッセージスキップをしかけて他のことをやっていたからだ。埋まれ＞俺(ぉ

+ 話は…中身には触れないが、なんだかアリキタリな感じを受けた(*1)。ま、やめられないとまらない感はない。そういう点で「Phantom Of Inferno」や「ねがぽじ」や「青い鳥」に遠く及ばない。比べるな? まぁそうか。

+ 総論としては佳作。最近本業でココロがささくれてたから癒やされることを微妙に期待したのに、時間軸強制イベントや彩乃ルート探索(*2)でささくれだってしまった(ぉ

+ …心に余裕がある時期にやったらもうちょいと違う感想を持つかもしれんなー。

+ ああ、ちなみにキャラはけっこう好きです。俺的には彩乃>まりあかな。でも実はイトメ(仮名)が一番好きかも(ぉぃ

*1: 本当にアリキタリかどうかはここでは論じない。ただ、そう感じた。
*2: 結果としてはそう難しくなかったのだが。
