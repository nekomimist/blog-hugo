May.7,2008 (Wed)

■「朝凪のアクアノーツ」(Fizz) [GAME]

+ 終わったばっかなので、これは感想を書こう。

+ 最初に深緒。なんか評判悪いようだけど俺は案外好きだ(*1)。まぁ、たしかに共通部の言動は理不尽にワガママなのだが、後半はきちんとかわいい。声はいつもの榊原ゆい。榊原ゆいはいかにもなヒロインをやるよりもヘンなヒロイン(*2)をやったほうが面白いと思う。最後のシーン、あの人にやらさんでも、深緒と亜樹の二人でやってもよかったかもね。

+ 次に美幸。キャラ的にどーも好きになれんかった。わかったような先輩の言い草がムカつくのも一因。お話的にも深緒がかわいそうなのがなー。声はいつもの安玖深音。もうちょっと違う感じの声を出してもよいと思うのだが……。

+ かなか。彼女の境遇とセイレンのからみ具合だとか、お話的には一番まとまっていると思う。んが、木村あやかの声の出し方がどーも慣れんかった。こういうしゃべりをさせるなら木村あやかでないほうがよかったような気もするが、朝薫と夕薫の木村あやかは大変イイ感じなので、これはこれでよいのかな。

+ 友里子。主人公の双子の姉。異性一卵性双生児ならばかなりレアだね。お話としては、えろゲの実姉ものとしては一段ひねっているのだが、コミック等ではそれなりに見る展開、という感じ。でも悪かない。声はいつもの青山ゆかりだが、これはこれでよし(ひいきだなぁ)。

+ 最後にメリーとヒマ母さんのエンドを回収して、おまけの4つを全部見て終わり。おまけの1/2が近親ネタってのはどーよ?(笑)

+ 総評。前作「ましろぼたん」よりは人に勧めやすいデキだったように思う。ヒロインのイチオシとしてはユウ、次点はかなかだが、やはり特筆すべきはメリーのかわいさだろう。愛玩動物チックな言動と金松由花の甘い声の組み合わせが絶品。ヒマ母さんもよいのだが、友里子シナリオでちょっとアレな感じ(仕方ないんだけど)なのがマイナスかな。

+ システムの機能としてはまぁ普通。一部フォントの演出があるけどちょっと浮いてたかも。うちの環境ではたまに画面描画を一切しなくなったりしていて、システム的に不安定かもしれない。

+ ま、思ってたより楽しかった。評価は7点で佳作。

■その他、わりと最近に終わらせたゲーム [GAME]

+ 「かみぱに!」(クロシェット)。個人的な感想としては、天音かわいいよ天音、って感じだったな。ラストの古依シナリオが、直前の恵シナリオ――別にすばらしくはないが、終盤まではまぁ悪くない――に食われてしまってイマイチ盛り上がらなかった感じなのが残念だ。もっと古依の絶望感を出す何かがあればよかったのかねー。とはいえそんなに悪くはなかったかな。

+ 「Princess Frontier」(AXL)。別に悪かないのだが、印象に残ったのがヒロイン陣ではなく、野郎共のジンやアロンゾだったりしするのはいかがなものか :-) 前作の「恋する乙女と守護の楯」の方が俺的には好きだったかな。

+ 「こなゆき ふるり」(ブルームハンドル)。いつもの北側ノリなのだが……誉めづらいゲームだなぁ。メインの3人のヒロインがどーも好きになれずに、サブ風味の冴子と伊織の方が好きな時点で負けているのだろう、俺は。同じ北側シナリオなら前述の「Princess Frontier」の方がよかったように思う。

+ 「とらい☆すたーず」(CLOCKUP)。絵も声も悪かないのだが、敵(七影刃)に存在感のあるヤツらが少なく、かつ見えない所で倒されるヤツもいたりして、大変尻すぼみ感が強い。こういうゲームなら、もうちょっと燃え燃えな展開にしないといかんとちゃうかな? まぁ、ゼロがかわいいので許そう(ぉぃ

*1: 個人的には涙目になって地面でぴちぴちしているカットイン絵がすげー好き(ぉぃ
*2: 「六ツ星きらり」のほくととか「よつのは」ののんさんとか「はっぴぃ☆マーガレット!」の花梨 とか。
