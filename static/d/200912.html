Dec.18,2009 (Fri)

■きょうの自転車ネタ [BIKE]

Dec.20,2009 (Sun)

■自転車メンテ [BIKE]

+ ぶっちぎれたシフトケーブルを交換した。やってみればわかるが、ケーブルがどこをどう通ってどう固定されているかさえ理解できていれば何も難しい作業はない。一番苦労したのが余ったケーブルの切断だったというオチ。遠くない将来にブレーキケーブルも交換すると思うので、マトモなワイヤーカッターが必要だなー。

+ セットのを買うと楽かなと思ってシマノのXTRブレーキケーブルセットを買ったのだが、よく考えるとアビオスはアウターケーブルの長さがふつーと違うのだな。つーことでインナーはそのまま使ったけど、アウターは一部しか使えなかったので元のままの部分が多くなった。切れてないアウターを買っとこう。

+ あとフレームをバイクウォッシュで全面的に拭いて、スプロケットをディグリーザーできれいにして、先日交換したチェーンから2リンクだけ抜いて(*1)メンテ終了。

+ しかし、こうやって自分でメンテしたりパーツ交換したりしていると愛着が湧くね。高校〜大学時代に乗ってたレイダックもトラベゾーンももっときっちりメンテしていればよかったなーと思うことしきり。アビオスツアラー買って3年弱。あと何年無事でいてくれるかわからないが、なるべく長生きでありますように。

*1: 交換前と同じ長さにしていたのだが、ちょっと長い感じがしてたのだな。

Dec.21,2009 (Mon)

■寒い中自転車で走るということ [BIKE]

+ 先日買ったKG-8Wを気温0℃〜3度くらいで試したが、ちょっと冷たい感じがする。まぁ、しばらく乗ってると体から熱が出て手もそんなに冷たくならなくなるから問題なしかな。ちなみにLを買ったらちょっと小さめだった。夏頃だったら手が入らなかったかもしれない。

+ これで手は大丈夫になったが、足先がどうにもならなくなるほど冷えるので、次は足の防寒が必要だな。上着もベンチレーションがしっかりした自転車用のが欲しい。それ以外は安物のサイクルジャージとパールイズミのバイカーズパンツで特に問題ない。

+ まぁしかし、気温0℃近辺だと、走ってるうちはいいけどパンクして修理とかしてたら死ねるね、きっと。

■体重を減らすこと [MISC]

+ この週末で体重が83kgを切った。夏期休暇前から本格的に体重減少に努めて4ヶ月半で18kg減。元が脂肪だらけだったので減りやすいやね。まぁ、標準体重まではあと20kg減らさないのいけないのだが(笑)

+ ま、体重計を信じるなら俺の体脂肪率はそんなに高くないし筋量も多めなので、BMIの標準には拘らずにあと10kgくらい落として維持を目指したい所。

+ で、まぁ、別にムチャはしていない。

  • 1日1時間程度の運動をする。夏〜初秋は自転車で20km〜30km走ってた。寒くなってからは踏み台昇降メイン。
  • 朝はちゃんと食う。楽するためにシリアルとヨーグルトがほとんど。
  • 昼はおにぎり2個+菓子パン1つなノリ。栄養的にどうよ?とは思うが。
  • 夜は米と肉を減らして野菜類を増やす。しかし全体としては以前の70〜80%くらいの量。
  • 飲み物はコーヒーとウーロン茶、甘いものはペプシNEXあたりのノンカロリー飲料で。
  • 間食はたまにならしてもよいが、ドカ食い厳禁。
  • 腹が減ったらノンカロリーのゼリーを食う。もっとも最近はぜんぜん食わなくても平気だが。

+ 食うことについては特に無理していないので、今後も続けられそうだ。運動については踏み台昇降に飽きてきたのでちょっと悩ましい。雨や雪の中、自転車で外を走るのは勘弁願いたいからな。スピンバイクでも買おうか……。

■帰省準備でノートPCの整備 [COMP]

+ 帰省時にしかまともに使わないノートPCのRIPPLE(DELL XPS M1330)だが、せっかくの機会なので、Windows7RC Ultimate x64からWindows7 Professional x64に入れかえた。

+ 素でWindows7 Professionalをインストールして、GeForce 8400MのドライバをnVIDIAから取得、タッチパネルドライバとカードリーダドライバはStudio XPS 13向けWindows7 x64対応ドライバを取得すればOKっぽい。

+ その他入れたソフトはFIENA(自作PC1号)準拠だが、キーカスタマイズはyamyじゃなくのどかにしてみた。ついでに未プレイのエロゲを何本か(*1)入れたが特に問題ない様子。

+ XPS M1330もCore2Duo T8100でGeForce8400Mとちょっと古めではあるのだが、SSD(X25-M 80GB)でメモリ4GBなのでなかなか快適だ。まぁ、負荷かけるとファンが五月蝿いのとGeForce8400Mが持病不良のはずなので、来年末あたりにノートPC入れかえを考えますかな。

*1: ちなみに「しろくまベルスターズ」と「幻月のパンドオラ」と「eden*」。
