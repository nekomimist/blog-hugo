Mar.1,2015 (Sun)

■艦これ。 [GAME]

+ イクとイムヤがLv99に到達したので51人目・52人目のケッコンカッコカリ。はっちゃんもあとわずか。

+ そして天城さんが改に。偽装の着物を脱ぎすてて雲龍と似た感じの大胆な服装に。かなり好みです。表情が生き生きとしているのは、空母としてちゃんと役に立てるからだろうかね。

+ おや、摩耶改二か。うちの摩耶さまはLv137で、重巡No.3にして全艦隊No.3なのでもちろん改装に問題なし。起工(1928年3月26日)・進水(1931年4月5日)がちょうど今の時期になる鳥海さんの改二も期待できるかなぁ。

+ 今月の第五戦隊任務は1回、水上反撃艦隊任務は2回で終わった。いつもすんなり終わる前者はともかく、敵艦隊の引きが悪いと1戦目から大破する後者が2回で終わったのは大変ありがたい。今月もいい月かもしれん(ぉぃ

Mar.5,2015 (Thu)

■艦これ。 [GAME]

+ はっちゃんがLv99になったので53人目のケッコンカッコカリ。潜水艦の残りはまるゆLv76と呂500になったばかりのろーちゃんLv55。ちょっと遠そうである。

+ これでLv98は夕立・時雨・島風・利根・筑摩・羽黒・あきつ丸の7人。この中だと趣味枠のあきつ丸と駆逐主力最古参の一人島風を優先したいかな。趣味枠という意味では明石さんLv93もLv99にしたいのだが……遠い。

Mar.10,2015 (Tue)

■艦これ。 [GAME]

+ 摩耶さまはLv138になった。改修餌も充分に溜めてるし万全である(*1)。

+ 菱餅は11コになった。今週の北方任務消化でもうちょっと増えるかも……。

*1: レベルが高いから、改二にした時点でかなりカンストに近くなるとは思うのだがね。

Mar.13,2015 (Fri)

■艦これ。摩耶改二 [GAME]

+ もちろん何の問題もなく改造できた。近代化改装なしで全パラメータMAXになった。Lv122の蒼龍改二の時に続いて2回目かな? 改装餌が無駄になってしまった。鳥海さんはさらに高いLv147だしこちらも不要の可能性が高いかな。

+ それにしても、ちょっとヤバいくらい好みですぜ摩耶さま。帽子や色に高雄愛宕の要素を入ってきたのと、艤装がリッチにいなってるのが特徴的なのかな。こりゃ鳥海さん改二も期待が溢れますぜ……。

■ingress [GAME]

+ LEVEL7になった。先は長いが、まあ、それなり。

Mar.18,2015 (Wed)

■艦これ。[GAME]

+ 先日利根と筑摩さんがLv99になったので、54人目55人目の以下略。

+ 摩耶改二。航空優勢以上取れてるなら今までと同じ装備で問題ないけど、制空権がない状況なら、3号砲+15.5cm三連装副砲+12.7cm連装高角砲+機銃集中配置で、対空カットインと昼火力96と夜連撃を得るのもよいかな。この装備での対空カットインは摩耶さましかできないし。

+ 今月も500位に入れたので、九七式艦攻(九三一空)と零式水中聴音機をもらった。前者は2つ目だが、天山(九三一空)も1つ持ってるし、あまり使い道はないのだよなー。後者は航戦に積む方向で。

Mar.21,2015 (Sat)

■艦これ。 [GAME]

+ 3/27メンテで鳥海さん改二の確定予告が出た。ああ、去年秋から待ちつづけた改二がついに……。

+ やっとレーベがzweiになった。そんなわけでZ1 zweiを旗艦にして23回目のビスマルク狙いの大型建造を敢行。結果は扶桑。大鳳は10回、大和は6回で出たのにビス子さんはなかなか出てくれませんなあ。

Mar.22,2015 (Sun)

■艦これ。 [GAME]

+ 鳥海さん改二前記念ってことでLv148にしておいた。カンストまであと経験値38万。遠いような近いような。

Mar.27,2015 (Fri)

■艦これ。鳥海さん改二 [GAME]

+ さて、鳥海さん改二だけど、Lv148かつ勲章23個持ちだったのですぐに改装。

+ Lv148時点で火力+7装甲+3雷装+17回避+1対空+1搭載+4索敵+15運+9かな。

+ 重巡昼火力最強で夜戦火力最強はすごい。索敵の向上もありがたい。対空はお察しだけど、まあ、鳥海さんだし! 回避もあまり上がってないけど、Lv148時点で99なので問題なし。

+ そして運+9が嬉しい。ケッコンカッコカリとまるゆで21上がってたのでこれで運4。イベント鉄火場海域においてカットイン運用ができなくもない運になった。

+ ……改二の姿にまだ慣れないけど、すぐに慣れるに違いない。うん。

■艦これ。 [GAME]

Mar.29,2015 (Sun)

■Ingress [GAME]

+ LEVEL8になった。やっとnovice脱却ってところかね。

■艦これ。 [GAME]

+ 5-2ウイークリー処理中、ひさびさに三隈さんがドロップ。通常海域でのドロップは2回目、前回は2013年12月。イベント海域ではそこそこドロップするけど、たいへんレアな娘だよね実際。初風よりはマシだろうけど。

+ そして、残ってたマンスリー水上打撃部隊5-1任務を片づけた。いつも通り、大和武蔵長門陸奥のうち2人+伊勢or日向に瑞雲12型ガン積み+利根筑摩のどちらかに瑞雲12型を2つ+大淀さんは三式ソナー3三式爆雷1で突入して3回で終了。……しかし、これを制空一切なしの三川艦隊でやるのか……悪夢だなあ。

+ 明石さんがLv96になった。明石さんから指輪買って明石さんに渡すのってちょっと面白いし、ケッコンカッコカリを目指そう。

Mar.31,2015 (Tue)

■艦これ。 [GAME]
