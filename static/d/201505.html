+ 初戦は3回目でやっと非雷巡チ級flagship構成を引けて突破。この任務、初戦が最大の山なんだよね。2戦目の夜戦は水雷戦隊にとってはわりと余裕だし(*1)。

+ ボス戦。夜戦突入時点でこちら中破2(Prinz Eugen/天津風)で敵は上4人が残る悪い展開。おかげで最後は中破島風と中破天津風が、中破エリル・大破エリルを確殺しなければいけない状況になって超おっかなかった。まあ、二人ともちゃんと仕事をしてくれて今月も終了。

■艦これ。春イベE-6 Roma掘り [GAME]

+ 攻略構成でボスまで行って数回掘ってみたけど、なぜだか攻略時より事故って資源とバケツと精神に痛かったので、まるゆデコイの軽量艦隊でFHKで2戦して撤退する作戦に変更。

+ 第一艦隊。艦載機以外の装備は適当。最上型4人はちょいダメを食らったら旗艦に移動して守ることでバケツ消費を減らしてみた。まるゆは3人でローテーション。隼鷹はダメージをちょっと食らうごとに装備ごと千歳Lv121→千代田Lv121とローテーションしていった。

  • 三隈改 Lv136　20.3cm(3号)連装砲★4,瑞雲12型,艦隊司令部施設,20.3cm(3号)連装砲★2
  • 最上改 Lv125　20.3cm(3号)連装砲,試製晴嵐,熟練艦載機整備員,20.3cm連装砲
  • 鈴谷改 Lv126　20.3cm連装砲,試製晴嵐,熟練艦載機整備員,20.3cm連装砲
  • 熊野改 Lv124　20.3cm(3号)連装砲,瑞雲12型,瑞雲12型,20.3cm連装砲
  • まるゆ Lv17
  • 隼鷹改二 Lv122　烈風改,天山一二型(友永隊),九九式艦爆(江草隊),彩雲

+ 第二艦隊。潜水艦構成にはせず雷巡×3にした。駆逐艦2番艦はまるゆ退避で超絶疲労するので、叢雲Lv72・初霜Lv79・吹雪Lv81・初春Lv84の4人でローテーション。軽巡は小ダメを食らったら入渠するルールで、神通Lv96・矢矧Lv94・大淀Lv95で適当に入れかえ。

+ Kマスで開幕以外の攻撃をすることはほとんどなかったので、雷巡以外の装備は超適当で問題なし。雷巡も魚雷魚雷甲標的のほうが安定したかもしれない。

  • 神通改二 Lv96　20.3cm(3号)連装砲★6,20.3cm(3号)連装砲★7,九八式水上偵察機(夜偵)
  • 叢雲改二 Lv72　12.7cm連装高角砲(後期型),10cm連装高角砲,WG42
  • 初春改二 Lv84　12.7cm連装砲★6,12.7cm連装高角砲(後期型),新型高温高圧缶
  • 木曾改二 Lv122　15.5cm三連装副砲,15.5cm三連装副砲,甲標的
  • 大井改二 Lv122　20.3cm(3号)連装砲★7,20.3cm(2号)連装砲★4,甲標的
  • 北上改二 Lv123　20.3cm(3号)連装砲★6,20.3cm(3号)連装砲,甲標的

+ 上記構成で16回突入。1回だけ泊地水鬼の開幕爆撃のせいで大破艦が2人になったので撤退。他の15回は全部KでS勝利だった。ちなみにまるゆ大破での司令部退避は9回。

+ 15回のドロップは陸奥×1、隼鷹×3、飛鷹×1、愛宕×1、足柄×1、熊野×1、皐月×1、Roma×1。そしてドロップなしが5回。皐月ドロップはなにもなしより気分的にキツかったw

+ 16回出撃での資源消費は油2.5k/弾1.7k/鉄0.8k/ボーキ0.6k/バケツ10(遠征込み)。ボス掘りは6回でバケツ24個だったので比較にならない。まあ、U-511・Roma双方狙うならボス行くしかないのだが。

*1: たまには大破するけどねぇ

May.7,2015 (Thu)

■艦これ。春イベ感想 [GAME]

+ E-5で朝雲掘ろうと思ったらバケツ消費がキツすぎて笑った。1-6で出ることを祈って撤収しましょう……。

+ そんなわけで、イベントも一段落かな。あと欲しいのはまるゆくらいだし、だったら大型建造最低値をやったほうがたぶん楽だしね。

+ 難易度的は全て甲で通しても2014夏E-6のような凶悪さはなく、それなりに歯応えはある悪くないバランスだったかな。能動分岐も少なくともE-6では有効に働いていたように思う。

+ 新艦娘の秋津島。「かも!」一号。かわいいけど今のところ無力だなあ。ま、演習で育てましょう。

+ 高波。「かも!」二号。ルンガ沖夜戦で需要な役割を果たし壮絶な最期を遂げた高波が、こういう気弱なキャラになるのは面白い。髪色的に夕雲姉さん似でちょっと好み。

+ イタリア(リットリオ)。なんて地中海的な! 「かわいいお姉さん」風味な感じで実によいねえ。

+ ローマ。ツンツン風味な戦艦ってそういえばいなかったな。強気のようで裏に弱い所を感じさせられて、そこがかわいい。

May.17,2015 (Sun)

■艦これ。春イベ終了 [GAME]

+ 鳥海さんがLv149に到達。あまりカンストさせたくはないのだが、使わないのも最愛の嫁に悪いので、遠くないうちにカンストするであろう。
