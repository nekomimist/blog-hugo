Jun.3,2014 (Tue)

■艦これ。[GAME]

+ 愛宕さんがLv99到達したので19人目のケッコンカッコカリ。重巡4人目である。あとは高雄さんLv98をLv99にすれば高雄型4姉妹全制覇だ。残り経験値12万くらいだったかなー。

+ 1-5と2-5をのんびりやることにして後回しにしているのだが、そうしたらさすがにランキングが2000位以下になった。ちょっと新鮮。

Jun.6,2014 (Fri)

■艦これ。 [GAME]

+ 予想通り飛龍さんが改二。蒼龍さんにも夏に改二が来るようで大歓喜である。しかし、飛龍さんの所に友永隊が来たということは、蒼龍さんの所には江草隊が来るのだろうか。

+ 飛龍改二Lv111+ヴェールヌイLv92+夕立改二Lv90+長門改Lv96+龍驤改二Lv80+蒼龍改Lv114の構成で5-2に行ったら一発でS勝利できたうえに大鯨がドロップしてちょっと笑った。大鯨はもう一人欲しいね。

+ 1-5と2-5のゲージを削りきってしまったので、ランキング500位以内に入ってしまう気がする。いやもう今月はランカー狙いはしない、しないんだ……。

Jun.8,2014 (Sun)

■艦これ。 [GAME]

+ 高雄さんがLv99になったので20人目の以下略。

+ 高雄型4人・金剛型4人・綾波ともケッコンカッコカリ済みなので、あとは敷波とすればCV東山奈央全制覇だが……我ながら何をやっているのだろうなー。ちなみに敷波はLv71。磯波Lv71と共に育成中だが、まだまだ遠い。

Jun.10,2014 (Tue)

■艦これ。 [GAME]

+ 大鯨は初日に出たが、浦風がなかなか出ない。4-2で出るとのことなので、東方とろ号消化と、磯波&敷波レベル上げのための4-2巡回密度を上げたが出ないものは出ない。うーむ。

Jun.18,2014 (Wed)

■艦これ。[GAME]

+ 川内はLv60になった。川内の改二Lvが神通さんの改二Lvを超えることはなかろうからこれで大丈夫じゃないだろうか。改装設計図がいると泣くけど。

+ 綾波さんはLv111なので改二に向けては何もしなくてよし。川内と違って、綾波さん改二にもし改装設計図が必要なら躊躇なく使用する所存。

Jun.20,2014 (Fri)

■艦これ。 [GAME]

+ 川内がLv60、綾波がLv111だったのでサクっとどちらも改二に。

+ 川内さん。なんというニンジャスタイル。とりあえず川内さんと那珂ちゃんはキラキラ遠征で経験値稼ぐか。弾薬コストが悪いと言っても阿武隈・鬼怒と同じだしな :-)

+ 綾波さん。ちょっと大人っぽくなったが、追加ボイス含めて隙なく「綾波」な感じなのがよい。ケッコン後母港ボイスと放置ボイスが実に実に愛しい。これで普通に強くなったので、イベントなどの攻略で出しやすくはなったかなー。

Jun.22,2014 (Sun)

■艦これ。7月の改装艦 [GAME]

+ ふむ。榛名は確定として、重巡の候補は妙高・高雄かねぇ。最近のbob艦更新ラッシュを見てると妙高さんだろうか。うちの妙高さんはLv82で高雄さんはLv109。どっちが来ても何も準備する必要はなかろう。

■艦これ、放置していた任務をちょっと消化する。 [GAME]

+ 第30駆逐隊(二次)の1-5ボス到達任務。睦月Lv51・望月Lv48・弥生Lv51・卯月Lv37でトライ。大破撤退事故は1回だけだったが、ボス前で5回連続で逸れて泣きたかった。6回目に到達してA勝利して終了。航戦いないとS勝利は厳しいねー。

+ 次に第5戦隊で2-5ボス任務。妙高Lv82・那智Lv58・羽黒Lv72に、残りの妙高型である足柄Lv58と、ドラム缶を1つづつ積んだ利根改二Lv86・筑摩改二Lv85を入れて、bob艦艦隊で突入。一発S勝利で終了。やっぱ2-5は北回りだな!

Jun.24,2014 (Tue)

■艦これ。[GAME]

+ 綾波改二がかわいい。ほんわかほっぺは失われたが、あの細っこい胴回りとケッコン後母港台詞の最初のちょっと拗ねたような感じがあまりにも愛しい。

+ 鳥海さんがLv134になった。Lv150への道の約1/2行ってないと思うと先が長い。それにしても、鳥海さんは好きすぎて俺の那珂で別格。改二がすぐ来なくてもまあいいけど、補給ボイスと放置ボイスは欲しいなぁ。

Jun.27,2014 (Fri)
