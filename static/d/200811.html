Nov.1,2008 (Sat)

■寝込みモード [MISC]

+ 金曜の終業時刻前から悪かった体調がさらに悪化したので、今日は一日熱でフラフラしてた。トシ食ったせいかたかだか38℃超え程度の発熱でもツラいぜー。

Nov.3,2008 (Mon)

■きょうのよしなしごと [MISC][AUDIO]

+ 体調不良モードから抜けきらずに三連休が終わってしまったー。結局積んでいた「てとてトライオン!」(PULLTOP)を終わらせただけだったぜ。てまーっ。

+ ……まぁ、残業時間削減が厳しくなったせいで、平日の労働時間はかなりマシになっているので、前ほど連休の重要度は高くないのだが、それでもやっぱり勿体ないよね。

+ そして体調不良の勢いでリアSPを発注してしまった俺。フロントと同じS-PM300。最近フロント2chのS-PM300の左右の音色の違いが最近気になりだして、リア用にもう2本買って、一番音色が合ってる2本をフロントにすりゃいいかなーと考えてしまったのが運のつきなんだぜー。

Nov.10,2008 (Mon)

■きょうのよしなしごと [MISC]

+ リビングに敷いているコットンラグが痛んできたので、せっかくだからマトモなカーペットを買うかなーとか思って、サンプルなど取りよせつつ発注。……週末に届くらしい。早いなー。リアスピーカーが今週届くらしいし、週末はまたリビングのモノの再配置だなぁ。

Nov.15,2008 (Sat)

■わが家のオーディオ事情 [AUDIO]

+ リアSPが届いたので4.1ch環境にした。FIENAのサウンドカードもXonar D2Xに変更したのでPCからDTSでマルチチャネル出力もでできちゃうんだぜー(*1)。さーて、センターはどーすっかなー。

+ 手持ちのiPod with video(60GB)のバッテリーがすっかりへたってしまった。ま、買ったの3年前だしなー。120GBのiPod classicを買うのも捨て難いけれど、最近は散財しすぎだし、しばらくは初代iPod touch(16GB)でがんばってみますかねー。

*1: まぁ、ココに書いた理由で使い所が難しいのだが。

Nov.21,2008 (Fri)

■きょうのよしなしごと [COMP][GAME]

+ Core i7 965 Extremeがちょびっと欲しくなったが、さすがにCore2 Extreme QX9650@3.6GHzから乗りかえる価値はないよね。やはり冷静にWestmere以降まで待つべきやな。

+ SSDも欲しいのだが、現状のVelociRaptorをシーケンシャル・ランダムR/Wの速度の双方で上回るSSDはまだ出てないので(*1)、これも来年以降まで待つべきだろうなぁ。

+ そんなわけで満たせぬ物欲を抱えつつヤフオクを眺めていたら、「ワンコとリリー」初回版同梱のサントラ「Especially for you」が手が出せる価格で出品されていたので、つい落札してしまった。ま、これはこれでよいさー。

*1: ランダムR/Wに限ればIntel X25-EやX25-Mの足許にも及ばんがなー :-p

Nov.23,2008 (Sun)

■きょうのよしなしごと [COMP]

+ QX9650を4.0GHzで動かそうと思ったのだが、3.6GHzより上げようとするとなかなか難しい。3.6GHzは定格で行けるのだが、それ以上はコア電圧を上げなきゃいかん。すると負荷時にVcoreが電圧降下する。……つまり電源が弱いのだな。電源を800W-1KW級にしてまで最大1割程度の向上をゲットしてもしょうがないし、無理なく3.6GHzで運用することにしましょう。

Nov.28,2008 (Fri)

■きょうのよしなしごと [COMP]

+ 先日のVcoreが低下する件、Rampage Formulaは腐ってもR.O.G.シリーズなのだから、BIOS設定でなんとかなるよな?と思って、LLC(Loadline Calibration)を有効にしたら安定した。とりあえずこんな設定でTripcode Explorerで一日まわしても腐らない程度には安定動作している様子。ま、これでWestmere世代までがんばれるかな?
