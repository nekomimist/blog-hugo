■毎日のようにセキュリティホール? [COMP]

+ さすがにそこまでは出てないと思いますけど。出てるんですか? 実際の例を挙げてもらえるとありがたいですけど。

+ ServicePack2以降に出たHotFixはこれら。こちらも非常に参考になります。QChain.exeをうまく利用すると、複数のHotFixをあてたあとの再起動を1度で済ませることもできます。

+ でまぁ、問題は「既知の問題に対処(しない|できない)管理(者|体制)」だと思うんですがねー。例のRamen Wormにせよ、Code Redにしても以前から知られている(*1)問題が放置されたからこそ起こった事なわけで。

+ WindowsNT/2000が滅びたとしても、その後普及したOSの既知の問題が放置されててまた違う何かが起きるだけじゃないんかなぁ…。

■Windowsとセキュリティ [COMP]

+ デフォルトがinsecureだというのは問題ですよね、やはり。

+ IISに関して言えば、入れるときに「IISが何なのか知っているか?」「セキュリティ情報を追いかける覚悟があるか?」とか質問されて、その後簡単な試験をうけさせられて、さらにコレに自動的に登録されるくらいはあったほうがいいのかもしれませんねぇ。

+ 実家のPCがズタボロなのはうちも同じでもうどうにもなりません (^^; 相手が母なのでやる気もイマイチです(ぉぃ Win95OSR2だわ、IE3.02だわ、Internet Mailだわとヤバさテンコ盛りなんですが、何をやってもフリーズするので対処不能という…(苦笑)

+ ただ、Win9xよりはWin2kはずっとマシなので、WinXP Homeで大分ましにならないかなという淡い期待はありますね。User権限で使う限り、/WINNTや、/Program Filesの下は書きかえられないはずので、ふつーのUserがシステムを腐れた状況に落としこむ可能性はだいぶ減りそうですから。…ただ、Administrator権限で使われちゃいそうなんですよねぇ(笑)

*1: Code Redとその亜種が突く穴はコレのはず。6月18日にはHotFixがあったことになりますよね。

■MPIOの使い勝手 [AUDIO]

+ モノとしての作り。各所で言われてるようにサムスティックがもうひとつ。でも、画面がデカくていろいろ情報が見れるのはOK。仕上げ。さすがにNW-E5の足許にも及ばないな (^^; でも、安物っぽさがないのはいいかも。

+ メモリ。128MBのスマートメディアを足してあるので192MB。ふんふんと曲を放りこんでいくと47曲、3時間超入った。電池の保ちのよさとあいまって延々聴ける感じがよい(*1)。

+ 電池。7〜8時間は聴いたところで目盛が1つ落ちてちょいと経ったところ。電池は本当の保つねぇ。これならアルカリ電池でも気にならない。しかし、あんまり目盛が減らないもんだから、残量検知まわりが壊れてるのかと思った (^^;

+ 音質。ヘッドフォンにSONY MDR-E888SPを使っているのだけれど、電源入れただけでサーっとホワイトノイズが聴こえる。実際に音楽聴いててもちと冴えないかも。まぁ、誰もうちのリファレンスオーディオ環境(LINN GENKI, STAX SRM-T1S, STAX SR-007)の音質なんか求めてないし、これでおけおけ(笑)

+ 転送ソフト。今一使いでが悪いかなぁ。まぁ、使いこなしでどうこうなるレベルだけど。腐れOpenMG Jukeboxに比べると気楽だし :-p

+ 内蔵フォントはLCフォントに入れかえてみたり。S社の技術だLCフォント(笑)

+ 総合評価としては「よい買い物でした」という感じですな。

*1: NW-E5はATRAC3 105kbpsで約2時間弱。電池は4時間しか持たなかった。
