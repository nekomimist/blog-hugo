May.24,2009 (Sun)

■「星空のメモリア」(Favorite) [GAME]

+ 明日歩→衣鈴→こもも→こさめ→千波→夢→メアの順でクリア。

+ 明日歩。メインっぽいわりに損な立場だよなこの娘。佐本二厘は好きなのだが、明日歩みたいに元気キャラををやっているとちょっとやかましいと思ったりも……。この娘のやたら近寄るクセの理由にはっとさせられた。うまいもんだね。

+ 衣鈴。「傷つくから近寄らない娘」なわけだが、まあ、そういう娘がくっついてしまえばもう離れられないよねって感じの話だったな。後半、洋と千波が衣鈴に何も言わなすぎて、衣鈴がかわいそうだったりもー。館長が何者がちょっとわかる話。

+ こもも。「コガヨウ」なんて言って文句言ってたのが、中盤以降徐々に洋の存在が支えになっていくあたりの過程がよいと思う。シナリオ自体は「展望台の人身事故」の真相をころころミスリードさせる展開でなかなか上手かったように思うね。

+ こさめ。こももが終わったあとに開くシナリオなので、彼女がどういう人であるかはこももシナリオでよくわかっているわけで、ヒネりの少ないシナリオと言えようか。

+ 千波。やかましすぎとは思うけれど、もしこのゲームに千波がいなかったらそれはそれでちょっとつまらんとも思う :-) シナリオの中心は「親の世代に何があったか」であり、洋と千波の異父兄妹が結ばれることは中心じゃないように思う。まぁこの二人、くっついてからもあいかわらずだし :-)

+ 夢。存在が隠されている人なのでいろいろ触れづらいなー。まぁ、夢のシナリオというか、親世代と今の世代の天クルメンバーたちと、夢とメアとみんなのシナリオというか。

+ 最後にメア。キャラも声も一番かわいいと思うわけですよ。一方、シナリオとしては夢のオマケみたいな話なのでちょっと勿体なかったかも。「ウィズアニFUNTA!feat.RURU」の例のあることだし、別途補完せんかなーとか思ったりも。

+ 評価。Favoriteの4作目となる本作だが、やっと2作目の「WIZ ANNIVERSARY」を明確に上回った感があるね。発売直後にCROSSNETが残念な事になったようだが、Favoriteは生き残っているよなので次作も期待したい。

May.25,2009 (Mon)

■Bazaarのこと [COMP]

+ 2月頃からちょいちょいとBazaarを使ってみての感想など。

  • Windowsでもファイル名のダメ文字問題を食らわらない(*1)のはいいことだろう。ただ、bzr-svn、qbzr、TortoiseBzr等のプラグイン側の多言語対応はイマイチという話も聞くので注意は必要かも。
  • TortoiseBzrで作業が完結しないので、会社で皆に使わせるのはやっぱりハードルが高そうだ(*2)。
  • レポジトリをsftpだけで公開できるのは楽だ。
  • 地味にイヤな問題 : "bzr"は入力しづらい。

+ 総合的には、細かい文句はあるが悪くないといった感じ。現状小規模のプロジェクト(高々数ファイル程度)しか管理してないので、もうちょっと大規模なモノを管理してどうなるかを見てみたくはあるな。

■バックアップのこと [COMP]

+ KONA(FreeBSD 7.2Rマシン)はぜんぜんバックアップしてなかったことを今さら思い出した。RAID1だからHDD故障への対応はできるけど、定期でバックアップ取ってないのはマズい。とりあえず楽なところでUFS2 snapshotで行こうってことで下記の作業をほいほいと。

  • ports/sysutils/freebsd-snapshotを入れる
  • /etc/crontab と /etc/periodic.conf を変更 (snapshot呼び出し)
  • /etc/rc.conf を変更 (amd関連)

+ 基本的にfreebsd-snapshotインストール後に出てくるメッセージ通りにいじれば何事もなくOK。homeについてはweeklyも多めに残すようにしてみた。ユーザー操作による破壊はsnapshotで、ディスク破壊はRAID1でそれぞれそれなりに救えるだろうか。

+ ちなみにWindows PCはWindows Home Server(LULLI)のバックアップ配下にあるので、黙ってても1日1回バックアップが取られてゆくし、復旧時にはDISC1枚でサーバのデータから復旧できるのでKONAより安全だったりする。Windows Home Server自身の定期バックアップもとりたいけど、これができるソリューションはないような気がする。PowerPack3以降に期待したいところだが……。

■そろそろボーナスなわけですが [AUDIO][COMP][MISC]

+ オーディオ方面。センタースピーカーにS-A4SPT-PMを置こうかなーとか思ったが、5.1chでどれだけ使うかを考えると……要らんな。それよりもSTAX SR-007が夏場暑苦しいのをなんとかしたい。休眠中のSR-003を復活させるために延長ケーブルを買うのが一番安上がりかなー。

+ PC方面。FIENAの冷却系だが、先月末にRESERATOR XTを完全洗浄してから絶好調なので、特に問題なし。買うとしたらSSDか。DELL XPS M1330(RIPPLE)用と合わせてSSDを2台、あるいはFIENAはRAID0にして合計3台あたりで。

+ あと、GW中に欲しまり度がアップしたマッサージチェアのEP-MS40はぜひ買いたい。リビングに置いても邪魔じゃないってのはポイント高い(*3)。

+ ……軽く合計してもムチャクチャな額にはならないので、全部買ってもよいかもな。

*1: 一律utf8扱いだったかな?
*2: コマンドラインってだけでイヤがる人が多いからなぁ……。
*3: 置き場所が別にあるなら他の選択をしたほうがいいと思うけどな。
