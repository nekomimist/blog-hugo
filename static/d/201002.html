Feb.1,2010 (Mon)

■自転車のこと [BIKE]

+ ドロップハンドルが好きなので、Abios Tourerのドロップ交換という手を一瞬考えた。

+ 標準ではフラットバーロード用のFDがついている。これを普通のロード用のFDに交換すればロード用レバーで引けるんではないか。RDはそのままロード用レバーで引けるだろう。ただ、標準のVブレーキをロード用STIで引くにはレバー比変換アダプタが必要だし、ちょっと面倒だ……。

+ やっぱドロップがいいなら、コンフォート系のロードかシクロクロスベースのツーリングバイクを新規に買うべきだろうという結論になる。

+ でも新しい自転車を買うのはそれはそれで面倒だ。2台置く場所もないし。結局ドロップをあきらめてAbios Tourerをちまちまパーツ交換していってフレーム寿命まで乗り続けるのがベストなんだろう。

+ そんなわけでSPDペダル(PD-A530あたり)を買おうかな。靴はSH-MT22あたりが見た目によさげだ。

+ せっかくだからフェンダーも交換しようかなー。

■「しろくまベルスターズ♪」(PULLTOP) [GAME]

+ りりか→硯→きらら→ななみで終了。

+ りりか。CVが途中で有栖川みや美に代わったわけだが、これはこれで正解だなー。お話はりりかの弱い所をさらけ出すような話で、ちょっと痛々しい展開。だがそれがいい。ジェラルドといっしょに飲むシーンがよい。あと、エロシーンのめろめろっぷりがすげえ。

+ 硯。りりかやななみと違って彼女自身のドツボ度が低めだったように思う。その分安心して見てられる話だったので、最後にやるよりは中間に挟むとよい話かも。よだれがかわいいぜすずりん :-)

+ きらら。一人だけノエルの人じゃないので、町の人中心のお話。土地に根付くもとと外の人(エトランジェ)の関係、老いて人が亡くなってゆくことを感じる話だろうか。きららがやりたい事と、勉強に対する姿勢のミスマッチがちょっと何だ。

+ ななみ。姫もすずりんもきららもかわいいのだが、やはり圧倒的に「ヒロイン」なのはななみだろう。頼り頼られ二人三脚の冬馬とななみの関係が実によい。

+ 以上。例によって面白かったのですよ。次も買うぜ。ハッピー・ホリデーズ!

Feb.11,2010 (Thu)

■PC系の物欲のようななにか [COMP]

+ なんかムダに資料が満ち溢れているので、ScanSnap S1500を買ってしまおうかとか思ったが、ちょっと高いよな……うーん。

+ ML115G1のBroadcom BCM5721はジャンボフレーム非対応だし、他の2台はオンボードのMarvell YukonでいろいろアレなのでPCIeのNICを買おうかな。今買うならIntel EXPI9301CT一択だろうか。これに替えたらiSCSIのパフォーマンスも上がるかもしれん(←まだやる気かよ)。

■「星空のメモリア Eternal Heart」(Favorite) [GAME]

+ 「星空のメモリア」のファンディスクだが、デモムービーを見れば明らかに夢とメア中心のファンディスク。ま、ほかの娘のちょっと短いアフターストーリーもあるけどなー。

+ 最初は夢とメアしか選べないので、まずは夢から。夢が退院してから退院から、メアの隕石を探し出して、本編のエピローグへ至るまでの物語。夢といちゃいちゃはしているけれど、それだけの話に終わっていないよい話やね。芽愛への過去語りという形式もよい。そして最後の一枚絵が実に反則。

+ 次。メアの前に、千波→衣鈴→こさめ→明日歩→こももの順でミニストーリーを潰すことにした。

+ 千波。本編と声が変わったわけだが、これはこれでよいかもしれん。本編同様、外に見えているものだけが千波ではないという話だろうか。

+ 衣鈴。前向きになった衣鈴の話だろうか。洋くんにめろめろのくせになかなか素直になれない娘だ。

+ こさめ。満月だけじゃなくて「それ」もダメなのかーという話。まぁ、でも、お互いの想いの力が存在を強くするのでOK。

+ 明日歩。後輩にモテモテな洋くんが印象的 :-)

+ こもも。本編で唯一名前で呼びあわなかった二人が名前で呼び合えるようになる話。洋くんの積極的行動に直撃されると思考が飛んで対応不能になるのがたいへんかわいらしい。やっぱメアに負けないくらいこももが好きだなー、俺は。

+ 最後にメア。飛鳥・伊麻兄妹の件の解決編でもある。つーか、姫榊姉妹にも雪菜先輩にもレンにも見せ場があって異能者オールスターキャスト風味。そしてメアははてしなくかわいくて、えろシーンの洋くんはちょっとキチクだ :-)

+ 総評。本編で積み残しだった飛鳥・伊麻兄妹の件もケリがついたし、メアを選ぶと夢がどうなるかも見えてくるしで、よいファンディスクだったように思う。Favoriteの次回作も当然買いますよ。

Feb.14,2010 (Sun)

■きょうのよしなしごと [MISC]

+ 昨日いつものように近所一周(約9.1km)走りに行ったら、5km地点くらいで気持ちも悪くなって足が進まなくなった。引き続き今日一日も体調が冴えない。ま、今年で40歳になるわけだし、無理すんなってことだろうかね。

+ 体重は先日80kgを切ったものの、70kg台に安定する日は遠そうだ。仕事が忙しくなってきたので、以前ほど運動できなさそうなのが悩ましい。ま、今後は月1kgづつくらい減らしていって、今年の健康診断(6月あたり)で75,6kg前後になるのが目標というところか。

+ 去年の今頃の体重は107kgくらいだったはずなので、今でも27kg減なんだよな。ウエスト100cmのチノパンを着ていた俺が、今は34インチのジーンズもユルいわけで、痩せたなーと思うよホント。
