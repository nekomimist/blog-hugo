Nov.21,1999 (Sun)

■鎮花祭(はなしずめのまつり) (Misty/Studio Air) [GAME]

+ 開発はStudio Air。 販売はMISTYっと。

+ 「痕」(Leaf)とか、 「冬虫夏草」(ぷち(Assemblage)) とかと似たテイストかな。話そのものは神話色・心霊色がもっと強くて、 展開自体はもっと軽いけど (^^;

+ 感想は、絵がぷにぷにでよいっすね。 CG塗りもきれいでよし。リアリティとかそういうのを求めてるって方向じゃないけど、 俺的にはこういう方向の絵はすっげー好きだったりする。

+ んで、キャラがなんかいい。一人挙げるなら…「とらいあんぐるハー ト」の小鳥もびっくりの身長127cmの高校2年生・緋美子さんかなぁ。 小さいくせに迫力があって、ギャグも効いててとてもおいしい。

+ まだ全部終わってないけど、 10段階で…んーさすがに8とは言いづらいな…8にかなり近いけど7って感じですか。

+ 万人向けではないけど私のツボです。 よくできました (^^)って感じですな。

■トゥインクルレビュー (F&C) [GAME]

+ F&Cのひさびさの新作。 PALETTEとかVirtuaCallシリーズのかおりがするのでそういうスタッフ陣ですかね (*1)。

+ とっても安定したデキ。驚きはないけど、安心してできる。

+ 原画もCGも高安定。 あさのさんの絵の瞳の描きかたがちょっと気になるけど、 前からこんなんだったかなぁ。もちっと違ったような気がするけど。ま、いいか(笑)

+ ツボな娘は今んところ静夜ちゃんかな。 キャラとしては太めのまゆ毛がかわいいし、性格もいい味出してるし、 シーンとしても「ひまわり」のシーンが印象的だし。

+ ゲーム時間長めなのにエンディングが多いので、全部終わらせるのは 大変そうなのがちとマイナスだけど、現時点では10段階で7かな。
