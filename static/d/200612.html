Dec.10,2006 (Sun)

■最近やってたゲーム [GAME]

+ 最近ちんたら進めてたゲームをまとめて。

+ 「めがちゅ!」(FrontWing)。ジブリールから凌辱分を除くとこういうゲームになるだろうか。一番期待してなかったレアが予想外に一番可愛いかったなーというのが感想か。

+ 「Really?Really!」(Navel)。楓は苦手だが亜沙先輩は好きなので買ってみた。その点では満足した。ところでリシアンサスが脚光を浴びる日は来るのでしょうか?(ぉぃ

+ 「委員長は承認せず!ファンディスク」(Chien)。やっぱり経が好きー。メアリーさんのアフターストーリーが徹頭徹尾妙で苦笑。ま、「ファンディスク」という名前から想像できるものから一歩もはみ出てない感じのモノですなぁ。

+ 「シスターシスター」(ROLL)。誤字脱字がやたらめったら多くてとにかく読み進めづらい。セーブデータ画面のコメント表示がはみ出すあたりとか、いろいろ「時間切れ」臭いデキだったなーと。

■きょうのよしなしごと [COMP]

+ 気が向いたのでkonaのFreeBSDをRELENG_6_2先頭に上げてみた。例によって何ごとも起こらなくてちょっとつまらない :-)

Dec.18,2006 (Mon)

■きょうのよしなしごと [AUDIO]

+ ふと気がつくと、iTunes配下のClassic&合唱の容量が、ゲームの曲/J-POPの容量を超えていた。まぁ、前者はApple Losslessで後者は192kbps近辺のmp3なので、曲数的の差はまだまだ大きいが。

+ AirMac ExpressとLavryBlack DA10を買ってなかったら、こんなにClassic CDの取り込みを進めなかっただろうな。iTunes+AME+DA10に頼る一方で、うちのメインCDプレーヤのLINN GENKIはほどんど火を入れられなくなってしまった。ま、こいつは新居に越したら寝室用のBGMシステムの中心に据えてやろうかね。

+ にしても、iTunes配下に合計53.7GBってのはなかなか扱いに困る容量だ。どこか日頃の作業場所から見えない所に持っていきたいものだが……。

Dec.24,2006 (Sun)

■きょうのまんそん部活動 [MISC]

+ 今日は内覧会ってことでまんそん部活動の日。一ヶ所ちょこっとアレな部分があったのでチェックしておく。しかしあれが我が家になるんだよなー……まだ実感ねぇなぁ(笑)

+ 次のイベントは約一ヶ月後の入居説明会。それまでに史上最高額の振り込みをしなきゃいけないし引っ越しの荷造りも準備しなきゃいけない。フレッツ光プレミアム移転の件やエアコン・冷蔵庫・洗濯機などの必須家財調達の件も段取りとっとかないとマズそうだ。これで、仕事がヒマな状態ならいいんだけど、新機種の開発と現行機種の市場問題調査が重なっててかなり忙しいわけで、これからはちょいと大変そうだ。
