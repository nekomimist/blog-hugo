Apr.5,2010 (Mon)

■きょうのよしなしごと [BIKE]

+ RNC7フレームの値段とのバランスや、New 105の入手時期の問題(早くとも5月末)を考えると、さっさとUltegra一式で揃えるのがいいな。アルテのクランクはRNC7につけるには見た目ゴツすぎるのが悩ましいが、まぁ、フレームが来たらえいやっと方針確定して進めてしまうだろうな、きっと。

Apr.10,2010 (Sat)

■RNC7フレーム到着 [BIKE]

+ 届いたので　手持ちのホイールと合体させてみた。色はいろいろ悩んだが、イリュージョンナイトブラック+Anchor赤ロゴという地味な配色にしてみた。ホイールが安物(WH-R500)すぎな気がするな。

+ これは510サイズなのだが、もうちょっとフレームサイズが大きい方がヘッドチューブが長くなって見栄えがいいんだよな。まぁ、俺の身長と股下ではこのサイズあたりが限界なので仕方がない。

+ ポジションは全パーツ揃った後走りながら決めるつもりだが、フォークコラムは3cmくらいは切らないとダメっぽそう。全部自分でやるつもりなので、切るための工具を手配せねば……。

+ しかし、のんびり夏までかけて組むつもりだったが、フレームが来て気分は盛り上がったので、ハンドル・サドル・ペダルをまず発注。たぶん来週中にUltegra 6700のSTI/FD/RD/CS/BR/FC/CN/BBの8点を発注してしまうと思う :-)

Apr.14,2010 (Wed)

■きょうのRNC7 [BIKE]

+ SM-ST74(アウターストッパー)をゲットしたのでRNC7フレームに装着。無駄にぴかぴかだ。今週発注したULTEGRA 6700一式(BBとチェーンだけDura Ace 7900)はもう発送されるようだし、先週発注したサドルとシートポストとハンドルもたぶん週末には届くだろうから、土日は本格的に作業開始できるかな。

Apr.17,2010 (Sat)

■RNC7完成 [BIKE]

+ 16日夜にパーツが届いたので、17日昼までかけてえいやっと組んで、ついに完成。

+ フォークコラムは現状のポジションだと3cmほど切るといいのだが、まぁ、もうちょっとサドルとの落差があってもよさそうだし、ハンドルを下げるのも試してみたいということで、しばらくはこのまま。

+ んで、組み上がってからさっと30kmほど走ってきたのだが、なんというか、走ってて実に気持ちいい。

Apr.18,2010 (Sun)

■パンクした [BIKE]

+ 昼にRNC7で走りに出たら20kmほど走ったあたりでパンク。たった二日・合計50km走っただけでパンクとは……orz

+ 幸い家まで1kmほどだったので歩いて帰宅できたが、やっぱパンク修理セットはちゃんと持ち歩いとかんといかんな。家に帰ってタイヤを確認すると、ガラスの破片っぽいものが突き刺さっていた。弱いぜStradius Elite Refrect。

+ ま、普通の貫通パンクなのでとりあえずパッチで対応した。ただ、パッチあてたチューブは緊急用ならともかく常用するのはイヤなのと、携帯ポンプで9-10bar入れるのは数回やってもうイヤになったので、チューブ2本とフロアポンプを発注。

+ また頻繁にパンクするようなら、タイヤをRiBMO PTかDuro PTにでも交換するとしよう。

■今後の展開 [BIKE]

+ とりあえず梅雨に向けて固定ローラーを買う。ミノウラV270-2あたりでいい。

+ あと、現状はホイールだけ安っぽいので交換したい。現状のWH-R500は固定ローラー用にして、WH-6700かWH-RS80あたりを狙う。固定ローラー用に安い12-25なスプロケを手配して、今のCS-6700は新しいホイールへ。カンペキ。

+ あとは、バッグとかウェアとかシューズとかは適宜。少なくともペダルがPD-A530なのでロード向けのSPDペダルを買わねばもったいない(*1)。

+ そういやボトルケージも買わなきゃな。2つつけて一方をツールボックスにするのがいいかな。そうすればサドルバッグが不要になってシートポストにフラッシングライトをつけられるわけで。

+ 散財は続くよどこまでも……。

*1: ちなみに、SPD-SLじゃない理由は「歩くから」。

Apr.21,2010 (Wed)

■RNC7雑感 [BIKE]

+ まずは見た目の話。

+ ホリゾンタルフレームではあるのだけれど、RNC7のチューブ形状は懐古厨的な古くささからは逸脱していると思うので、スレッドステムでなくアヘッドステムであることは気にならない。クランクまわりのデザインが特徴的なULTEGRA6700と組み合わせも、実際組んでみたら「これはこれでアリ」と思う。

+ 色については、黒系フレームに赤文字は、2chで見てイイと思ったので通常のブラックのかわりにイリュージョンナイトブラックでチャレンジしてみたが、地味すぎなくて良い感じだと思う。ただ、傷が入ったらごまかしようがない気はする……。

+ 次に走り。

+ 日頃通勤で乗ってのは一般車寄りクロスバイクのAbios Tourerだし、安物RADACに乗ってたのすら20年も前で、他のロードと比べてどうかは言えないのだが、大変気持ちよく乗れている。やはりドロップハンドルはいい……。

+ 地面からの振動は、通勤に使っているフルアルミのAbios Tourerよりやわらかく伝わる。かといってヤワくてフワフワなわけでもない。まぁ、40歳目前の貧脚おっさんでは走りは追及できないので多くは語らない。

+ その他。

+ ホイールにWH-R500を選んだことはちょっと反省。フレームやコンポとのバランスを考えると最初からWH-6700にチューブレスタイヤを買っておくべきであったように思う。

+ あと、比較用にフルカーボンのロードに乗ってみたくなった :-)

Apr.25,2010 (Sun)

■Office 2010のエディション構成 [COMP]

+ 日本向け設定にHome and Student (Word, Excel, PowerPoint, OneNote)がないことに絶望した。そしてPersonal (Word, Excel, Outlook)がムダに高い。PersonalとHome and Business (Word, Excel, PowerPoint, OneNote, Outlook)の値段差を見ると、Home and Businessを買う以外の選択が思いつかんじゃないか。

+ OOoは昔からずっと馴染まないし、会社ではOffice2007を便利に使ってるしで、2010は買ってもいいかなという気分になってたんだが、もうちょっと安くならんもんかねぇ……。

■RNC7のこと [BIKE]

+ ブラケットを持ってる時の感覚からすると、もうちょいとハンドルを下げてもいい気がする。ただ、下ハンを持つときに窮屈なのもイマイチなのであまり下げたくもない。むーん。サドル高とサドル前後位置とハンドル高とハンドルリーチとステム長。パラメータが多くて大変だ。

+ さて、そろそろSPDシューズを買わねば。パールイズミのX-Roadがよさそうな感じだろうか。

+ サイクルコンピュータCC-RD200をAbios TourerとRNC7とで共用しているのだが、タイヤサイズが違うので、毎回設定を変える必要があって面倒だ。ついでにRNC7単体の合計走行距離がわからないのもイマイチなので、もう一つCC-RD200を買わねばな。

+ ああ、また出費……。

Apr.29,2010 (Thu)

■きょうのよしなしごと [MISC]

+ 体重が77kg台に入った。1kg/月減のペースをだいたい維持できてるかな。

■RNC7のこと [BIKE]

+ RNC7に50-34のコンパクトクランクと12-25のスプロケをつけているわけだが、アウターで走ってると、リアは15までで足りて14から下はほぼ使えない。もうちょっと脚を鍛えないとダメだね。やはりローラー台を買わねば。

+ ハンドルポジションを5mmほど下げて、かわりにちょっとだけ前側を高めに変えてみた。これで様子を見て、よさそうならフォークコラムを切ってしまおう。ハンドル上に4cmもコラムが出てるのは大変かっこわるいしな。
