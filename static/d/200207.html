Jul.13,2002 (Sat)

■えろげ [GAME]

+ 何かおはなしを作るとき、えろが必須なことも、不必要な事もある。

+ 後者のものをおざなりにえろシーン追加して売るから妙なことになるんじゃないかと。それは「本質でなくかつ実用にならない(*1)機能を、いかにも本質であろうかと思わせて売る」のに似た行為であって、売り手としての誠意が足りないだろう。えろくないならえろくないと諸元に書くべきだし、そもそもえろゲとして売るべきでもないさ(*2)。

+ でまぁ、前記のよーな事情によって一般ゲームではなくえろゲとして売ることに決めたのだったら、エロシーンはきちんと作り込むのが作り手に必要な誠意だと思うし、その題材が「エロと親和性の悪いストーリー」だったら、エロを含ませる方法をもっと工夫すべきだと思うのさ。

+ そう考えると、「みずいろ」(ねこねこソフト)の手法(*3)はそう悪くないよなーとも思うのだ。なんぼでも増やせるしな :-)

*1: 売り手が「これはえろいんです」と強弁できるか否かが境目と思いねぇ。
*2: ただ、えろゲの方が雑誌がいっぱいあって宣伝しやすくて流通もさせやすいだろうから、理想を語ってもどうしようもない側面はあるのだけれど。
*3: 「みずいろ」のソレがよいという話ではないのでそこは注意。

■「とらかぷっ!」(PULLTOP) 感想 [GAME]

+ 全CGと全回想が埋まって、全記録が「神」になったのでこれにて終了かな。

+ 60年に一度の四方山大大祭。3柱の神は各々パートナーである獲り手を選び「黒」を狩る。とある事情でいきなり3柱のうち獣ノ神・桜姫の獲り手となった主人公優太は1月に渡る祭をどう過ごすのか…てな感じのAVG+SLGなゲーム。

+ SLG部はヌルい。コツさえ掴めば楽勝もいいとこ。NightmareモードとかHellモードとか欲しくなったりしないこともない。

+ 1stプレイは竜ノ神チームで終了。一見やさしげだが怒ると怖い静波と、常識人だが必殺剣術娘の夏葉。最後のえちシーンのシチュエーションが実に好みだった(ぉぃ

+ 2ndプレイは鳥ノ神チームで終了。おバカなチビガキ風味の神様・御翼と、いたずら好きでいつもよからぬことを考えているおさわがせお嬢様・歌夜。始める前は御翼も歌夜も好みじゃなくて、最終的にも御翼は好みじゃないんだけど、素直になった歌夜は実にズルかった。

+ 3rdプレイは獣の神の桜姫狙いで。ちんまくて元気で意地っぱりで虎娘だなんて防御不能にきまってる(笑) こういう系統のキャラを日向裕羅がアテてた記憶はあまりないのだが、なかなかハマってて良し。おかーさんが美人でまたよし :-)

+ 4thプレイは総ざらえシナリオ。微妙にわかりづらい部分がないわけじゃないけど、俺的にはそれなりには消化できたのでまぁいいかなと。

+ ま、絵もよかったし、システムもそう悪くなくて、なかなか楽しかった。10点満点で7点…いや、桜姫のキャラに免じで8点。「佳作」でどーよ?
