Jan.1,2009 (Thu)

■謹賀新年 [MISC]

+ 誰が見てるやらさっぱりわからないこの日記だけれども、まぁ、2009年もほどほどによろしくお願いします、ということで。

+ 以下抱負っぽいものを。

+ 仕事方面。去年は秋までは仕事密度的に厳しく冬は社会情勢的に厳しいという激動の年だったが、今年はどうなるかね。ま、どうあろうが組み込み屋として2009年も生きていくのだろうから、オレらしく開発とマネージメントのバランスを取りつつ仕事をしよう。

+ 趣味方面。去年は散財が多めだったので、今年は抑え目の方向で(*1)。まぁ、幸いPC方面とオーディオ方面は全面的に更新が終わったので、今年はソフト中心で控え目に行けるはず……。あと、時間的余裕が去年よりも多いと思うので、趣味で「何かを創る」方向も手をつけたいな。

■emacs-23.0.60 [COMP]

*1: 家計簿も実運用に乗ったので、散財が「見える」のもポイントだ。

Jan.21,2009 (Wed)

■「ティンクル☆くるせいだーす」(Lillian) [GAME]

+ 積みゲ消化月間ということでやっと終了。

+ ヒロインみんな魅力的ですな。特にリア先輩と聖沙が好き。でも、ナナカもロロットもアゼルも捨て難いね。

+ ナナカ。シンと親父さんのやりとりがちゃんと「ナナカが"ああ"なること」の伏線になっててちょいと面白かったな。あと、告白できなさっぷりが大変愛しい :-)

+ ロロット。この娘は人なつっこさがかわいい。「天使じゃないですよ!」と言いつつバレバレなのがまたよい。恋愛話というよりはアゼルとロロットの二人の関係とそれを支えるシンの話って感じでしたかな。

+ 聖沙。たいへん王道っぽい三角関係シナリオ。やっぱナナカはフラれるシナリオの方が輝いてるよな(ぉ さておき、歌うシーンやキスシーンで、「パティシエなにゃんこ」(ぱじゃまソフト)で冬華を思い出してしまう俺だった。本作にも出てるしね、冬華。

+ リア先輩。年上っぽい所とそうでない所が同居しているのが実によい。微妙に太めなとことも好みかも。恋愛的にも展開的にもあまり重いところのない話だったので、安心してプレイできる話でした。

+ アゼル。不愛想な娘だけど最後はホントにかわいい。そして一人だけとび抜けて貧乳なのが実に不憫だ(ぉぃ

+ 脇役もよかった。メリロットさんが実に好み。味方にいる時の効果が微妙だが :-) あと、パッキーがイキイキとしていて印象的。「ななついろ☆ドロップス」(ユニゾンシフト)のユキちゃんもそうだったけど、中の人はこういうキャラをやらせても楽しいね(*1)。

+ バトルはちょっとランダム要素が強すぎだろうか。後半は苦しい配置からスタートしたら何もできずに終わってしまう。ハンディ有りスタートでも創意工夫で善戦できるシステムの方が好みだが、まぁ、バランス調整が難しすぎるかね……。

+ まーなんつーか、楽しかったですよ。9点。

*1: CV非公開だから中の人に触れづらいぜなー。

Jan.29,2009 (Thu)

■きょうのよしなしごと [COMP]

+ 計画有休取得で一回休み。3月15日までにあと最低3日休まなきゃいかんのだよな……。

+ RESERATOR XTのエラーはたまにフローレートが下がってしまうのが原因っぽい。で、中開けて流路を確認したら、チューブが潰れかけてる部分があった。チューブの固定を少々工夫することで対策はしたけれど、本質的には潰れ防止に何か(針金巻くとか)しないとダメだな。んで、そこまでやってだめなら何かが詰まってる可能性が高いので、いっぺん冷却水抜いて確認しないといけない。面倒臭いのう。

+ クルくるをプレイしていて、どうも発音のタイミングあたりで画面が一瞬止まる現象が出てEXを出せなくなったりするので、切り分けのためにXONAR D2Xを外してRampage Formula付属のSupremeFX IIを有効にしてみた。……やっぱり止まるので別原因らしい。まぁ、XONAR D2X由来っぽいシステムロックも見てるので、ドライバが更新されるまでXONAR D2Xは外しておこうか。

Jan.31,2009 (Sat)

■PC関係で省エネしよう [COMP]

+ 今年の冬はほぼガス床暖房オンリーで通してたのに電気代が高かった。電力を多く食ってるのはPCとAQUOS LC-46RX1Wだと思われる。AQUOSはどーにもならんので、PCの省エネ化を考えてみよう。

  • Vistaマシン FIENA (自作1号, Core2Ex QX9650@4GHz, RADEON HD4870)
  • Windows Home ServerマシンLULLI (自作2号, Core2Ex X6800, RADEON HD3870)
  • FreeBSD 7.1RマシンKONA (HP ML115G1, Opteron1210)
  • Vistaノート RIPPLE (DELL XPS1330, Core2Duo T7100, GeForce)
  • XPノート FUYUKA (Panasonic Let's Note W2B, PentiumM-1GHz)
  • MacOS Xマシン (Mac mini, CoreDuo 1.6GHz)

+ と、全部で6台のPCがうちにある。ノートのRIPPLEとFUYUKAは基本サスペンドしているので問題ないだろう。Mac miniは普通電源が入っていないのでこれも問題なし。というわけで、対策を行うなら24時間運転しているFIENA/LULLI/KONAの3台だろう。

+ FIENAは一番電気を食っているPCだと思うが、何分リビング設置のメインPCなので、パーツ交換での省エネは難しい。HDDのVelociRaptorをSSDに交換すると省エネ効果はありそうだが、金がかかりすぎるので意味がない。こいつは24時間運転を止めるのがいいんだろうな、たぶん。

+ LULLIのCPUはConroe-EX(TDP75W)なので、WolfdaleかYorkfieldに変えればアイドル時の消費電力はたぶん下がる。でも、チョロいのにしないと出費につりあわないし、チョロいのは大幅性能ダウンになるので微妙。一方、HD3870はファンレスのHD3450やHD4350あたりなら4000円程度だし、アイドル状態の電力が10-20W程度違いそうなのでメリットありそうな予感。

+ KONAはOpteron1210(TDP103W)をAthlon X2 5050e(TDP45W)に変更すれば省エネな上に性能向上もできそうだ。7000円程度で安いしな。ML115G1での運用を止めてLULLI上でVMware Serverを動かすのもよいかもしれないが、インストールが面倒臭いのが難点 :-)

+ そんなわけで、Athlon X2 5050eとファンレスRADEON HD4350安物カードをぽちっと発注。

■家計の話 [MISC]

+ 去年12月20日頃から家計簿をつけているわけだが、2009年1月を終えたことで、ついにまる1ヶ月分の収支データが手に入った。

+ 1月ということで団信特約料の支払いがあったのが地味に痛い。あと、本+同人誌+えろゲ+CD+DVDで10万超えはさすがにマズい。正月頃に同人誌買いすぎです俺。ま、それでも月の収支はなんとか黒字だけれども……。

+ 残業削減による収入減という状況は今後も続くわけだが、忙しくない分、自炊で食費を減らしやすいだろう。電気代の削減も進めよう。本やゲームは「ついでに買っとこう」レベルのものを買わない方向にする。ま、そのへんを行えば、今月分から数万支出を減らすのはたやすいハズだ。
