Mar.19,2011 (Sat)

■ヘルシオAX-CX1 [MISC]

+ 先月買ったヘルシオAX-CX1がけっこう便利である。

+ ウォーターオーブンっつーても気にするのは、料理開始前に水を入れることと、終ったらつゆ受けの水を捨てることくらいで特に気にすることはない。

+ エンジンはAX-MX1(容量26L)のものと同じなせいか、オーブンのパワフルさは今まで使っていたPanaのオーブンレンジ(庫内容量15L)とは比較にならない。チルドピザなどがよい感じで焼けて大変嬉しい。

+ 電子レンジ機能も、オートは貧弱だが、庫内がそこそこ広いので大き目の弁当も気にせずあっためられるのは良い。前のオーブンレンジも残してあるのでオートでやりたい時は前のを使うし。

+ 出てすぐの頃より値段がこなれてきているので、独り者にはおすすめかな。

■Dyson Digital Slim DC35 [MISC]

+ これも買ったんだぜ。まあ、掃除機にしては高いけど、この程度の金額でへこたれる収入では全くないので気にしないことにした :-)

+ 使いたいと思った時にほいほいと取りだしてさっと掃除機かけてさっと充電状態に戻せる便利さはすばらしい。おかげでメインの掃除機(自社製 :-p)を使う頻度が大幅に下がってしまった。

Mar.22,2011 (Tue)

■さいきんやったえろげー [GAME]

+ けど感想を書く元気がない。

+ 「よついろパッショナート」(SkyFish)。悪くもないが良くもないというデキなのだが、個人的には案外好きだった。音奈も奏も響花も好きだが、はづみんがなんか好きだったな。大変クセのある声とキャラだがね。

+ 「With Ribbon」(Hulotte)。誰がヒロインに選ばそうかで、はるかの母親が切りかわるので、はるかの言動が変わるというのはなかな面白かったな。ヒロインとしては、陽奈・ゆみみ・華澄さんあたりが好み。はるかは沙蘭はるかが実に好み。よいゲームでしたよ。

+ 「Flyable CandyHeart」(ユニゾンシフト)。例によってただいちゃいちゃするだけのFDを作らない人達だなー。まぁ、それがいいのだが。「Flyable Heart」と「君の名残は静かに揺れて」をやったならば買わない理由はなかろう。

+ 「ラブラブル」(SMEE)。バカップルぶりがたいへん頭が悪いのだが、わかっててそれを狙って作ってるゲームにそれを言ってしかたがないよね :-) さつきのかわいさに悶絶。

+ 「アネカノ」(chococo)。姉バカっぷりがたいへん頭が悪いのだが、わかってて(以下略)。肉でぷにぷにって感じの絵はけっこう好みであった。椿お姉ちゃんが一番好きだったかな。

■きょうのよしなしごと [COMP]

+ ASUS P8P67 Deluxe交換の連絡が来ない。どーもASUSはH67系の交換が先行していて、P67系の交換はまだ開始されていないっぽいな。

+ CygwinのRuby 1.8.7のpatchlevelがp72と古いのが気になっていたので、代わりにmswin32版を入れたら、migemoが動かなくなった。Cygwin版rubyを使ってmigemoの辞書を作った環境だったので、mswin32版で辞書を作りなおせばいいのかな?と思ったらそれもエラーになる。Cygwin版のRubyを自前でビルドしたほうがいい気がしてきたのでそこで考えるのを止めた :-)

+ Cygwinでのビルドは_longjmp()あたりでエラーが出るのを対処すれば普通に通る。make testは通ってもmake test-allが通らないけどな……。
