Aug.5,2007 (Sun)

■「HoneyComing-ハニーカミング-」(HOOK) [GAME]

+ 「ハニカミ」と言いたかっただけかね、このタイトルは。

+ 麻里乃。髪飾りが洗濯挟みっぽいなーと思ったら本当にその通りで苦笑。なかなかやわらかくなってくれない娘をおっかける形になるので、主人公がなんかストーカー風味に見えなくもない。終盤は「奴」がもうちょっときちんとヤなヤツだったほうが盛り上がったような気もしないでもないな。

+ 由馬。麻里乃ルートを先にやっていると、"麻里乃が食料をガメるのを邪魔する大食いの先輩"という印象が強い。さておき、とある事情で「自分のことなんてどうでもいい」と思っている由馬に主人公が積極的にかかわりに行く話なので、やはりどこかストーカー風味だ。

+ 未央。あのシーンは真剣なシーンのくせに仮面のせいで大笑いという……。後半の演劇の所は、主人公の行動がちょっと思慮に足りない(倒れたら舞台が台無しになるのに、その準備段階で体調が不調なくせに無理をする)のが気にくわんかなー。

+ クレア。イタリア語なら"r"はきちんと巻き舌してほしいぜ。クレアより椋子の方がイイなとか一瞬思ったりもした。いい娘なのだが、何か食い足りない感じがするなぁ。

+ 朝陽。素直になれないあまりに理不尽な怒りを爆発させてしまう困ったちゃんな娘なのだが、愛に目覚めた主人公は、そんな朝陽を我慢強くもつかまえちゃうのでした。まる。朝陽の両親の問題がアレで解決するとか、展開がいろいろこっぱずかしい。まぁでも、朝陽は「ちゅんちゅん」で全て許したい。

+ ま、HOOKらしいズレ方をしたゲームだったな。でも嫌いじゃない。7点。

+ 余談だが、Windows Vista(32bit)環境だと終了時にSEGVる。おかげでVistaの信頼性モニターの値がガタガタになる :-)

■「あかね色に染まる坂」(feng) [GAME]

+ 前作主人公の秀樹が最初っからムダに持ち上げられていたのに比べると、今回の主人公の準一は基本2.5枚目で後半周囲の評価が上がるタイプ。俺的にはこっちの方が好みだ。

+ なごみ。デレ方とかが「ぱるてるチャイムContinue」のルーシーにいろいろ近い感じがするねぇ(*1)。後半、準一の素直な気持ちを表したセリフで頬を染めるようになるのが実にかわいい。

+ つかさ。もともとこういう悪友タイプのヒロインの扱いは難しいんだろうが、個人的にはキャラ造形が下品方向に行きすぎてる気がする。もうちょっと上品にエロいことを言う方が好みだな。

+ 湊。湊シナリオにはわざと主人公(とプレイヤー)をイラっとさせる展開があって、俺も相当イラっとさせられたで、上手い演出だったと言えよう。後半は展開そのものはさておき湊の意図が見えづらいので、幕間のヒロイン視点で「湊が何を思ってそうするのか」をもっと描写すべきたったような気もする。まぁ、湊がよいキャラなので許す。

+ 観月先輩。「青空の見える丘」もそうだったのだが、サイトウケンジは「いつも笑顔の先輩には裏がなければならない」とか「先輩は胸が大きくないと!」とか思っているのだろうか。いやまぁそこがいいのだがっ。後半、すげぇテレてくれるのがかわいくてよいね。

+ 優姫。ズ抜けたお嬢様のはずなのに、描写上あんまりお嬢様感がなかったなーと。お話の最後あたり、ベタベタな展開なのは悪くないのだが、主人公のへたれ行動の意図がが見え見えすぎなのがちと惜しい。優姫は、観月先輩シナリオや湊シナリオでの脇役としての立場の方がなかかよかった。

+ ミコト。「青空の見える丘」ではわりと好みだったのに、今回ちょっとイマイチな気がしないこともないなー。別の話にせずに本筋に混ぜこめなかったのだろうか、とか。

+ 総評。「あかね色に染まる坂」ならではの"何か"はあると思うのだが、その"何か"はちょっと突飛な感じがあるってところだろう。一部で地雷扱いする人もいるが、これを地雷と言うのはいくらなんでも許容範囲が狭すぎだと個人的には思う。まぁ他人の評価は気にしない方向でプレイするが吉。俺としては7点はあげられる。佳作。

*1: まぁ中の人が同じだしなー。
