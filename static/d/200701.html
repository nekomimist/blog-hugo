■きょうのよしなしごと [MISC]

+ 今日は仕事始め。っつーでもうちの会社は新年一日目は12:00までだったりするのであっという間なのだが :-)

+ 残念ながら1/9に休みを取る作戦は失敗した。とはいえ、まんそん部関連でいろいろ振り込みにいかないといけないので、来週中に休みとらないとマズいな……。

+ 正月に実家に帰ったらまんそん部関連で親が1/11ほど支援してくれることになってしまった。俺はいらんと言ったのだが「それくらいさせろ」と言われて負けてしまった。まあ、いい。いつか何かで返してやろう。

■「ぽっと -Rondo for Dears-」(あんく) [GAME]

+ 世のえろげーにファミレスモノは多いのだが、主人公がエリアマネージャーとして数店舗を統轄するけっこう偉い人で、しかし人の気持ちのわかるよい奴――それゆえに舞台となるトリアノン北橋店を気に入って、なのにその店を閉店させる決定を下す側の立場であることを悲しんでしまうわけだが――というのはあまりなかったように思う。

+ 一人目は静。社員のヒト。最初主人公の一誠に反発するけれど、冷静になって見ると彼は彼女の「理想の男性」の条件をほぼ揃えていることに気付いてあとはメロメロという感じですなー。

+ 次に日和さん。店長で未亡人で年上。エロかわいい年上っつーのもツボだし、中の人がぴったりあってて実に良かった。亡くなった前の夫への心の整理がつかないままに、一誠との体の関係がなんとなく進むんだけど、一誠がわりと思慮深いせいでよい感じの展開に仕上がっていたと思う。イチオシ。

+ 明日葉。バイトの娘。超絶にドジなのは目が悪いくせに矯正をしないせいなのはすぐにわかるのだが、実はその奥にもう一つ真の理由がありました、と。そういう事情自体は悪くないが、彼女自体が俺的にはわりとどうでもよかった(ぉ

+ 輪廻。バイトの娘。この娘自体はとんでもねークセ者だが、話自体は一誠のことが好きだという明日葉を、自分の想いを抑えつつ応援しようとする恋愛モノの王道。まあ、なんだ、とにかく輪廻はかわいかったな、と。

+ 希美果。ちょっと乱暴なしゃべりの榊原ゆいっつーのも新鮮かも。「そっちかよ!」とか「ほら、休みたくなったら膝枕〜」とかノリが楽しくかつかわいいのがツボ。話はどうにもガチャガチャしているのだが、そこが却って彼女というキャラには似合っていたように思う。ただ、えろシーンが薄いのだけが残念だ。日和さんばりにやってくれよー(ぉぃ

+ BGMがヘンだとか、やけに地味だとかの文句はあるのだが、俺的にはあんまり嫌いなところのない佳作って感じでありました。7点あげよう。

■「遥かに仰ぎ、麗しの」(PULLTOP) [GAME]

+ 11月24日発売だっつーのに手をつけられたのが2006年12月末、終わったのが2007年1月頭と来たもんだ。最近ゲームの消化が遅いよな、俺。

+ 分校組(栖香→美綺→邑那)→本校組(殿子→梓乃→みやび)→エピローグで終了。

+ まず分校側。こっちが丸谷シナリオかね。主人公はふつーの新人教師って感じ。実は第一印象ではすみすみが一番好みだったので後まわしにしたくなったけれど、某所のアドバイスに従って上記の順で進めてみた。

+ まず栖香。彼女の両親の思いは最初から読みとれちゃうので、単にすみすみがからまわりしてるだけの話に見えちゃった気もするね。キャラ紹介だと中心人物っぽいのにねぇ……。しかしすみすみは愛いのでなにもかも許す。

+ 続いて美綺。栖香の件はここでも発生するけどあっさり解決してあれ?と思ったら思いもよらぬ展開になってほぼオールスターキャストな展開に。分校側で主人公の過去のトラウマにきちんと触れるのはみさきちだけだったよーな気がする。

+ 分校最後は邑那。どーにもドロドロした感じの話になったわけだが、最後の最後のクソジジイがあっぱれだったので感じは悪くない。ただ邑那自体は実にどうでもよかった :-)

+ 続いて本校側。こっちが健速シナリオか。主人公は過去のトラウマがらみで激情に流されさえしなければ常に正しい……って、どんな超人ですか。好きだけど :-)

+ 最初に殿子。なんか愛いなぁこの娘。こんな娘(むすめ)なら俺も欲しい。しのしのルートやみやびールートの司よりこのルートの司の方が子供っぽい前向きさを持ってるような気がするやね。最後のオチはトントン拍子すぎるきらいはあるけれど、このルートの司と殿子にはああいう幸せが似合うかも。キャラ的にはイチオシ。愛い。

+ 続いて梓乃。怖い事考えてるわりにやることがかわいいと思ってたら、思いつめたあげくにとんでもない事をやろうとしてとんでもないことに。対人恐怖症でいつもびくびくしていたしのしのが、最終的には主人公の全てを受けとめるっつーのはなんかカッコよかったかも。キャラ的にはイマイチかと思ったが存外面白い娘だった。

+ 最後にみやび。リーダさんの怒り→理解の流れとか、みやびが司の脅迫の真相を知る前後とか好きだなー。あと、司に甘えるみやびはかわいいし、世話焼きリーダさんも劣らずかわいくて良し。「司の根っこにある暗いもの」はこのルートでも首をもたげるわけだけど、そこは司が育てた周りの人との優しい関係と、なによりリーダさんに支えられてきれいに解決。しかし、あの流れならえちシーンはリーダさんもいっしょじゃないと不公平と思うな俺は(ぉぃ

+ あとは、立ち絵のない脇役の方々の存在感が大きいのはよかったように思う。個人的には下手に立ちCGがあるよりよかったかも。分校側の双子とか神とかいろいろ強烈なメンバーが揃っているが、本校側も結城ちとせとかメイドの係長さんとかが地味によい。

+ 評価は9点あげようか。よくできました。

+ ……しかし、PULLTOPと言われて思いうかべるのが藤原々々原画チームの方になりつつあるなぁ、俺。

Jan.14,2007 (Sun)

■FIENAのシステムドライブを変えてみる [COMP]

+ 手持ちのパーツを整理してみると、WD Raptor WD740GDが2つ余ってた。FIENAにはICH8Rが載ってるので、試しにシステムドライブをWD Raptor WD1500ADFD単体からWD740GD×2のストライピングに変えてみた。

+ ICH8Rでアレイを作った後、Acronis TrueImage 10 HomeをCDから起動して、WD1500ADFDからWD740GD×2のアレイに中身をコピーした後、in-place upgradeかけてICH8Rのドライバ入れて作業は完了。時間かかったけど作業自体は楽。

+ んで、WD740GDは一世代前のRaptorなので単体では今となってはそんなに速くはないのだが、さすがに2台でストライピングすると現世代のRaptorであるWD1500ADFDを上回る。しかしHDtuneで見ると全域100MByte/sec程度なんで何かで頭打ちしてる予感。なんだろうねぇ。

+ これでWD1500ADFDを浮かせられた。Windows Vistaが出たらこいつに入れて試そう。

■「雪のち、ふるるっ!〜ところにより、恋もよう〜」(RUNE) [GAME]

+ 積みのこしのゲームがなかなか減らないよー。

+ 最初はなるみ。学校の妖精。ふるると同じ"妖精"のカテゴリのくせいにいろいろふるるといろいろ設定が違う\のが謎。声はぎりぎり許せる感じ。

+ 次に紗璃奈。高飛車お嬢様。……えーと、なんかあんまり覚えていないな? 高笑いぶりに無理を感じたのは覚えてるが :-)

+ 3番目は野々花。いわゆるダメ系幼なじみだが料理は得意。そして基本に沿って前から主人公の一樹の事が好きなわけだが、なぜか他の娘のルートであっけらかんとしているのが気にくわんかも :-) が、野々花は案外かわいかったので許す。

+ 4番目にふるる。デカい妖精槍(フェアリーランス)を武器に戦う雪の妖精のプリンセス。「……バンキッシュ!(ふるるの勝ちです)」がキメセリフ。なるみの時と違って、ふるるが妖精であることに主人公が悩むせいか、えちシーンの回数でワリを食って不憫かも。キャラ的にはもうちょっときりっとしてる方が好みだな。

+ 最後に恋夏。親友の立ち位置にある女の子。ショートカットでボクっ娘なところとか「めっ!」のセリフと立ち絵とかがツボ。ヒロイン中で随一の常識人で落ちついてるのも○。声も好みな系統で、俺的にはイチオシ。

+ 絵について。「思春期」の実験的な絵柄からちょっとだけ以前の方向に揺り戻した感じかするやね。たいへんかーいいです。好し。

+ CV。微妙と巷で言われているけれど、一部(*1)を除けば俺的にはそんなに悪くなかった。まぁ、俺はえろげーのやりすぎで、よく出てる方々にちょいと飽きてきているせいもあるかもしれんが。

+ シナリオ。鬱鬱としてないので進めやすいのだが、結局どのルートでも大差ない話なのは弱点だわな。一応ラストバトルでは各ヒロインの見せ場があるんだけど微妙だし。ミニゲームについても、それがあることに効果があるかと言われると悩ましい。カタルシスに欠けんのよね。

+ まぁなんだ。評価としては「ふつー」って感じ。まぁ次も買いますよ。次作は"新生RUNE第1弾"を銘打たれた「Purely」。原画はCAGE主力のむにゅう氏らしいので期待したいところ。

*1: 姉とか姉とか姉とか、光歩とか光歩とか光歩とか。
