Aug.20,2011 (Sat)

■FIENAが壊れた話 [COMP]

+ FIENAの水冷系の掃除しようってことで、クーラント抜いて、リザーバはシャカシャカ掃除して、数時間水だけで回して全体をきれいにする手順で作業していたのだが、、最後の水だけで回す時にクイックカプリングを一ヶ所はめ忘れて起動してしまった。2分くらい水が回らなくて首をかしげてたのだが、はっと気付いてクイックカプリングをつないだら水は回ったのだが、こんどはマシンは起動しなくなっていた。

+ M/BはASUS P8P67 Deluxeなので、何の要因で起動しないかLEDの点灯状態でわかるようになっている。よく見るとCPU LEDが赤々と点灯していた。水冷系が止まってたからCPUを焼いちゃったかな? ということで代替のCore i7 2600KをAmazonで手配して換装した……が、それでもCPU LEDが点灯して起動しなかった orz。

+ これはM/Bが壊れたかな? ということで、その日のうちに直したかったので近所のパソコン工房でP8Z68-V Proを買ってきて換装。そしたら、もともとついていたCore i7 2600Kでも普通に起動したさ orz。

Aug.21,2011 (Sun)

■PC群の再構成 [COMP]

+ そんなわけで、Core i7 2600Kが一個余剰してしまった。せっかくなので、自宅サーバ群の再構成を考えよう。

+ うちの常時稼働マシンは以下3台。

  • メインPC FIENA (Core i7 2600K@4.5GHz, Windows7)
  • ファイルサーバ LULLI (Core2 Extreme X6800@3GHz, Windows Home Server)
  • iSCSI target/外へのsshの提供など KONA (Core2 Extreme QX9650@3.6GHz, FreeBSD)

+ この構成だと、やはりKONAをCore i7 2600Kにするべきだろう。Z68のM/Bを買ってグラフィックカードは使わず内蔵GPUを使う。メモリを16GBにすればzfsのARCも増やせる。グッドだ。

Aug.29,2011 (Mon)

■FIENAさん連鎖故障と水冷一旦停止の話 [COMP]

+ いろいろ壊れまくりですよ!

+ 先日P8P67 Deluxeが壊れて、P8Z68-V PROに交換して一息ついたと思ったら、こんどは前触れなくRADEON HD5850が死んだ orz。

+ こいつは濡らしてないのに何だろうと思ったが、まあ、P8Z68-V PROだからCore i7 2600K内蔵GPUでしのごう、ということでRADEONを外した。ただ、水路を変更するのが面倒でケース内にななめに立てかけておいたら、水枕の横のパッキンから水漏れしてXONAR D2Xをショートさせてしまった。なんでだよ orz。

+ さらに、あわててCPUだけの水路に継ぎかえていたら、その過程でPIONEER DVR-S16Jを濡らして調子が悪くしてしまったので、もうFIENAの拡張パーツ系はほぼ全滅。

+ そもそも外付けGPUがないなら水冷はいらんよね、ということで一旦水冷系を外すことにした。4.5GHz運用でリテールクーラーは心許無いので、ちょっとだけマシなCPUクーラーを手配してみた。

■そしてFIENAさんの行く先は [COMP]

+ グラフィックは何かの用途で力不足を感じた時点で買って、その時点で再水冷化の検討をしよう。中古でRADEON HD5850を買いなおそうとも思ったのだが、パッキン劣化で水漏れした水枕を再利用するのはアレなので、水枕ごとRADEN HD5850の事は忘れる :-(

+ サウンドカードは、デジタル出力である以上P8Z68-V PROオンボードのRealtek ALC 892で問題あるまい。XONAR D2Xは電気食いだったので省エネって観点でもこっちの方がよい。Realtekのドライバがクソで悲鳴を上げることになるかもしれないが、まあ、クソって点ではXONARも遜色なかったし :-)

+ DVDドライブはPIONEERのDVR-S17Jを手配してみた。LGとかLITE-ONとかにあまり好印象がないからねぇ。
