Dec.5,1999 (Sun)

■SPARK! (Cat's Pro/Pandahouse) [GAME]

+ 発売はCat's Proブランド。ぱんだはうすの抱える2つのブランドのうちの1つ。

+ 簡単に言ってしまえば、恋愛AVGなんだけど、

SPARKにはとりあえずのオープイングとエンディングが存在しますが、 別に結末に向けて収束していくような話ではありません。 どこからはじめても、どこで終わってもかまいません。 お気に入りの詩集や短編集を、パラパラと気ままにめくるような感覚で、 プレイしてもらえたら幸いです。

+ と、パッケージ裏に書いてある通り、独特のノリでゲームが展開します。

+ ゲームの目的は47のキーワード(エンジェルヘア)を集めることだけど、 集まったキーワードをを引きついで次のゲームを始められるので、ま さにパッケ裏の言葉に近いノリでゲームをやってると、いつのまにか 集まるでしょう……といいつつ、私はヒロイン全員のエンディングを 見たところで39個しか集まってなかったりするので、けっこう面倒な のかも (^^;

+ テンポ的はCat's Proの前作「微熱・情熱」に近くて (*1)、 設定の「ふわふわ」感は、ぱんだはうすのもう一つのブランドMelody の1997年春発売のゲーム「メロディ〜恋のメッセンジャーガール〜」 に近いって感じです。

+ しかし、この独特のテンポはぱんだはうすにしか作れないでしょうねー。 このノリが合う人には楽しいゲームだろうけど、合わない人にはとこ とんダメだとも思います。 んで、「暗闇」の頃からぱんだはうすデフォ買いな私にゃ当然合うので、 評価としても10段階で7…よりちょっとだけ高いって感じです。

+ よーするに変で楽しいってこと。うんうん。

■Alive (Witch) [GAME]

+ 発売はWitch。どうやらPinPaiの「Missing」を作ったチームらしい。

+ 以前から友達だった女の子とやっと恋人同士になったら、彼女が事故 で……ってな、どっかで聞いたような設定のゲーム。

+ 文章に愛がない。 これで愛を込めたつもりなら技倆がない :-p

+ シナリオ自体も展開が読める。読めすぎる。 このゲームを通して一体何を描きたい? 伝わってこない。

+ BGMにWMA使ってきたのは面白いかもしれない。が、なんかエンコード をミスってるのか、途中で一瞬片チャンネル抜けることがある。 こんなこと事前にチェックすりゃわかることじゃん…。

+ というわけで、総じて楽しくない。 「Missing」の方が俺的にはマシかなぁ…てことで10段階で4点。

*1: そういや微熱・情熱のヒロインのうち2人が、ゲスト的に出てたりも。

Dec.14,1999 (Tue)

■Cygwin上でrubyをmakeする話 [COMP]

+ 難しくない どころか、実のところ ruby-1.4.3.tar.gzを展開して、ruby-1.4.3ディレクトリ直下で、

./configure; make

+ するだけだったりします (^^; (*1)

+ あとはてきとーなディレクトリに放りこんで環境変数RUBYLIBを設定すれば、 バイナリ配布同様に使えるでしょうし、 /usr/localほげほげ〜ってなUNIXライクなディレクトリ構成で使ってる人は、 私みたいにmake installしちゃうのも悪くないしって感じですね。

■改造 [nDiary]

+ 私がnDiary-0.7.0をいじった箇所というと、

  • 改行を<br>に変換するのをほぼ止めた
  • align属性やとかwidth属性とかを全部CSSに移した
  • footnoteの<div>〜</div>の最後の不要な<br>を削除
  • Diary::makeParagraph(str)に""が渡ると妙な<blockquote></blockquote>を吐くのを修正
  • 「Code:」を指示したときに、<code>の中に<pre>を入れるけど、これはhtml的にはマズいはずなので修正
  • その他もろもろ

+ という感じなので、今のところはバグ修正というよりは、 私が気にくわないところを私の趣味に合わせただけって感じです (^^;

+ ま、いちおうndiary-lib.rbの差分。

■でも [nDiary]

+ 時間がとれたら機能追加とかするかもしれません (^^;

*1: バイナリ配布と全く同じ構成にするためには、いくつかの拡張モジュール をさらに自分でmakeしないとダメだけど…。

■Tyan Tiger100のBIOS [COMP]

+ TyanのTiger100のBIOSページを見ると念願のBIOS Ver.2.00が出ている。でも、

Added new processor support, and latest Win2000 support. Does not support Deschutes * processors or 133Mhz bus processors.

* Deschutes referes to processors with 66Mhz front-side bus and 2.8V core

+ ま、冷静に考えれば、

  • FSB66で2.8VのPentiumIIのサポートは切ったよ。
  • FSB100のCoppermineに対応したよ。

+ ってことなんだろうけど…でも、これが正しいって保証は誰もしないよな ^^;
