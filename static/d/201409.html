Sep.6,2014 (Sat)

■艦これ。[GAME]

+ 三隈さんが航巡初のLv99になったので29人目のケッコンカッコカリ。そこでも「くまりんこ!」とはさすが三隈さんだ。装備は20.3cm(3号)連装砲/瑞雲12型/熟練艦載機整備員/20.3cm(3号)連装砲にしているが、火力100を超えるし開幕爆撃もするし一人で制空値7を稼げるしでなかなか強い。……利根筑摩と比べてはいかんがな!

+ 扶桑さん山城さんもLv99になったので、30人31人目の以下略。交互に1-5/4-3対潜担当と4-3随伴火力をやってたらこんな所にまで来てしまった。ケッコンオコトワリ勢の山城さんだが、カッコカリ後母港台詞は案外よい感じだったりして面白い。

+ そして五十鈴と由良さんが軽巡初のLv99へ秒読み。この二人も1-5/4-3対潜専用になりつつあるなー。五十鈴改二に水偵が積めるようになったり、由良改の装甲が上がったりしないかなぁ……。

Sep.13,2014 (Sat)

■艦これ。3-5攻略 [GAME]

+ 昨日のメンテで北方のExtra Operation海域3-5が開放されたので突入。

+ 2-5は重巡航巡の北ルートでやってる俺なので、とうぜん3-5は南ルートの水雷戦隊ルートを選ぶぜ!

+ 軽巡は神通Lv79→80(20.3cm×2/水観)と大淀Lv67(20.3cm×2/夜偵/32号電探)の二人のどちらかを旗艦に投入して運用。駆逐は綾波Lv120、夕雲Lv120、ヴェールヌイLv96、島風Lv95→96、夕立Lv94、時雨Lv94、雪風Lv87→88から5人を適当に選んで突入。

+ キラづけしなかった前半はボス到達率2/7くらい。初戦の昼戦で砲撃食らって大破や雷撃食らって複数大破などの事故だらけで苦笑。後半は駆逐艦全員キラづけして、駆逐艦2人に新型缶を積んで行ったら、ボス到達率は2/3に向上。最後もあっさり雪風と夕立が一発づつボス軽巡に入れてゲージ破壊した上にS勝利。

+ やはりキラづけは重要だな! (←キラキラ信者脳)

■艦これ。[GAME]

+ 五十鈴と由良さんは9/10にLv99に到達して32人33人目のケッコンカッコカリ済み。この次は今Lv97の北上さまなのでまだまだ先だなー。

+ 8月も500位以内に入ったので、4つ目の試製35.6cm三連装砲と3つ目の瑞雲12型をゲット。瑞雲12型(六三四空)が欲しいなぁ。

Sep.20,2014 (Sat)

■艦これ。[GAME]

+ 金剛型が全員Lv126〜127になって戦艦系の中でレベルが突出してきた。

+ Lv120台前半の伊勢型、Lv110台中盤の長門型大和型、Lv100台後半の扶桑型とあまり差が開くのもなんなので5-4巡回から金剛型を抜いて重巡系×3+ドラム缶担当(大淀or航巡)+正規空母×2にしてみた。まあ、特に問題ないかな。

+ ビスマルク建造は10連敗中。ビス子が来ると金剛型のレベルが上がりすぎる問題がかなり改善するんだがなぁ……。

Sep.23,2014 (Tue)

■艦これ。まるゆ [GAME]

+ ビスマルク狙いの大型建造で7人目と8人目のまるゆが出た。そろそろ最初の1人以外も改に上げていって誰かに食わせないと母港が溢れるなぁ。

+ 実利で言えば素の運が高くカットイン火力も高い北上Lv97(運30)・妙高Lv93(運32)・綾波Lv120(運40+ケッコンボーナス3)。純粋に愛ならば筆頭嫁の鳥海Lv139(運10+ケッコンボーナス5)だが……ケッコンボーナスが最低値だった綾波にあげるのが実利と愛を兼ねてていい感じ?

Sep.29,2014 (Mon)

■艦これ 夏イベの娘たち。 [GAME]

+ 雲龍さんがLv83、大淀さんがLv80になった。5-4に投入できる娘らなので非常によくレベ ルが上がるな。

+ 一方、磯風・時津風・春雨・早霜・清霜はLv20-22jatari。改Lvが30の娘らはともかく、磯風だけはもうちょっとペース上げないとキツい。でも、春イベ入手艦の谷風天津風もまだLv40ちょいなわけで……駆逐艦育成は悩ましいぜ。

■艦これ。 [GAME]

+ 6-1は大鯨Lv20・大淀Lv80・しおいLv95・はっちゃんLv94・イクLv94・ゴーヤLv95で挑んだところ大破撤退2回。うーん、初手南側ルートも試してみるかな……。

+ ビスマルク建造13回目は長門 orz。13回中3回が5時間だったけど陸奥・陸奥・長門。
