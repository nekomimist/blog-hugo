Oct.3,2013 (Thu)

■最近の艦これ [GAME]

+ 3-4と4-4を終わると5-xに行けるのだが……元気が出ずにちょいちょいと育成ばかりしていた感じ。

+ まずは出遅れていた大井っちを同等レベルまで上げた。今は大井っちがLv59、北上さんLv61。それにしても大井っちはなかなかデレないねぇ。

+ あと、五十鈴の改二化のために育成してて、改装MAXになった軽巡が攻撃食らわないし夜戦強いし燃費いいしで頼りになることに気付いたので、2-xで行うデイリー消化組に長良・名取・球磨を編入してみた。よい感じかもしれない。

+ あと、積み残してた新「三河艦隊」出撃任務を終わらせた。天龍・古鷹・加古・青葉・鳥海・衣笠で2-3ボス攻略なのだが……鳥海(Lv48)は今でも摩耶(Lv45)と共にルート固定用重巡として鍛えてあるのでよいとしても他が厳しい。全員改造Lvの25を越えさせて、お気に入りの古鷹だけ改装MAXににして出撃して一発で完了。古鷹は重巡の中では弱い子なんだけど、癒し系でいい。今後はたまにデイリー消化につきあわせよう :-)

+ あとは駆逐艦。4-4ルート固定用の島風・響はLv50になった。3-2攻略組の残りの暁・雷・電・磯波(Lv33-34)は3-2終了後はキラキラ遠征組に編入してちくちくレベル上げ中。この娘らもLv50までは行こう。

+ 艦娘はあまり増えてない。加賀と鬼怒がドロップしたくらいか? 建造はオール30で流しているので長門・陸奥・飛龍・翔鶴・瑞鳳・雪風・伊58あたりは知らない子のままである。ま、4-4攻略で誰か落ちることを期待しよう。

+ 寝る前に4-4のゲージを一つだけ進めてみた。陣容は赤城Lv76, 金剛Lv74, 伊勢Lv73, 摩耶Lv45, 島風Lv50, 響Lv50。中破小破含みではあったけど、まぁ、なんとかなりそうな感じ。

Oct.6,2013 (Sun)

■土日の艦これ [GAME]

+ ちょっとだけやる気が出たので土日で4-4の攻略を終えた。

+ 10/3に書いたとおり、1回目突入時は響Lv50/島風Lv50/摩耶Lv45/伊勢Lv73/金剛Lv74/赤城lv76。大破なしで楽勝。思えばこのときが一番楽だった。

+ 2回目が響Lv54/島風Lv55/摩耶Lv47/日向Lv63/金剛Lv75/赤城Lv78。4-4-1で赤城や響が事故ったり、夕立のレベリングにつきあったり、デイリー消化したり演習したりでレベルがけっこう上がってるね。伊勢は入渠中だったかこの回だけ日向が代役。島風中破摩耶大破日向小破で夜戦突入して、金剛の夜戦ラストショットで勝利デース!

+ 3回目終了時が響Lv56/島風Lv56/伊勢Lv77/金剛Lv76/赤城Lv79/摩耶Lv48。島風が攻撃を吸って大破、あとは赤城と摩耶小破で夜戦。摩耶さまの夜戦ラスト連撃で勝利。

+ 4回目終了時が響Lv56/島風Lv56/伊勢Lv77/金剛Lv76/赤城Lv79/摩耶Lv49。夜戦突入時に むこうは姫小破フラ潜小破、こっちは島風中破に伊勢金剛小破だったので勝利を確信。最後は3回目同様に摩耶さまの連撃で勝利。夜戦の重巡怖いです……。

+ そんなわけで4-4も終了。駆逐×2+重巡×1のルート固定のおかげで羅針盤に悩まされることも(あまり)なく、8回くらいの試行で済んだので楽だった気が。さて、次は3-4だ。

+ ところでそろそろ陸奥とか雪風とかドロップしませんかね。3-4でやれ? まあそうだけどさー。

■艦これの育成たのしいです? [GAME]

+ 重巡を追加で育成開始した。なんしろ鳥海Lv50摩耶Lv49のトップ2の下は妙高Lv27なのだ。リンガ泊地の妙高提督ではないが、今後を見据えるためにはもう少し戦力になる重巡を育てるべきだろう。という愛宕と高雄のレベル上げをちくちくと開始。

+ 駆逐艦は夕立をLv41まで上げてみた。改二は夕立に違いない!と確信してメンテ前から育成してたんだけどLv55は遠いねぇ。まぁ響(ヴェールヌイ)のLv70よりは近いが……。

Oct.12,2013 (Sat)

■艦これ。我が艦隊の戦艦たち [GAME]

+ 手持ちの戦艦は金剛Lv79/霧島Lv70/比叡Lv65/榛名Lv60、航戦の伊勢Lv80/日向Lv66/扶桑Lv50/山城Lv50、陸奥Lv31。見てのとおり主力は伊勢金剛だけど、陸奥が育ったら主力に入ってもらう。長門? 知らない子ですね。

+ 46cm砲と15.5cm副砲を得るべくデイリー開発は10/251/250/10で回してたけど、ずっとゴミしか出してくれなかった。ところが、この2日の8回で46cm砲3本/15.5cm副砲1本が出てきた。どうした金剛。

+ そんなわけで46cm砲が5セットになったので、伊勢金剛に2つ陸奥に1つ載せ、余った41cmを日向・比叡・榛名・霧島も全部41cm砲×2にして、予備戦力の扶桑山城も41cmに乗せかえたの35.6cm砲がだいぶ減った。陸奥も46cm×2にすべくもうちょっと46cm砲レシピを回しますか。

■艦これ。我が艦隊の正規空母たち [GAME]

+ 赤城Lv82/加賀Lv48/蒼龍Lv54/瑞鶴Lv50。赤城→瑞鶴・蒼龍→加賀の順に手に入ったのでレベルもだいたいその順。まあ主力は赤城・加賀なので、加賀のレベルは遠くないうちに二番目になるはず。それにしても飛龍と翔鶴どこー。

+ 装備の手持ちは、艦戦が烈風1・紫電改二3、艦攻が流星改1・流星2、艦爆が彗星一二型甲4、艦偵は彩雲3。烈風も流星改があと1つづつは欲しい感じ? 今ボーキは余り気味なのでちょっとデイリー開発以外にも日に3回くらい作るか……。

+ 艦隊が対空重視なせいか、赤城も加賀も最大スロットに艦戦を詰んで運用しているせいか、今のところボーキ不足になったことはない。が、まぁ、慢心は禁物よねー。

■艦これ。我が艦隊の駆逐艦たち [GAME]

+ 筆頭は響Lv63島風Lv61。次いで夕立Lv55。あとは3-2突破後に遠征組に編入されてちくちく経験値を稼いでいる磯波Lv36/響Lv36/雷Lv36/電Lv36。なんだかんだでLv20以上が20人。最近は漣と初雪がお気にいりかなー。

+ 装備は基本的に10cm連装高角砲/61cm四連装(酸素)魚雷+α。最近は潜水艦相手が多いのでαは爆雷であることが多い。五十鈴改二の三式爆雷投射機1コ以外は九四式爆雷投射機なのがちと悲しいので、爆雷投射機とソナーをもっと開発しよう。

■艦これ。我が艦隊の軽巡洋艦たち [GAME]

+ 五十鈴Lv59の後は由良Lv30長良Lv29球磨Lv26多摩Lv26名取Lv24という五十鈴一強状態をもうちょっと是正したいねぇ。

+ 五十鈴はもともと主力軽巡だったけど改二が来てますます便利。高い対潜能力がよい。ちょくちょく使うのでLv70までは勝手に上がるだろうね。

+ 長良・球磨・名取は最近育成中だが2-xあたりで任務消化するには最適な気がするな。近代化改装MAXの長良・球磨のスペックはなにかおかしい。由良さんは対潜初期値が高いだけで上限は高くないので、育てれば育てるほど普通の人になるのが難点だが「てー!」がなんかカワイイので今後も使うよ(ぉ

■艦これ。我が艦隊の重巡洋艦たち [GAME]

+ 鳥海Lv50、摩耶Lv49に二強は変わらず。

+ 鳥海さんは初期に来た重巡でずっと主力だが、高雄型では唯一改装を受けていない史実を持つせいかパラメータがちょいと劣るのが悲しい(*1)。高雄型で一番活躍した艦なので改二が来たりしないかなー。

+ そして対空番長摩耶さま。鳥海さんが育っていたから摩耶さまを選んだのであって、逆ではないけど、これはこれでかわいい。

+ んで、5-xを見据ええて妙高と愛宕と高雄を育成中。愛宕と高雄はまず改造レベルのLv25を越えさせないとねー。ただ、運用コスト的にデイリー任務消化に使いづらいんで、"ついでに育てる"というのがやりづらいんだよね重巡は。トップ軽巡たちは重巡と大差なかったり越えたりしているのに燃費は大違いに安いし。

■艦これ。我が艦隊のその他の艦たち [GAME]

+ 重雷装巡洋艦の北上Lv63大井Lv61。ハイパー化はしているけど、もうちょっとレベルを上げたほうがいいよなぁ。

+ 軽空母はちよちと飛鷹隼鷹がLv66〜67。ワンパン中破率が高いので本気攻略には使いづらいけど任務・育成支援に便利。あとは鳳翔さんはちよちと甲といっしょに遠征によく行っている。そして当然のように瑞鳳がいない。まあそんなもんよ。

+ 潜水艦。先日やっと伊58がドロップしてくれた。3-2-1引率用に伊168は2隻維持しているので、これで3隻体制で回せる。ゴーヤもイムヤもなんかかわいいよね。

+ 航空巡洋艦の最上がやっとLv21。鈴谷が後をおっかけてるけど改造Lvが35なので今はただの重巡だ。"航空水上打撃艦隊を編成せよ!"の任務が終わる日はまだ遠い……。
