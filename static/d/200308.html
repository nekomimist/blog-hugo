■「お姉ちゃんの3乗」(Marron) [GAME]

+ 1stプレイはりあねえエンド。えちシーンの無茶苦茶さに脱帽(ぉ

+ 2ndプレイはなつねえエンド。後半の甘え&甘えられ全開ぶりに脱帽。そしてえちシーンのさらなる無茶苦茶さに脱帽。一体どういうセンスしてるんだか (^^;

+ 3rdプレイはかもねえエンド。かもねえがいちばんかわいいやね、ちっちゃくて。えちシーンはもーなにがなにやら。

+ 4thプレイはななねえエンド。短かいのう。それにしても、ああ、かみさま、このゲームにまともなえちシーンはないのでしょうか。

+ 5thプレイはドリ子&タケル。ドリ子は立ち絵はアレだがイベントではけっこうかわいいと思う。

+ んでラスト。結局最後までふつーのえちシーンはありませんでした :-)

+ 総評。実に「Marronらしいゲーム」だ。ただ、シモネタ度が上がってる分「秋桜の空に」より人を選ぶかもしれない。俺的には「普通」で6点。「嫌いじゃないが高い評価はつけられない」という感じ。

Aug.12,2003 (Tue)

■「Maple Colors」(CROSSNET) 1stプレイ [GAME]

+ 夏休みに家にひきこもってエロゲばかりやってるように見える記述ばかりなのは、おれとしてもどーかと思うのだが、昼間の事はあまりネタにならんのでねぇ。

+ さておき、やっとこ手をつけた「Maple Colors」。後半になって手間をかけた演出が目減りしていくのをみて微笑ましく思ったりしながら、最初に出会ったいかにもな選択肢を素直に選んだ結果、1stプレイは短髪意地っ張り水泳娘・鈴原空で終了。青山ゆかりがアテるようなキャラに弱い、ということを実感した次第 :-)

Aug.13,2003 (Wed)

■「Maple Colors」(CROSSNET) 2nd〜最後まで [GAME]

+ 2ndプレイは古風で合気道な巫女さん・咲守素子。テレるとかわいかったりするのはよいんだけど、さすがに惹かれるようになるのが唐突すぎる感じがしたな。

+ 3rdプレイは上がり性眼鏡っ娘・桃井葉子。この娘のシナリオを進行するかぎり。この娘は劇では主役なのだが、劇のシーンが一瞬も出てこないんだなぁ (^^;

+ 4thプレイは役になり切るピンク髪のスローテンポ娘・秋穂もみじ。いろいろ盛り沢山な話なのだが、さすがにいろいろ無理がありすぎて苦笑(笑)

+ 5thプレイは素だとキツい美少女芸能人・卯月あむ。ちんまい娘キター(ぉぃ 素直じゃないところも、けっこう乱暴なところもよし。今んところイチオシ。

+ ラストは「キャプテン」・葵未来。二人して意地っ張りで天邪鬼でよい。さすがはメインヒロイン、いちばんケリがつく話ですな。あむを僅差でかわしてイチオシ風味。

+ 総評。前半の仲間集め部分、1回目は試行錯誤して楽しんで、2回目に詰め将棋風味に最短収集を狙って楽しむ。でもそれ以上はやらんわなぁ。3rdプレイ以降は2ndプレイの時に作ったセーブデータを活用して結局仲間集めはやらんかったし。

+ あと、ヒロインの個別シナリオが短いのも難点。おかげでお互いに惹かれる過程に説得力が足りない。これは素子と葉子に顕著だと思う。もみじは過程はいろいろあるんだけど、展開がさすがに無茶な感じ :-)

+ とはいえ他には大きな弱点は感じなかったし、結局のところ楽しかったので、8点の佳作をあげよう。
