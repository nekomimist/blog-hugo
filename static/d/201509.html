Sep.6,2015 (Sun)

■艦これ。2015夏イベ これにておしまい。[GAME]

+ E-7甲Y→E-6甲Mと掘りを行って風雲は無事捕獲したが、E-4甲ボスでの海風ゲットは失敗。というかあれ掘るの苦痛。総合的にはE-1〜E-7甲突破でドロップ限定の瑞穂・風雲をゲットということで、A勝利っぽい感じだろうか。

+ 新艦以外の成果はまるゆ3くらい。掘りが不調なイベントだったなあ。

Sep.8,2015 (Tue)

■艦これ。秋の浴衣 [GAME]

+ 神通さん。Lv102なので桜吹雪つきで、美人度が上がる気がする。ただ、色あい的に背景からちょっと浮いてるかな。夜景なら合いそう。

+ 大淀さん。正月の着物もよかったけど今回もいいなね。この人もLv102で桜吹雪を背負ってるのでよい感じ。でも、藤川さん絵ということなら、たまには衣笠さん夕雲さんに来るといいんだよ。

+ んで、今回はなんか長良と名取がむっちゃツボだった。油断してたらすぐ横から殴られた感じ。こんなおさななじみが欲しかったデース。

+ まず長良。弾着観測気球風味の風船とか、白地に大きい赤の浴衣が活発っぽい長良に似合っててとてもよろしい。日頃の鉢巻のせいでおでこが白いのとか、中破すると日焼け跡が見えたりするのも実にいい。……今Lv97なのでLv99までさっさと上げましょう。

+ 名取。引っ込み思案な名取らしい落ちついた浴衣が実にかわいい。ああ、こんな娘と祭りに行きたかった……(ぉぃ 今Lv89だけど浴衣期間内にLv99まで上げるよう頑張ってみるか。

+ 朧。7駆全員水着に引きつづいて朧だけ浴衣とは予想外だったら。そしてかわいい。

+ 浦風と浜風。安定のパセリニキ。だが、摩耶と鳥海のほうが嬉しかったのにーとか言ってはいけないのだろう。浜風の通常セリフの生真面目さと戦う気なさそうな外見との落差が面白い。

+ 江風と瑞穂。個人的な思い入れがまだまだ足りない二人だわな。瑞穂さん頬染めてえらくかわいらしい感じでよいかも。

Sep.11,2015 (Fri)

■艦これ。"「空母機動部隊」北方海域に進出せよ!"任務 [GAME]

+ 今月はまだ3-5をやってなかったのでEOゲージ削りも兼ねる。3-5は毎月準南or南ルートでこなすので北まわりだと新鮮である。

  • 飛龍改二 Lv136　流星改+7,流星(六〇一空)+7,零式艦戦21型(熟練)+7,彩雲+7
  • 摩耶改二 Lv145　90mm単装高角砲★5,20.3cm(2号)連装砲★4,FuMO25 レーダー,25mm三連装機銃 集中配備,<ダメコン>
  • 北上改二 Lv127　OTO 152mm三連装速射砲★4,20.3cm(3号)連装砲★4,甲標的
  • 赤城改 Lv140　天山一二型(村田隊)+7,烈風+7,烈風(六〇一空)+7,二式艦上偵察機,<ダメコン>
  • 大鳳改 Lv141　流星(六〇一空)+7,烈風(六〇一空)+7,流星(六〇一空)+7,彩雲+7,<ダメコン>
  • 呂500 Lv92　試製FaT仕様九五式酸素魚雷,61式五連装(酸素)魚雷★1,<ダメコン>

+ 夏イベ中に補強増設で穴を開けた娘には念のためダメコンを持たせた。最初摩耶さまの所には北上さんと同装備の大井改二Lv126がいたのだが、ほっぽマスの開幕で大破したので、対空カットインの摩耶さまと交換して上記の編成に。

+ 次の出撃はほっぽマスで全員が対潜装備駆逐艦を放置する自体が発生してろーちゃんが大破。まあ、ダメコンあるし、ろーちゃんなら避けるだろと思って進軍したら、無事避けてくれてボスマスは完全勝利Sだった。

+ 気をよくしてゲージ割り2回目と3回目も上記構成(たまに北上さまと大井さんを交換)でやってみたら、3回の出撃で2回ボス到達してS勝利だった。なお1回は大井さんがほっぽマスの開幕で大破だった。大井さんモロいなあ。

+ ラストは南まわりでやるかもしれないので、一旦今日の出撃はおしまい。航空戦力の熟練度の導入によって、到達率的には北まわりも選択肢に入るようになったと思う。資源消費は多めだけど、戦艦いないので思ったよりはマシだし。

Sep.12,2015 (Sat)

■艦これ。今月の5-5 [GAME]

+ 「今月の」とか言いつつ5-5やるのはじめてだけどな!

+ 艦載機の熟練度導入により、全域航空優勢以上を取りつつ空母が打撃力を発揮できる状況になったということでやってみた。

+ 高速戦艦0-2正規空母4-6という情報も溢れてるが、道中の安定性のために大和型を入れたかったので、戦艦3軽空母1正規空母2のオーソドックススタイルを選んだ。これ↓で制空を取れた上に、大鳳赤城は超火力を維持でき、空母とビスマルクに補強増設でダメコンまで積めちゃうという……おそろしいおそろしい。

  • 飛鷹改 Lv123　烈風(六〇一空)+7,烈風改+7,烈風(六〇一空)+7,烈風+7,<ダメコン>
  • Bismarck drei Lv124　38cm連装砲改★6,38cm連装砲改★6,Ar196改+6,九一式徹甲弾★4,<ダメコン>
  • 大和改 Lv132　46cm三連装砲★2,46cm三連装砲★2,零式水上観測機+7,九一式徹甲弾★4
  • 武蔵改 Lv132　46cm三連装砲★2,46cm三連装砲★2,零式水上観測機+7,九一式徹甲弾★4
  • 大鳳改 Lv141　天山一二型(友永隊)+7,流星(六〇一空)+7,彗星(江草隊)+7,烈風+7,<ダメコン>
  • 赤城改 Lv140　天山一二型(村田隊)+7,流星(六〇一空)+7,流星(六〇一空)+7,烈風+7,<ダメコン>

+ 感触がよくわからなかったので、念のために伊勢136日向136龍驤111隼鷹122駆逐2で道中支援を入れて、最後は道中に加えて飛龍136蒼龍138長門133陸奥133駆逐2の決戦支援も念のために入れた。

+ 最初2回連続ボス前でE風が吹いて心が折れかけたが、その後は5連続でボス行ったので出撃7回で完了。大鳳がエリレの雷撃砲撃にむっちゃ好かれてボス前大破が2回。ボスでのダメコン発動は1回。時間と資源と気分に余裕があるならラスト以外は撤退したほういいので、今後やるときはそうしよう。

+ 安定度のためには道中支援はあったほうがいいが、もっと軽くてもいいかもしれない。MVPは大鳳赤城が取りまくって旗艦の飛鷹と3人で終始キラキラしていた。決戦支援は、あって救われることはまれにはありそうだが、基本的には不要かな。

+ 空母の威力を感じた出撃だった。
