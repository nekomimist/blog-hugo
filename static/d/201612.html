Dec.4,2016 (Sun)

■艦これ。2016秋E-5 朝風掘り実践 [GAME]

+ ほぼこの通りの艦隊で掘った。10月作戦のランカー報酬で烈風改が1つ増えたので、違いはそれを組み込んだくらいか。なお、Oの制空権確保はできていたが、ボスでは確保だったり優勢だったりしていたと思う。

+ 瑞鶴と翔鶴さん、北上さまと大井さんは適宜交代で使った。また、高速戦艦枠はBismarck→比叡→霧島と変えていった。

+ この編成だと22出撃で大破撤退が2回だったけど、ボスで確保取れないなら優勢でもいいかなと攻略時並に制空を落としたら3出撃で2回大破撤退だった。たまたまだとはおもうが、Oで制空権取るのは安定のために必要だった気がする。

+ 基地航空隊は最終的には防空1・O集中陸攻隊L+二式大艇・T集中陸攻隊+二式大艇でやった。O集中なら二式大艇は不要だけど、最初O1T3でやってたのでそのままだった。

+ ボーキは一出撃あたり合計で900前後だった。えぐい。

+ 以上で24出撃、ボスS勝利8回、A勝利12回、大破撤退4回で朝風ドロップして任務完了。

■艦これ。2016秋イベ総評 [GAME]

+ 「そんなにキツくはなかったが、ギミック破砕は面倒だった」という印象だろうか。ただE-5の前進ギミック以外は必須というほどでもないので、これはこだわって破砕した人が負けなだけだったかもしれない :-)

+ 資源消費で見ると、ボーキ以外の消費は16春イベと同等程度、ただしボーキ消費は16春の倍で16夏をも上回る約36000消費で精神的にキツい。

+ 基地航空隊は1部隊分強制的に防空に向けるようになっていて、資源消費が減ると共に、報酬の局戦を活用してる感があってよい。E-5も基地航空隊の活用が重視されている上に、しかも正解が一つでもないのもよい(*1)。前線支援や決戦支援が不要になっていたのもよい。

+ 一方で、戦闘が長いのはなんとかならんかなと思う。連合vs連合に基地航空隊や支援攻撃が入るとホントに長い。単に演出を飛ばせるようにするとサーバの負荷が上がる(*2)から、単純には行かないんだろうが……。

*1: E-5の基地航空隊の出撃先は人によってけっこう違いそうだ。
*2: さらに通常海域の戦果稼ぎがさらにひどくなるだろう。

Dec.9,2016 (Fri)

■台湾でお仕事 [MISC]

+ 仕事で数日台湾に行ってきた。前回の海外出張は2000年なので超ひさしぶり。ま、台湾なので、ホテルはだいたい日本語が通じる人がいたし、自分の英語も16年前よりはマシになってる(と信じたい)ので、まあなんとかなったかなと。まあ、もうちょっと英語は勉強しないとな。

+ 事前にTaiwan Telecomの5日間用SIMをAmazonマーケットプレイス経由買って持っていったが、Nextbit Robinで何の問題もなく使えて便利だった。次は現地で買ってみようかな。

+ あと、自宅のひかり電話ルータ(RT-500MI)のVPNの設定をしておいて、台湾のホテルからVPN(L2TP/IPsec)で自宅に接続してみた。単にWindows10の標準の手順でL2TP/IPsecで設定するだけではなくて、追加でVPNのアダプタのプロパティの"セキュリティ"からCHAPを有効にしないとダメなのでちょっと首をかしげたが、まあ、それ以外は問題なかった。おかげで台湾のホテルでもブラウザ版艦これができたし(←やりたかったのはそれかよ)

+ 行きと仕事そのものは順調だったのだが、帰りの飛行機の出発が1時間遅れて関空から奈良への高速バスに間に合わなかった挙句、ラピートになんとか乗って新今宮で大和路線に乗りかえたら、車両故障の影響でダイヤが乱れていて予定よりさらに帰宅が遅れるなど、なかなか帰路は波乱に満ちていた。

■艦これ。クリスマスグラ [GAME]

+ コマンダン・テストさんは瑞穂の後を継いで差分の女王となるのかな。瑞穂さんすでに差分期間の方が長くなってしまっていて、これ以上差分増やせないからなあ。

+ 球磨。UGUMEさんは流行りの方向に絵柄が変えたっぽくて、球磨も多摩も追加グラがかわいいなあ。木曾や古鷹と加古にも欲しいねえ。

+ まるゆ。……なんだこれは天使か。Lv高めの1stまるゆを運上げに使ってしまったために、今うちのまるゆの最高Lvは47だから、もっとレベル上げてあげたい。

+ 利根は実に「らしい」感じだな。筑摩さんは露出少ないのになんだこの性的な感じ。明石さんの艤装にツリーが生えててちょっと面白い。

+ あとはクリスマスと無関係に、第7駆逐隊のラストを飾る曙改のグラ変更。これで初期から言われていた艤装の吸気口問題が解決か(*1)。それにしても曙は細すぎやろ。もっと食わせたい。

*1: まあ潮は改二じゃないと直らないし、他の子も改じゃないと変わらないが。

Dec.11,2016 (Sun)

■arrows NX F-01J [MISC]

+ さて、arrows NX F-01J。白ROM買おうかとも思ったけど、サポートとかの話もあるし、結局4年維持していたガラケー回線から機種変更した。はじめてスマホ割がついたので少しだけ月々の維持費がマシ。

+ で、台湾に行くちょい前に届いていたのだが、台湾ではsimフリーのNextbit Robin以外使わなかったので、日本に戻ってきてやっとぽちぽち設定してみている。

+ サイズはRobinとほぼ同じ。つまりRobinは画面サイズに比して少しデカい。発熱がarrowsの方が圧倒的に小さい。まあ14nmでCortex-A53x8のSnapdragon 625と、28nmでCortex-A57x2+Cortex-A53x4のSnapdragon 808を比べちゃいかんわな。

+ シングルスレッド性能は少し下がったハズだが、使用感はとしてはひっかかりが減った感じがする。GPU性能的に世代が上なおかげか、発熱が低いせいでサーマルスロットリングがキツくないせいか……よくわからん。

+ 認証は、指紋センサのRobinに対し、虹彩センサのarrowsとなった。メガネのレンズ越しだとちょっと認識が遅いので、レンズの外側から虹彩を認識範囲に入れるといい感じかな。手袋してても使えるのが虹彩センサのいいところだろうか。

+ docomo標準アプリがウザいが、まあしょうがない。無効にできるものは無効にしておけばいいしね。

+ 今のとこころは「悪くない買い物だった」かな。ちょっと高いが。

Dec.31,2016 (Sat)

■2016年の終わりに [MISC]

+ 会社方面では波乱の2016年であった。自分の周囲はFoxconnの傘下に入ったことで、変わったこともあり変わらないこともあり。自分の役割はあまり変わらない感じではある。ま、役に立てるように頑張りましょう……。

+ 趣味のゲーム方面は艦こればかりやりつづけてるなあという感じ。安上りでよいね。えろゲは本当にほとんどやらなくなった。たまには買うけどね。

+ 趣味の自転車。まあロードはあまり乗ってないけど、通勤用の丸石エンペラーツーリングテトラはちょこちょこ弄って楽しんでいる感じ。パキっと調整できている自転車は通勤で乗るのもまた楽しい。ミニベロだけどランドナーなので輪行してどこかに出掛けるのもいいかもしれない。

+ PC方面はハード的にはHTC VIVEを買ったくらいか? CPUはCore i7-4790Kで数世代前なので、そろそろ買いかえたい気分ではある。IntelのKabyLakeもいいけど、AMDのRyzen(Summit Ridge)はどうなるかな? GPUはGeForce GTX970でこれも1世代前。RADEON好きとしてはCPUと揃えてAMDにする夢も見てみたい。

+ ソフト面はあまり変わらず。ブラウザはFirefoxからVivaldiに行って、またFirefoxに戻っている(*1)。エディタはEmacsのままだが、.emacsの変更料がものすごくなっていて、自分のEmacsの使い方がいろいろ変わっている感じはする。言語としては最近の自分の流行りはgoかな? Rustも興味はあるのだが。

*1: Vivaldiはサブとして現役。
