Jul.1,2008 (Tue)

■きょうのよしなしごと [MISC]

■5月以降のえろげとか [GAME]

+ ちまちまプレイしていっているがなかなか感想を書く元気が出なかった。

★「暁の護衛」(しゃんぐりら)

+ 絵もキャラも悪かないんだが、ものすごく尻すぼみです……。6点。

★「新妻イカせてミルク!」(アトリエかぐや)

+ ダンナ共がオバカさとか、奥さんたちのどこかユカイな堕ちかたといい、俺はけっこう好きなのだが、かぐやから出た意味がよくわからんわなー。7点。

★「Sugar+Spice!」「Sugar+Spice!Party★Party」(チュアブルソフト)

+ ついかっとなって両方まとめて買った。別に反省はしていない(ぉぃ

+ 本編はたいへんよくできていたように思う。コマいイベントが一杯あって、かつ「みんな」感が強いので楽しい。

+ キャラ的には司がイチオシだな。「ピリオド」(Littlewitch)の朝姫あたりからずっと、金松由花の声に弱い俺なのでした。「朝凪のアクアノーツ」(Fizz)のメリーにも当然のようにやられたしなー。

+ そんなわけでオトメは特別だが、ピョンとミャンマーも好きなな。ジジとハモは一歩足りない。まぁとはいえアリガチでないヒロインたちはなかなかよかったように思う。8点。

+ 「P★P」の方はなんか残念だ。本編はちまちましたイベントが楽しいゲームだったのだが、そのへんを素直に補完する方向にもっていけばよかったのにねー。ミャンマーのアレみたいなイベントがなんでオトメにないんですかっ。6点。

★「彼女×彼女×彼女〜三姉妹とのドキドキ共同生活〜」(ωstar)

+ ただひたすらエロいだけだがそれがよい。メインとなる織節三姉妹の話の展開が全て同じっつーのも、「お話なんかどーでもよいのです」感が漂っていていっそすがすがしい。

+ その割り切りっぷりに免じて8点(ぉぃ

★「夏空カナタ」(ゆずソフト)

+ ヒロイン同士のかかわりがちょいと弱すぎなせいか、なんかこぢんまりとした感じがする。茅羽耶のアレはもっとみんなで取り組む感じがあったほうが面白かったような気もするしなぁ……。

+ ヒロイン3人のうち、茅羽耶と沙々羅はルートに入らなきゃぜんぜん出てきやしないので、誰のルートでもそれなりに出てくる残り1人の由比子が一番印象に残った。なにこの消去法。由比子の話は他とちょっと毛色が違いすぎてアレかもしれんが。

+ ま、7点かね。

★「ef - the latter tale.」(minori)

+ いろいろお見事と言えよう。幸せな感じに終わるゲームが好きな俺としては、終わり方に思うところがないでもないが(*1)、だからといってあの終わり方を変えてしまったらいろいろ台無しだ。

+ ま、評価は9点だ。終えたあと思わずfirst taleをリプレイしたのは俺だけじゃあるまい。

★「ユニティマリアージュ」(ユニゾンシフト)

+ 悪くはないのだがよくもない、ある意味とてもユニゾンシフト:アクセントらしいゲームであった。ま、「Chu×Chuアイドる」よりは明らかに俺好みだったが、最初にやった「おしえてRe:メイド」を超えたりはせんな。

+ 凌辱シーンがちょっとだけあるのだが、もっと徹底的にズルズルに堕ちるまでやってほしかったのは俺だけだろうか(ぉぃ 評価は7点かな。

★「こんぼく麻雀」(あかべぇそふとつぅ)。

+ 何も考えずにプレイしていると、終盤敵のHPが10万とか20万とかになる。役満上がっても1/3も削れませんよ? 一体これは何の苦行だろう。5点。

*1: そんなわけで総合的には「はるのあしおと」(minori)の方が好きかな。

Jul.11,2008 (Fri)

■FIENA冷却編 [COMP]

+ RADEON HD4870のせいで電源負荷が上がって電源ファンの速度が上がったのか、単にケース内の熱が増えてCPU・ケースファンの速度が上がったのかわからんが、とにかくFIENAの騒音レベルが気になるほど上がってしまった。

+ RADEON HD4870のクーラーを交換して、ケース内ファンの配置を見直して、電源をもうちょっと余裕があるものに交換する……なんて事を考えたが、この際水冷にするのもよいかもしれん。HD4870用の水枕が出てくるのはこれからだろうから、とりあえずQX9650から水冷化しよう。QX9650が性能のわりに低消費電力とはいえ、FSB1600MHzで3.6GHz運用してる以上、それなりに発熱してるしな。

栞 : 「……むりやり散財ネタをさがすのをやめなさい」
猫 : 「ストレス発散のために必要な散財なのだー」

+ といわけで、ぽちっとな(ぉぃ

Jul.12,2008 (Sat)

■iPod Touch 2.0 [COMP]

+ iPod TouchでRemoteで、リビングにあるPCのiTunesを制御できる。AirTunesの出力先の切り替えもできるので、寝室にAirMac Expressとパワードスピーカーを置けば、リビングと寝室の音楽出力を家のどこからでも制御できるようになるわけで、用途が広がりそうだ。

+ うちでは全く役立たずだったiPod Touchがやっと日の目を見るときが来たかもしれない :-)

Jul.14,2008 (Mon)

■iPod touch 2.0とRemote [COMP][AUDIO]

+ 「Remoteはかなり衝撃的」。まったくもって同意。Remoteがあまりにイケてるので、寝室用にYAMAHA NX-A01と、2個目のAirMac Expressを買ってきてしまった。

+ 「iPod touch(or iPhone)のI/FでiTunesを遠隔制御できる」ただそれだけなのだが、やはりAirTunesの切りかえができるのがすばらしい。

+ 寝室のベッドの上でiPod touch経由でリビングのPCのiTunesを操作して、寝室のAirMac Expressに音楽流せるし、リビングに戻ってほいっとAirTunesの出力先をリビングのAirMac Expressに変更すればリビングのオーディオセットで音楽を聴ける。

+ もーうちのiPod touchは壊れるまでRemote専用でいいや(ぉぃ
