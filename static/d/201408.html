Aug.9,2014 (Sat)

■艦これ。夏イベE-1 [GAME]

+ 阿武隈Lv64/那智Lv78/足柄Lv78/山城Lv98/飛鷹Lv100/隼鷹Lv100でE-1に突入。

+ 阿武隈と那智と隼鷹は史実に沿って。飛鷹も竣工が間にあっていたら北に行ったはずなので参加。足柄と山城はそれぞれ重巡・戦艦系で最低Lvだったので。

  • 阿武隈は三式爆雷+三式ソナー+22号対水上電探改四
  • 那智・足柄は20.3cm連装砲×2+33号対水上電探+水偵
  • 山城は46cm+試製35.6cm三連装砲+徹甲弾+水観
  • 飛鷹は烈風・流星改・流星改・彩雲
  • 隼鷹は烈風・紫電改二・流星改・二式艦上偵察機

+ 烈風も32号電探も水観も20.3cm(3号)連装砲も、他の娘からもってくればあったのだが、なんか面倒だったのでそのまま :-)

+ 全員3回キラづけして、途中キラが切れたら再度キラづけする作戦。9回突入・3回大破撤退・6回ボス到達&撃破で、バケツが30個くらい減ったかな。

+ E-1にしては過去最凶のマップだったけど、2-4突破陣の一人だった山城を除くと新規海域の突破に使われたことのない娘ばかりを率いて戦うのは悪くなかったかも。

Aug.10,2014 (Sun)

■艦これ。夏イベE-2 [GAME]

+ 磯波Lv88/敷波Lv88/三隈Lv94/熊野Lv94/飛鷹Lv100/隼鷹Lv100でE-2に突入。

  • 磯波・敷波は33号対水上電探×2・22号対水上電探改四
  • 三隈・熊野は20.3cm連装砲・瑞雲12型・20.3cm連装砲・三式弾
  • 飛鷹は烈風・彗星十二型甲・天山十二型(友永隊)・彩雲
  • 隼鷹は烈風・烈風・彗星(江草隊)・彩雲

+ いわゆる索敵装備マシマシの南側ルートで進行。全員3回キラづけして、途中キラが切れたら再度キラづけする作戦。

+ 結果としては、11回突入で11回ボス到達だがボス撃破は4回。昼で周囲を整理して夜戦で潰す作戦なのだが、なかなかそこまで三隈熊野が無傷で残ってくれなかった。おかげで最終形態を倒すのに3回連続で妖怪チョットタリナイが発生した。最後は中破の三隈さんが決めてくれた。

+ 大破撤退が一回もなかったので、道中支援は入れなかった。後半5回くらいは決戦支援を投入。1隻潰してくれる程度だったが、まぁ、最後は支援がなかったボスは潰せなかったはず。

+ さて、これでAL方面は終了。残存資源は油155k/弾151k/鉄230k/ボーキ165k/バケツ1116。まだまだ行けるはず。

Aug.11,2014 (Mon)

■艦これ。夏イベ休止中 [GAME]

+ ちょっとこわいアナウンスが出てるので、MI出撃は中断中。大淀さんを育成したり、春雨さんをバケツ回収にまわしたりして過ごす。

+ E-2ゲージを破壊してくれた三隈さんがすげぇかわいいことに今さら気付いた。そもそもしばふ艦には弱いし、まめぐ声にもわりと弱いのだよね。というわけで、航巡初の嫁は三隈さんになってもらおう。現在Lv94だし、わりとすぐだよね。

Aug.31,2014 (Sun)

■艦これ。夏イベ感想 [GAME]

+ 今さら夏イベの感想を。

+ ボスゲージ回復がないのはよかったけど、ボスゲージが長すぎてダレたかな。ボス撃沈10回分とか勘弁願いたい。

+ AL/MI/E-6で出撃艦を絞られるのは面白かった。E-1の旗艦阿武隈と随伴の那智・足柄・山城という構成とか、E-2の駆逐艦2人に磯波・敷波を、主火力担当に三隈・熊野を選んだりするのは、こういうイベント形態でなければ絶対やらなかったしね。

+ E-3〜E-5の連合艦隊はちょっと大味だったかな。艦隊司令部施設を活用する必要がない程度に楽勝だったし。可能性は感じるのでブラッシュアップに期待したいところ。

+ そしてE-6。北上・大井・大和・武蔵・千歳・千代田・翔鶴・瑞鶴・大鳳・重巡主力たちを残していたし、全資源も140k以上あったので資源と戦力の心配はいらなかった(*1)が、キラづけして突撃のループでちょっと辛かったかな。

+ 攻略中のレアドロは早霜・清霜・あきつ丸×2・まるゆ×4・三隈くらいだろうか。E-6まで攻略完了して早霜清霜が出たので、うちにいない娘はビスマルクだけの状態に戻った。……そろそろ大型建造しなければな。

+ AL方面のMVPは三隈さん。MI方面のMVPは神通さん。E6のMVPは北上さまかなぁやっぱり。

*1: なお翔鶴瑞鶴に出番はなかった模様 :-p
