■Windows10 TH2とCygwin? [GAME]

+ いわゆるTH2、Novemver Updateがメイン自作PCとVAIO Duo11の両方に降ってきたので当ててみた。

+ するとckw-modでcygwinのzshを起動すると良く見るfork()のエラー(↓)が出るようになった。rebaseallしてもダメ。Cygwinのforkはホント魔窟だ。

      0 [main] zsh 8392 c:\cygwin64\bin\zsh.exe: *** fatal error in forked process - fork: can't reserve memory for parent stack 0x600000 - 0x800000, (child has 0x400000 - 0x600000), Win32 error 487
    260 [main] zsh 8392 cygwin_exception::open_stackdumpfile: Dumping stack trace to zsh.exe.stackdump
      0 [main] zsh 15136 fork: child -1 - forked process 8392 died unexpectedly, retry 0, exit code 0x100, errno 11
compinit:534: fork failed: resource temporarily unavailable

+ まあ、ckwのようにコマンドプロンプトを取り込んでないと困るケースも減った(気がした)ので、この際だからCygwin distributionに含まれているminttyに切りかえてみたら何の問題もなかったので、このままminttyを使うことにした。
